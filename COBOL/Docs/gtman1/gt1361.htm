3.5 Acucorp's Indexed File Record Editor - alfred

alfred is an interactive, menu-driven, record-level editor for use with indexed files. It can be used to edit any indexed file supported by your ACUCOBOL-GT runtime.

alfred allows you to view, modify, add, or delete individual records stored in an indexed file. It's especially valuable to developers engaged in testing and debugging data file layout and file I/O actions.

alfred is also valuable to application administrators who need to make occasional, record-level changes to data files. It is not well suited to making large scale, global or repetitive changes.

alfred incorporates floating windows, dialog boxes, and bitmapped buttons in its user interface. It uses character-mode emulation to present the graphical interface in character-mode environments.
