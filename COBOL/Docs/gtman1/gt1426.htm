4.4.2.1 SCREEN examples

The following sample recaps the default settings and provides an example of how to specify SCREEN configuration entries.

SCREEN       

Convert-Output=No

SCREEN

Edited-Updates=Converted, Auto-Prompt

SCREEN

Error-Bell=Yes

SCREEN

Error-Color=4096 Error-Line=-1

SCREEN

Input-Display=Spaces Input-Mode=Normal

SCREEN

Prompt=_ Justify=Auto

SCREEN

Numeric-Updates=Converted, Auto-Prompt

SCREEN

Alpha-Updates=Unchanged

SCREEN

Shadow-Style=Black
