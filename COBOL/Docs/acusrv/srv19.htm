1.2.6 Vision Files

The Vision file system is the default file system used with ACUCOBOL-GT for all environments other than VMS.

By default, AcuServer Version 7.2 creates Vision Version 5 files. (AcuServer can also be used to build and access Vision Version 3 and 4 files.) Version 5 files are generated in a dual file format, with data records stored in one segment and overhead key information stored in another.

To cause AcuServer to build Vision Version 3 or 4 files, see the configuration variable V_VERSION in section 4.2.2, "Server Configuration Variables."
