3.6.4 Confirming Network Services

On UNIX and Windows clients, confirm that your TCP/IP software is loaded and running.

If desired, use "ping" to confirm that the server is up and responding to the network. From your client system, type: 

ping hostname

Ping should return: 

hostname is alive

Alternatively, you can use Acucorp's AcuPing program, which is found in the Start/Programs/Acucorp 7.x.x/AcuServer menu. AcuPing is discussed in section 8.3, "Diagnosing Errors with C$PING."
