3.6 Installing the Client

There is nothing on the AcuServer distribution media to install on the client machine. However, you should ensure that every client system that will use AcuServer has a licensed copy of the current ACUCOBOL-GT runtime. To check that your runtime is current, "-v" option on the runtime command line of the client machine. The runtime will display information (either at the command line or in a Windows message box) similar to the following: 

ACUCOBOL-GT runtime version 7.2 
Serial number 1234 
Licensed for 2 user(s)  
AcuServer client 
Vision version 5 file system 
Copyright (c) 1985 - 2006, Acucorp, Inc.

Look for the line "AcuServer client" and a runtime version number that matches the AcuServer version. If either of these criteria is missing, the client machine will not be able to access data using AcuServer.
