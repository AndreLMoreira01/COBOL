4.2.1 Runtime Configuration Variables

In addition to the runtime configuration variable ACUSQL_RUNTIME_DLL, described in section 4.2, AcuSQL support for SQL Server includes additional runtime configuration variables:

ASQL_BUFFER_SIZE

ASQL_CONNECT_DATABASE

ASQL_DISCONNECT_ACTION

ASQL_ERROR_MAP_FILE

ASQL_SCROLL_CURSOR

ASQL_START_TRANSACTION

ASQL_TRANSACTION_METHOD

These variables are of particular use when working with transactions. The variables can be set in the environment or in a runtime configuration file.
