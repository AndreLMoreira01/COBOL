2.5.1 Cursors

Cursors are defined in the Procedure Division or in the Data Division. For example: 

EXEC SQL DECLARE MYCURSOR CURSOR FOR 
   SELECT FIRST_NAME, LAST_NAME 
   FROM TEST_TABLE 
END-EXEC.

See section 5.6.1 for a brief introduction to cursors. Consult any of the commercially available books on SQL for more complete details.
