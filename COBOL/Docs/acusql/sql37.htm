ACUSQL_DATASET

The ACUSQL_DATASET environment variable is used in conjunction with the "-Pc" pre-compiler switch ("-Pc" directs the pre-compiler to connect to the target database engine to perform syntax checking). ACUSQL_DATASET should be assigned the name of the ODBC data source to use. (Windows users refer to this value as the User DSN.)
