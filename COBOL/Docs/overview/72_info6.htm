Enhanced Access to Data

THE EXTFH interface has been enhanced to support variable length files. When using this interface, you can now indicate whether the file system you are accessing can handle variable length files. Three new configuration variables correspond to indexed, relative, and sequential files:

A_EXTFH_VARIABLE_IDX

A_EXTFH_VARIABLE_REL

A_EXTFH_VARIABLE_SEQ

When the file system does not process variable length files, set these configuration variables to "0" and the EXTFH interface treats variable length records as fixed length.

For more information, see Appendix H in Book 4, ACUCOBOL-GT Appendices.
