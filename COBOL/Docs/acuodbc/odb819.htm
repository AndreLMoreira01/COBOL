ACCESS_FILE

The ACCESS_FILE variable affects AcuODBC Server security.

ACCESS_FILE must hold the full path and file name of the server access file, if the name or path are other than the defaults ("/etc/AcuAccess" on UNIX systems; "\etc\AcuAccess" on Windows NT or Windows 2000 servers).
