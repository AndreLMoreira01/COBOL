Division by Zero

This list box allows you to chose how AcuODBC handles queries in which a number is divided by zero.

If the value is set to... AcuODBC returns...

Error

the error message "Division by zero"

Value

the dividend (divides by one)

Zero

0

The "Division by Zero" setting only affects queries carried out by AcuODBC. Some applications (including Microsoft Access) override this option.
