To start and stop acuodbcs using the ODCP

  1. At the Windows Start menu, go to Programs/Acucorp 7xx/AcuODBC/AcuODBC Server and select AcuODBC Server Control Panel.

    2. The AcuODBC Server Control Panel opens with the Access tab selected.

  2. Select the Services tab.

    3. A list of installed services appears. Services that are currently running are marked with a green bullet, while services that are currently stopped are marked with a red bullet.

  3. To start a service that is currently stopped, click on the port for the service to select it, then click Start.

    4. A green bullet appears next to the port number for the service to indicate that acuodbcs is running.

  4. To stop a service, click on the port for that service to select it, then click Stop. A red bullet appears next to the port number to indicate that the installed service is not currently running.

Note: Using the ODCP, you can modify startup options for services without removing and reinstalling them. For instance, you can use the Properties button on the Services tab to modify dependencies or other service properties, and then restart the service using the new options.
