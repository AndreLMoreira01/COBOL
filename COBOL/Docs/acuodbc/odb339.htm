Deleting an individual company entry

Follow these steps to delete an individual company DSN:

  1. Open the Windows control panel.
  2. Do one of the following to open the Data Source Administrator:
    • On Windows NT machines, double-click ODBC Data Sources.
    • On Windows 2000 machines, open the Administrative Tools folder and double-click Data Sources (ODBC).
  3. Select the appropriate DSN, and click Configure.
  4. Select the File Alias tab.
  5. Click on the appropriate row and then click Delete.
  6. Click OK to close the AcuODBC Configuration property sheet.
  7. Click OK again to close the ODBC Data Source Administrator.
