5.5 SQL Reserved Words Recognized by AcuODBC

AcuODBC recognizes the following SQL reserved words. Note that AcuODBC supports the same functions as your application. That is, if your ODBC-enabled applications support these functions, you will be able to perform them on COBOL data via AcuODBC. AcuODBC does not support these reserved words if your application does not.
