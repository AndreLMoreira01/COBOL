9.3 Configuring the Remote DSN on a Windows Server

The AcuODBC Server Configuration utility, which is based on the AcuODBC Configuration property sheet, is used to configure remote DSNs from a Windows server. Two noteworthy differences between the AcuODBC Server Configuration utility and the AcuODBC Configuration property sheet are as follows:

  • There is an additional field on the General tab.
  • The AcuODBC Server tab is not part of this property sheet because the information on that tab describes the server you are working on.
