3.6 Processing Remotely

The fields on the AcuODBC Server tab provide information about the remote processing server.

Usage

Indicate in this field whether SQL processing will occur locally or on a remote server. Local processing is the default. Selecting remote processing activates the "AcuODBC Server Location" and "Security Method" fields.

See section 9.2, "Configuring the Remote DSN on a Windows Client," for information on the remaining fields on this tab.
