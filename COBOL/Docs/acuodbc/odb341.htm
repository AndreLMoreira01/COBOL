3.9 Accessing Remote Files

With AcuODBC, you can use AcuServer to access remote files and process them locally. Supply the appropriate information on the AcuServer tab as described in the following sections. If you are processing remotely via AcuODBC Server, files located on the application server are not considered remote, and you do not need to complete this tab.

Note: If you are running AcuODBC Version 7.2 or higher, you must also run AcuServer 7.2 or higher. That is, your version of AcuODBC must match your version of AcuServer. Multiple versions of AcuServer can co-exist on the same machine (to support multiple versions of other Acucorp products), but running multiple versions is not recommended.
