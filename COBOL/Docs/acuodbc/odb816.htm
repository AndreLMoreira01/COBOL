Exiting the Access Manager

To exit the command line manager, select option 5 from the main menu.

To exit the graphical manager, close the AcuODBC Server Control Panel dialog box or click Cancel. Note that clicking Cancel does not undo any changes that you have made to the AcuAccess files.
