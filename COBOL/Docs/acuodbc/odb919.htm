State:[Enabled], Disabled:

"State" is equivalent to the check box on the File Alias tab of the AcuODBC Configuration Property sheet that indicates whether or not AcuODBC will include that entry when performing operations. Choose "Disabled" when you do not want AcuODBC to include this alias. AcuODBC keeps the alias information in memory so that you only need to change the "State" entry to indicate whether a particular file alias should be used by AcuODBC. See section 3.7, "File Aliases," for more information.
