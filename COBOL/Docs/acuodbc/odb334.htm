3.7.1 Modifying a File Alias

Follow these steps to modify an alias:

  1. Open the Windows control panel.
  2. Do one of the following to open the Data Source Administrator:
    • On Windows NT machines, double-click ODBC Data Sources.
    • On Windows 2000 machines, open the Administrative Tools folder and double-click Data Sources (ODBC).
  3. Select the DSN that you want to modify, and click Configure.
  4. Select the File Alias tab.
  5. Click on the alias you want to edit and then click Modify. The alias is removed from the list box and appears in the edit box at the top of the tab.
  6. Make any necessary changes, and then click Add to put the revised alias back into the list box.
  7. Click OK to close the AcuODBC Configuration property sheet.
  8. Click OK again to close the ODBC Data Source Administrator.
