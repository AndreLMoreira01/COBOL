8.6 Checking Version Information

From the command line, type "acuodbcs -version". This returns a message with version and copyright information, similar to the following: 

C:\>acuodbcs -version 
ACUCOBOL-GT AcuODBC Server version 7.x.x 
Serial number XXXXXX 
Licensed for 2 user(s) 
Copyright (c) 1985-2005 Acucorp, Inc. 
 
C:\>

Note that "-version" must be the only argument on the command line.

In the AcuODBC Sever Control Panel, click on the AcuODBC Server icon on the left-hand side of the title bar to open the system menu, then select About.
