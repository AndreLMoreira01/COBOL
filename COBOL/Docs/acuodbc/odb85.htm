8.3.1 Using the Access File Manager Utility

Create and maintain the server access file with the access file manager utility.

The access file manager is an interactive, menu-driven utility that is accessed through the command-line in UNIX and through a GUI interface in Windows. In both platforms, the access file manager allows you to:

  • create an access file
  • add access records
  • remove access records
  • modify access records
  • display access records

Note that both AcuODBC Server and AcuServer® have a Windows graphical interface with an Access tab and an access command. They both control files that are identical in format, and so can be used interchangeably. This chapter, however, focuses on the "AcuAccess" file and AcuODBC Server. (For information specific to the "AcuAccess" file and AcuServer, please see your AcuServer documentation.)
