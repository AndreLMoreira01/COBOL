To delete access records

You can delete access records either from the command line or, on a Windows server, from the AcuODBC Control Panel.

  1. To remove a record from the command line, select option 2 from the main menu.

    2. The manager presents two prompts, one for the client machine name, and one for the client user name. The prompts look something like this: 

    Enter official client machine name: 
Enter client user name:
  2. If a matching record is found, it is removed. Press Enter to return to the access file manager main menu.
    3. Record removed 
Press <Return> to continue...

To remove a record from the ODCP, highlight the record name and click Delete.
