To edit remote DSNs using srvconfig.sh

You can make changes to an existing DSN through the srvconfig.sh utility.

  1. Start the srvconfig.sh utility
    2. $ srvconfig.sh
  2. When prompted, enter the name of the appropriate data source as it appears in the listing. UNIX is case sensitive, so "multi.dsn", "Multi.dsn", and "MULTI.dsn" refer to different data source names.
    3. Current Data Sources: 
multi.dsn   testing.dsn 
 
Data Source Name:  DSN-to-be-edited

    You may enter only the base name of the DSN. It is not necessary to include the ".dsn" extension.

  3. Enter new values or accept the defaults to keep the existing entries.
