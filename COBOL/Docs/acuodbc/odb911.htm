Deleting remote DSNs using srvconfig.tcl

  1. Change to the directory containing the DSN you wish to delete.
  2. To start the UNIX graphical configuration utility, enter the following on the command line:
    3. wish srvconfig.tcl

    The AcuODBC Server Configuration dialog box appears.

  3. Select the appropriate data source and click Remove.
  4. Click OK to close the AcuODBC Server Configuration dialog box.
