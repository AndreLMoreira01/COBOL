1.3.1.1 License file names

The name of each license file distributed by Acucorp is based on the name of the product's executable file, with a special extension. For example, if your runtime executable is named "runcbl," the runtime license file is named "runcbl.alc," and the Acu4GL for Sybase license file is named "runcbl.ylc." If your compiler executable is delivered as "ccbl," then the compiler license file is "ccbl.alc." The ACUCOBOL-GT Transaction Server runtime license file is named "runcbl.1lc" (UNIX/Linux) or "wrun32.1lc" (Windows). Each product has its own license file.

Note: If you rename the product's executable file, you must change the base name of the license file to match it, and leave the license file extension unchanged. For example, if you rename your compiler from "ccbl" to "mycomp," you must rename the license file from "ccbl.alc" to "mycomp.alc."

License files with an extension ending in an exclamation point (!), are back-up copies of old license files. See section 2.1 for more information.
