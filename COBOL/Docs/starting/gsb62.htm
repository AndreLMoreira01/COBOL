6.1 Using Multiple Versions of ACUCOBOL-GT

Version 7.x license files work only with the Version 7.x series of products. You may easily run Version 5.x or earlier versions of Acucorp products alongside Version 7.xproducts. To do this, simply install the products and license files in a separate directory for each version.

For example, if you place the Version 5.2.1 compiler and license file in the "C:\Acucorp\Acucbl521\AcuGT" directory, and place the Version 7.0 compiler and license file in the "C:\Acucorp\Acucb700\AcuGT" directory, either version of the compiler will execute properly.
