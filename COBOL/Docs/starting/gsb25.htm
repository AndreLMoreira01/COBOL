2.2.2 Installation from Floppy Disk

  1. Insert the first product installation disk into your floppy drive.
  2. Run "setup.exe". Click on Start and select Run. Type x:setup in the entry box (where "x" is the letter of the drive containing your disk).
  3. Follow the instructions on the screen. When prompted for subsequent installation disks, insert the disk requested.
  4. At the Choose Destination Location screen, you are prompted to specify an installation directory or to accept the default location.

    5. Note: Do not specify a directory name that has embedded spaces. Embedded spaces prevent the PATH variable from being properly set and can cause other problems.

  5. After you install your Acucorp products, "setup.exe" launches the Activator automatically. Type the product code and product key that came with your product package in the appropriate entry fields. The Activator is case insensitive and displays only uppercase characters. (Note that product codes and keys do not contain the letters "O" or "I.") The Activator ignores embedded spaces and separating characters.



    6. If you have more than one code and key pair to enter, select the More button to continue entering code/key pairs. Each time you press More, the Activator creates the license file for the product code and key you just entered and returns you to the code and key entry screen. You may select More as many times as you have product code and key pairs. Select the Finish button to create your license file. There will be a separate license file for each Acucorp product.

  6. Repeat this installation procedure on each workstation or client that will require Acucorp products (such as AcuODBC).
  7. Begin using your Acucorp products. Once the above steps are complete, the Activator displays a dialog box indicating whether or not license file activation was successful.

    8. Note: If license activation was successful, but you get a message during product startup indicating that the license file cannot be found, the license file may not be in the proper directory. The Activator determines where to place the license file based on entries in the Windows registry. If no registry entry is found, the license file is placed in the same directory as the Activator executable. (The default location for the Activator is "C:\Acucorp\Acucbl7x\AcuGT\bin".) If this is not the location of the product's executable file, move the license file to the directory containing the corresponding executable file. For example, move "wrun32.alc" to the folder containing "wrun32.exe".

When installation is complete, you will find an icon named "tour.acu" in your "Acucorp\Acucbl7x\AcuGT\sample\samplprj\object" folder. Double-click this icon to execute a sample program that displays a window with a graphical interface containing a variety of controls and demonstrates how they work.
