11.6.2 Drawing Controls

The Screen Designer lets you draw standard and ActiveX controls directly on a graphical or character-based screen form as follows:

  1. Open a screen.
  2. In the Screen Component Toolbox, click the Standard or ActiveX (for graphical screens only) button to display the type of control that you want to draw on your screen.
  3. Click the icon for the specific control that you want to draw.
  4. Move the pointer to the screen form. The pointer changes to a crosshair.
  5. Drag with the left mouse button to position and size your control. Note that graphical controls and character controls are measured differently in the screen form. Graphical controls are measured by pixel, and character controls are measured by cell.

The Property window displays a control's properties and events based on its initial state. You can change property settings by clicking on a value and entering a new value, choosing from values listed in a drop-down box, or selecting values in a dialog box. More information about property settings can be found in Book 2 of the ACUCOBOL-GT documentation set. Information about event editing can be found later in this chapter.

After you finish adding controls to the screen, use the Align commands to size, align, and space the controls. To preview the appearance of your screen, choose the Build/Execute command. When you use this command, the workbench performs several functions: save screen, generate code, compile program, and execute program. After previewing your screen, you can return to design mode by closing the preview window.
