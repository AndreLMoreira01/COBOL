11.6 Designing a Screen

This section describes how to design your screen with controls. A control is a graphical object that you draw on a screen using tools from the Screen Component Toolbox. In addition to the Selector, a Menu Designer, and a Toolbar Designer, the toolbox contains the following standard controls for graphical screens:

Bar
Bitmap
Check box
Combo box
Date entry
Entry field
Frame
Grid
Label
List box
Push button
Radio button
Scroll bar
Status bar
Tab
Tree view
Web browser

Character-based screens can contain all of these controls except the bitmap, date entry, tab, grid, Web browser, and status bar controls. When you work in a character screen, the Screen Component Toolbox icons for these controls are disabled. Each control has a set of properties and events that you can use to define the control's function in your application. AcuBench displays these properties and events in the control's Property window.

The Screen Component Toolbox also has a section for the ActiveX controls that you want to use in your screens. ActiveX controls are available only for graphical screens. More information about the addition of ActiveX controls to your Screen Component Toolbox can be found in section 10.3.2, "ActiveX Controls."

Control icons may be viewed in a list with labels or as bitmap push buttons with pop-up hints. Simply right-click in the Screen Component Toolbox and select List View to toggle these views of the controls. More information about the Screen Component Toolbox can be found in section 10.3, "The Screen Component Toolbox."
