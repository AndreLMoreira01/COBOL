8.4.2 Editor Toolbar

The Editor toolbar is a palette of command buttons for the Code Editor's special functions, which specifically facilitate the code generation and maintenance process. You can display or hide the Editor toolbar by choosing the View/Toolbars/Editor command. You can also display or hide the Editor toolbar by right-clicking in the toolbar and choosing Editor, or by selecting Tools/Customize and then Editor in the Customize dialog box. The Editor toolbar contains the following commands:

Find. Lets you enter the text string you want to find, or lets you select text from a drop-down list of recent search operations.

Find Next. Advances you to the next occurrence of the selected item in the active document.

Find in Files. Lets you locate text strings in files stored on your drives.

Replace. Lets you replace items in the text with other items.

Toggle Bookmark. Inserts a bookmark into the Bookmark pane for the selected line.

Next Bookmark. Automatically moves the cursor to the next bookmarked line.

Previous Bookmark. Automatically moves the cursor to the previous bookmarked line.

Clear All Bookmarks. Clears all bookmarks from the active document.

Indent to Next. Changes the indent of the current line to match that of the next line.

Indent to Previous. Changes the indent of the current line to match that of the previous line.

List Paragraphs. Lets you see the paragraph name listings for the active file.

List Variables. Lets you see the variable name listings for the active file.

List Constants. Lets you see all the level-78 constant names for the active file.

Open COPY File. Opens the COPY file selected in the source code.

Verb Block Match. Lets you see the opposite end of a verb block command statement.

Find Scope. Automatically selects the entire effective range of a selected variable.
