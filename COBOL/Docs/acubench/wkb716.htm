7.3.2.7 Removing a value or comment from the configuration file

The CFE allows you to remove unwanted configuration values and comments from your configuration file. You cannot remove pre-defined variables from the "Configuration file entries" tree view. You can, however, remove user-defined variables.

The CFE uses the Delete key for deleting values and comments.

  1. In the "Configuration file entries" box, locate and click on the variable that you want to remove from your file.
  2. Click Delete.
  3. Click OK to save and update the configuration file and close the CFE.
