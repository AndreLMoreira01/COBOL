7.6.13 Miscellaneous Options

The Miscellaneous options allow you to set a variety of compiler functions. Among them, you can specify a location for placing data dictionary files (XFD files). Set Miscellaneous options as follows:

  1. Open the Project Options or Program Compile Options dialog box and select Miscellaneous Options from the Catalog drop-down list box.
  2. Select the desired options.
  3. To include data dictionary files in the project, select the corresponding check box, click the Browse button, and follow the normal procedures for specifying a directory location.
