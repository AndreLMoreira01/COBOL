21.8 Screen Designer Pop-up Menu

Right-clicking anywhere in a Screen Designer window creates a pop-up menu containing commonly used editing and navigation commands. Right-clicking on some controls provides you with commands that are unique to that control.

Menu Item Description

Undo

Reverses your most recent editing action.

Redo

Reverses your most recent undo action

Cut

Removes the selection from the screen and places it on the clipboard.

Copy

Places a copy of the selection on the clipboard.

Paste

Places a selection that has been cut or copied to the clipboard into the active screen.

Delete

Removes the selection from the active Screen Designer window.

Move Up

Available for the Menu Designer and Tree View Designer. Moves the selected entry up one position.

Move Down

Available for the Menu Designer and Tree View Designer. Moves the selected entry down one position.

Insert Item Before

Available for the Menu Designer and Tree View Designer. Lets you insert an item before the selected entry.

Insert Sub-item

Available for the Menu Designer and Tree View Designer. Lets you insert a sub-item after the selected entry.

Insert Separator

Available for the Menu Designer and the Tree View Designer. Lets you insert a separator before the selected entry.

Select All

Selects all the elements in the active Screen Designer window. Not available for the Menu Designer or Tree View Designer.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Tab Order

Lets you determine the order in which the cursor visits controls as the user tabs through them on the screen. Not available for the Menu Designer or Tree View Designer.

Send to Back

Gives the selected control the lowest tab order number, reordering all other controls. Not available for the Menu Designer or Tree View Designer.

Send to Front

Gives the selected control the highest tab order number, reordering all other controls. Not available for the Menu Designer or Tree View Designer.

Component Toolbox

Toggles the display of the Screen Component Toolbox.

Drag-and-Drop

Toggles the display of the Drag and Drop window.

Custom Color

Lets you customize the Screen Designer's color palette. Not available for the Menu Designer or Tree View Designer.

Change Prefix

Lets you change the active screen's unique prefix. Not available for the Menu Designer or Tree View Designer.

Lock Controls

Lets you lock or unlock a control's position on the screen.

Generate STF Document

Lets you generate the ".stf" file for the active screen. Not available for the Menu Designer or Tree View Designer.

Property Window

Lets you display or hide the Property window.

Insert New Page

Available for the tab control. Lets you add another page to the selected tab control.

Next Page

Available for the tab control. Lets you move to the tab control's next page.

Previous Page

Available for the tab control. Lets you move to the tab control's previous page.

ActiveX Control Properties

Available for ActiveX controls. Generates a Properties dialog box for the selected ActiveX control.
