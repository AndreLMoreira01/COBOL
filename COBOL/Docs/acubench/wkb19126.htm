19.9.12.3 Options/Environment/Version Control

A description of the Version Control options appears in the following table. For a complete description of the Version Control interface, see Chapter 5.

Option Description

Delete ("X") button

Clicking this button causes the selected item to be deleted.

New (folder icon) button

Clicking this button opens a new entry in the list.

Move up (up arrow) button

Clicking this button causes the selected item to move up in the list order.

Move down (down arrow) button

Clicking this button causes the selected item to move down in the list order.

Menu command list

This is a working list of version control commands. You can change the definition of a command on this list by selecting it and then changing the values in the "Command", "Arguments", and "Working directory" entry fields. Commands in this list are displayed on the Version Control pop-up menu. Click the New button to add a command to this list.

Menu contents

Lists the names of commands that show up on the Version Control pop-up menu.

Associated command

Specifies the associated command, if any.

Fundamental source control commands

This area contains a list of fundamental commands.

Command

For each command, enter the name of the version control system executable in this field.

Arguments

For each command, enter the version control system command line arguments in this field.

Working directory

For each command, enter the name of the version control system working directory in this field.

Use Output window

For each command, enable this check box to display output from the command in the Version Control tab of the Output window.

Ask argument

For each command, enable this check box to prompt for additional command arguments before executing the command.
