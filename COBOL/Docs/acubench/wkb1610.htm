16.3 Creating Data Layout Files From Existing ".fd" and ".sl" Files

You can create a data layout file from existing file description and SELECT statement COPY files. If you do not already have ".fd" and ".sl" COPY files, they are easily created by extracting the code from your program and placing the file description and SELECT statement code in separate files. The primary advantages of doing this are:

  • You can use the File Designer to maintain the file description and SELECT statement.
  • You can designate the ".fd" and ".sl" files as a data set for the program (via the Data Set Designer), which gives the Screen Designer visibility into the file and causes standard file handling code to be generated (see section 16.4).

Use the following steps to create a data layout file from existing ".fd" and ".sl" files:

  1. Select New from the File menu and click on the FD/SL tab.
  2. Select the Import from Files option and complete the dialog.
  3. Click OK. You are prompted for the names of the ".fd" and ".sl" files. Input the names or use the browse button to find the files, and click OK. The imported FD/SL file pair can only contain the file description (".fd") and SELECT statement (".sl').

A data layout file is created and you can now use the File Designer to view or modify the file structure.
