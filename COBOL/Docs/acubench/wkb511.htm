5.6 Command Variables

The following list describes the AcuBench-defined variables that you can include in the "Arguments" and "Working directory" fields. To insert a variable, click on the right-arrow button to the right of the entry field and select the desired variable from the list. AcuBench assigns values to these variables based on the state of your current workspace.

File Path {FilePath}

This variable contains the full path and name of the currently selected file or folder.

File Directory {FileDir}

This variable contains the full path to the directory of the currently selected file.

File Name {FileName}

This variable contains only the name of the currently selected file.

Current Directory {CurDir}

This variable contains the path and name of the directory in which the AcuBench application is located.

Target Directory {TargetDir}

This variable contains the path to the target project or folder. When source files are checked out, they are stored in the target directory until you check the files back in.

Workspace Path {WkspPath}

This variable contains the full pathname and file name of the current workspace.

Workspace Name {WkspName}

This variable contains only the name of the current workspace.

Project Directory {ProjectDir}

This variable contains the full path to the current project, excluding the name of the project file.

Project Name {ProjectName}

This variable contains the name of the current project.

Folder Name {FolderName}

This variable contains the name of the current folder.

Comment {Comment}

This variable includes a comment defined by your version control system.
