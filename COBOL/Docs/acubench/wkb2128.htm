21.10.3 Key List Definition Area Pop-up Menu

Right-clicking anywhere in the key list definition area of the Key tab of the File Designer creates a pop-up menu containing the following commonly used commands:

Menu Item Description

Syntax Check

Runs the compiler to check the key list for correct syntax.

Add

Lets you add a key to the Key List definition area.

Undo

Reverses your most recent editing action.

Redo

Reverses your most recent undo action.

Modify

Lets you modify the selected key in the Key List definition area.

Delete

Lets you delete the selected key.

Delete All

Lets you delete all the keys in the Key List definition area.

Select All

Lets you select all the keys in the Key List definition area.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
