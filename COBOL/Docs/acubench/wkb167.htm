16.2.4 The Key Screen

Keys are selected from the "Fields description" list. Selected items are moved on and off of the "Selected fields" list with the arrow buttons.

You can specify a key for an indexed file as follows:

  1. In the "Fields description" list, select the field or fields to add to the "Selected Fields" list and click on the right arrow (">"). Note that all of the fields in the "Selected Fields" list make up the new key. To place all fields on the list, click the ">>" button.
  2. To add the fields in the "Selected Fields" list as a key, click the New button (first in the set of four to the right of the "Key list" label). The selected fields appear in the "Key list."
  3. To specify a key name or to change the key type, click in the corresponding field. To insert a comment, click in the comment field and then click on the button that appears.

Use the following steps to change a key for an indexed file:

  1. In the "Key list" select the key that you want to change. Note that the key fields of the selected key are listed in the "Selected Fields" list.
  2. Add or subtract items from the "Selected Fields" list until you have the fields that you want.
  3. Change the key by clicking the Modify button (second in the set of four to the right of the "Key list" label).

Key Screen Options

Option Description

Key List

A list of keys. There are four buttons in the upper right corner of the list.

Add button

This button creates a new key using the items in the "Selected Fields" list.

Modify button

This button replaces the selected item in the Key list with the items in the "Selected Fields" list.

Delete button

This button deletes the selected item in the Key list.

Delete All button  

This button deletes the entire Key list.

Key Name

Double-click in the field to enter a key name.

Type

Double-click in the field and select a key type from the drop-down list.

Fields

This field lists the record item names of all items in the key.

Comment

Double-click in the field and then click on the button to open a dialog that allows you to enter a comment for the key. The comment is placed before the key definition in the ".fd" file.

Fields description

The "Fields Description" area is composed of a list of all fields (the list on the left) and a list of selected fields.

Selected Fields list

Lists all currently selected fields.
