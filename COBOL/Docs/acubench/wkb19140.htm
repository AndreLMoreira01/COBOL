19.9.12.17 Options/Code Editor/COPY File List

A description of the Code Editor's COPY File List options appears in the following table:

Option Description

Sort by

Lets you determine how the COPY File List box is sorted.

Occurrence in the program

Causes COPY file names to be sorted in order of their occurrence in the program.

Alphabetically

Causes COPY file names to be sorted alphabetically.

Directory

Causes COPY file names to be sorted alphabetically by directory.

Declared file

Causes COPY file names to be sorted based on the names of the files in which they are declared.

Show

Lets you determine what columns of information are displayed with COPY file names.

Directory

Generates a column in the COPY File List box that contains each COPY file's directory.

Declared file

Generates a column in the COPY File List box that contains the name of the file in which each COPY file is declared.
