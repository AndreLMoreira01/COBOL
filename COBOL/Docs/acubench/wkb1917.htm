19.1.15 Print

Select File/Print to print your active file. To print the current file, adjust the default dialog settings as desired and click OK. The Print dialog options include:

Option Description

Name

Select the name of the printer you want to use.

Properties

Click Properties to change options for the selected printer. The options available depend on the printer selected.

Print to file

Check this check box if you want to print your work to a file instead of a printer.

Page range

Define the pages you want to print.

 

All prints all of the pages in the active file.

In Pages, type the page numbers and/or page ranges you want to print.

If you selected text in your source code before choosing the Print command, the Selection radio button is automatically enabled, and only your selected text is printed.

Number of copies

Define the number of copies you want to print. The default is 1.

OK

Click OK to print your file.

Cancel

Click Cancel to exit the Print dialog without printing your file.
