9.3.18 Previewing the Active File Before Printing

The workbench allows you to preview a file before you print it. You can also preview a copy of the Control Flow Diagram (CFD) for the active file.

To preview a file before printing, choose the File/Print Preview command. The Print Preview screen appears. Use the following Print Preview screen options to view the file:

Option Description

Print

Lets you print the document. When you click this button, the Print dialog box appears.

Next Page

Sends you to the next page in a multiple-page document. This button is disabled for single-page documents.

Prev Page

Sends you to the previous page in a multiple-page document. This button is disabled for single-page documents.

One Page/
Two Page

Lets you view one or two pages at a time of your multiple-page document. For example, when the button displays "One Page", click it to see a single page of your document.

Zoom In

                                    

Magnifies your view of the document. Print Preview has three zoom levels: at the minimum level of magnification, click the button once for medium magnification; click a second time to go to maximum magnification.

Zoom Out

Reduces your view of the document. Print Preview has three zoom levels: at the maximum level of magnification, click the button once for medium magnification; click a second time to go to minimum magnification.

Close

Closes Print Preview.
