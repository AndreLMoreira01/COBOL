19.4.3 Settings

Select Project/Settings to review or modify the settings for your project. Project settings allow you to specify compiler, runtime, environment, and library options for the project. The options available on the Project Setting dialog depend on the tab you select.

Compiler tab

provides access to the compiler option settings

Runtime tab

provides access to the runtime option settings

Environment tab

allows for the addition, modification, and removal of environment variables

Library tab

allows for the creation of object libraries, as well as additions, modifications, and removals of modules in pre-existing object libraries

For more information regarding the use of this dialog, see section 7.1.2, "The Project Settings Dialog."
