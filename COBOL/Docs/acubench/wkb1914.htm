19.1.12 Page Setup

Select File/Page Setup to define the look of your printed page. The Page Setup dialog options include:

Option Description

Header

Allows you to define a header (if desired) that prints at the top of your page. Click the arrow to the right of the Header field to select the items you want to include or design in your header.

 

Full Path Filename prints the full path and file name in the header. The code displayed in the Header field when this is selected is &U.

Filename prints the file name in the header. The code displayed in the Header field when this is selected is &F.

Page Number prints the page number in the header. The code displayed in the Header field when this is selected is &P.

Current Time prints the time the document was printed in the header. The code displayed in the Header field when this is selected is &T.

Current Date prints the date the document was printed in the header. The code displayed in the Header field when this is selected is &D.

Left Align prints the header in a left alignment. The code displayed in the Header field when this is selected is &L.

Center prints the header in a center alignment. The code displayed in the Header field when this is selected is &C.

Right Align prints the header in a right alignment. The code displayed in the Header field when this is selected is &R.

Footer

Allows you to define a footer (if desired) that prints at the bottom of your page. Click the arrow to the right of the Footer field to select the items you want to include or design in your footer.

 

Full Path Filename prints the full path and file name in the footer. The code displayed in the Footer field when this is selected is &U.

Filename prints the file name in the footer. The code displayed in the Footer field when this is selected is &F.

Page Number prints the page number in the footer. The code displayed in the Footer field when this is selected is &P.

Current Time prints the time the document was printed in the footer. The code displayed in the Footer field when this is selected is &T.

Current Date prints the date the document was printed in the footer. The code displayed in the Footer field when this is selected is &D.

Left Align prints the footer in a left alignment. The code displayed in the Footer field when this is selected is &L.

Center prints the footer in a center alignment. The code displayed in the Footer field when this is selected is &C.

Right Align prints the footer in a right alignment. The code displayed in the Footer field when this is selected is &R.

Style

Select a style (if desired) from the drop-down list.

 

Black and White prints your document in black and white.

Color prints your document in color.

Zebra prints your document using alternating gray and white stripes.

Print line number

Check this check box if you want the line numbers of your file to print on your document.

Frame

Select a type of frame to print on your document.

 

None prints no frame

Inner prints an inner frame

Margins

Allows you to specify margin settings for your printed document. Set the left, right, top, and bottom margins in measurements of inches.

OK

Click OK to set your page according to the settings defined in the Page Setup dialog.

Cancel

Click Cancel to exit the Page Setup dialog without saving your settings.
