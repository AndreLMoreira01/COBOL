7.3 Using the Configuration File Editor

An integrated Configuration File Editor (CFE) allows you to open and edit any ACUCOBOL-GT runtime configuration file. With the CFE, you no longer need to access the host system's text editor in order to modify a configuration variable. Instead, you can modify the variable from within the workbench.

The CFE also allows you to add user-defined configuration variables. These variables can then be modified within the CFE like any other configuration variable.

As noted earlier, the CFE is accessed from the Project/Settings/Runtime dialog. With Common Options, the Edit button adjacent to the "Alternate configuration file name (-c)" check box launches the CFE and opens the specified configuration file. You can also access the CFE from Thin Client Options to edit your remote configuration file. You can edit any configuration file that is named in the entry field. Be careful, however, to restore the value of the entry field to the name of the configuration file that you want to use when you run the associated program.

Currently the CFE has one notable limitation. Although it can be used to add or remove a configuration variable, it cannot be used to "comment out" or "uncomment" a variable (place or remove a "#" in front of a variable).
