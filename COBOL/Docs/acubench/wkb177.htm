17.3 Using Project Management Capabilities to Create Object Libraries

In addition to the cblutil utility accessed via the Tools drop-down menu, the object library creation function is integrated into the workbench project management interface via the Project/Settings and Project/Properties commands. In this case, AcuBench calls cblutil during a build or rebuild process to perform the library creation. Note that the library creation process is performed every time the workspace is built.

This section contains procedures for creating an object library using the Project/Settings and Project/Properties commands. You should note that the library creation utility accessed from the Project drop-down menu does not include an option to remove an object module from a library. We recommend that you create object libraries only after you have debugged all of the programs. Of course, you can rebuild an object library at any time.
