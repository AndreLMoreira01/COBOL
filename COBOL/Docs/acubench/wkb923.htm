9.3.12 Working with Constant Names

The Code Editor maintains a list of the level-78 constants currently declared in your program. You can view the Constant List box via the Edit/Advanced/List Constants command or the List Constants push button in the Editor toolbar. You can also access this list by right-clicking anywhere in the Code Editor window and choosing List Constants.

The Constant List box is a modeless window that allows you to work in the Code Editor window while the list is displayed. In addition to the name of each constant, the list box contains columns showing each constant's value and the COPY file in which it appears. You can control which columns are visible and how the constants are sorted via the Tools/Options/Code Editor/Constant List interface, which is described in section 4.5.8, "Setting Constant List Options." You can also determine the sort order by clicking on the column headings in the list box.

You can easily insert into your code any constant name that appears in the Constant List box. Position the cursor in your code where you want the constant to appear. In the Constant List box, select a name, right-click in the list box, and choose Paste. Note that you can change the cursor's position in the code and paste a constant name multiple times. You can also double-click a name to insert it at the cursor's location.

Right-clicking in the list box gives you access to other functions in addition to Paste. The Copy to Clipboard command lets you copy a constant name to the Windows clipboard. You can move the cursor automatically to the declaration of a constant in your program via the Go to Definition command. Finally, if you modify your code while the list box is open, you can update the Constant List by using the Refresh command.
