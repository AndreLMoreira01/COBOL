8.1.1 Title Bar

The title bar displays the project name, program name, and file name of the active file. The Minimize, Maximize, and Close buttons appear at the right end of the bar.

You can double-click on the title bar of a Code Editor window to maximize the window within the workspace. When you maximize an editor window in the workbench, the Minimize, Maximize, and Close buttoms for that window appear just below corresponding buttons for the entire AcuBench workspace. Title bar information for the maximized editor window appears between square brackets ("[]") on the AcuBench title bar.
