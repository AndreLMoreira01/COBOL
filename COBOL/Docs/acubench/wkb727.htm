7.6.5 Source Options

The Source options modify the way the compiler treats the physical source files. Set source options as follows:

  1. Open the Project Settings or Program Compile Options dialog box and select Source Options from the Catalog drop-down list.
  2. Enable the desired options.
  3. If you enable the "Include source lines with pattern..." or "Exclude source lines with pattern..." option, enter the respective name in the field.
  4. If you enable the "Tab stops every number of columns" option, select a value in the spin box.
  5. To specify a directory to search for COPY files, select Directories to search for COPY. You can enter the path directly, or click the Browse button and follow the standard procedures for selecting a path.

Note: By default, AcuBench uses the value of the environment variable COPYPATH to locate COPY files. If you enable the "Directories to search for COPY (-Sp)" option, AcuBench ignores the value of COPYPATH. Should the auto-parsing process fail to locate a COPY file, you should check to see if the "-Sp" option is enabled. If it is, check to see that the COPY file is located in the specified directory. If "-Sp" is not used, you should check the value of COPYPATH.
