8.1.2 Ruler Bar

The ruler bar appears at the top of each Code Editor window. It shows the current document's column numbers and tab settings. You can display or hide the ruler bar by choosing the View/Ruler Bar command.

Column numbers are used in the Tools/Options/Code Editor/Format interface to define the area occupied by each editing area component. Tabs are used to move quickly between columns in text. You can set an interval for your tab stops or up to 32 custom tab stops in the Tools/Options/Code Editor/Tabs interface. Tabs may also be used to move blocks of text in your code. Highlight the lines of code that you want to move and press the Tab key. The entire block of text shifts to the next tab stop. More information about the Tools/Options dialog can be found in Chapter 4 of this manual.

If tab settings exist for your Code Editor window, you can press Ctrl+Tab to move the cursor from tab stop to tab stop without affecting the text on the line. If you have not established your own custom tab stops in the Tools/Options interface, Ctrl+Tab moves you to columns 7, 8, 12, 73, and 80, the start of each ANSI-defined display area.
