17.2.1 Creating an Object Library

  1. To create an object library using the ACUCOBOL-GT cblutil object file utility, select the Tools/Cblutl command. The Cblutl32 dialog box appears.Select the Library tab (equivalent to the ACUCOBOL-GT "cblutil -lib" command).
  2. Set the desired options in the Parameters section. Specify a name for your object library in the entry field to the right of the "Name of object library (-o)" check box.
  3. Specify the modules you want your object library to contain in the "Additional Modules" entry box. Use the New and Delete buttons located at the upper right of this box to add and delete modules. When you press the New button, you can use the browse button that appears to the right of the blank entry to locate the desired file. Note that the first module named is your primary module.
  4. Use the Up and Down arrow buttons to re-order the modules.
  5. The "Options" entry field displays the command line that reflects the options chosen. You cannot directly edit the contents of this entry field.
  6. Click Apply to create your object library. If the "Name of object library (-o)" entry field contains a path name, that is where the object library is created. If that entry field is blank, the library is created in the same directory as the primary module.
  7. After you click Apply, information about your created object library appears in the box located below the "Options" entry field.
