19.1.2 Open

Select File/Open to open an existing file. Locate the desired file and click Open to open the file.

The Open dialog options include:

Option Description

Look in

Locate and select the file you want to open. The drop-down list allows you to select the directory or folder that contains the file, and the files included in the selected directory or folder are listed in the box below.

File name

Displays the name of the file to be opened. This field is populated using the file name you select from the box above, or you can type the name of the file (and the full directory path if necessary). You can also use the "*" wildcard character to help define the file you want to open.

Files of type

Allows you to select a specific file type to open. The default file extension for source files is ".cbl".

Open

Click Open to open the selected file.

Cancel

Click Cancel to exit the Open dialog without opening a file.
