13.5 The Drag and Drop Window

The Drag and Drop interface contains Field, PIC, and Occurs columns, which describe your data items. Select the report control type that want to create in the drop-down box, then select the data item(s) you want to associate with the control. Click on the selected data item(s) and drag the selection to the desired report section. The report controls are drawn in the Report Composer window, and you can arrange them by dragging them, using the arrow keys, or using the Align menu commands, as with Screen Designer controls.

How your data looks and behaves depends on which report control you choose from the drop-down list. For example, when a variable is dragged to the Report Composer window, the data name is transferred to the report label title field, and the field name is associated with the value variable of a report entry field. Another drop-down box lets you display only those data items in the Working-Storage or Linkage sections.
