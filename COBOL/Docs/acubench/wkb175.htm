17.2.2 Displaying Object Library Information

  1. To display information about an object file (equivalent to the ACUCOBOL-GT "cblutil -info" command), select the Tools/Cblutl command. The Cblutl32 dialog box appears.Select the Information tab.
  2. Specify the module(s) for which you want to display information in the "Additional Modules" entry box. Use the New and Delete buttons located at the upper right of this box to add and delete modules in the box. When you press the New button, you can use the browse button that appears to the right of the blank entry to locate the desired file.
  3. Use the Up and Down arrow buttons to re-order the modules.
  4. The "Options" entry field displays the command line for the information function. You cannot directly edit the contents of this entry field.
  5. Click Apply to activate the information function. Information from the header of the selected object file appears in the box located below the "Options" entry field.
