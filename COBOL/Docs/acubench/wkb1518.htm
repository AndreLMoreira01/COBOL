15.4.8 The Report Label

The report label corresponds to the ACUCOBOL-GT standard label control. You add a report label to a report via the Report Component Toolbox. As in the Screen Designer, you click the label icon in the toolbox and position the label in the desired location in the report. The report label Property window contains the following control-specific properties in addition to the properties described in section 15.4.1, "Common Report Control Properties."

Hyperlink

Use this property to designate your report control as a hyperlink when it is displayed in a Web browser. A hyperlink may reference either a local HTML file or a Web site.

To reference a local HTML file, click the ellipsis push button in the Value cell to display the Open dialog box. Navigate to the location of the local file and select it.

To reference a Web site, enter the entire address in the Value cell (e.g., http://www.acucorp.com). If you omit the "http://" prefix, AcuBench interprets the entry as a file name. In the latter case, the result is a "File not found" error message when you attempt to access the link.

This property has no default value.

Hyperlink Variable

The Value cell for this property has an ellipsis push button that opens the Select Variable interface. If a variable is selected in this dialog, subtle modifications to the generated code occur, such that it references the Hyperlink Variable instead of the Hyperlink value directly. This property has no default value.

Justification

A drop-down box allows the selection of "Center", "Left", "Right", or "Unaligned" justification for the report label. The default value for this property is "Unaligned".

Title

This property determines the text portion of the report label. The title of a report label can also be modified by double-clicking the label and typing directly in the Report Composer window or by changing the default title in the Property window.

Title Variable

The Value cell for this property has an ellipsis push button that opens the Select Variable interface. If a variable is selected in this dialog, subtle modifications to the generated code occur, such that it references the Title Variable, instead of the Title value directly. This property has no default value.
