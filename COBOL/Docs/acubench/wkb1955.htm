19.4.2 New Folder

Select Project/New Folder to create a new folder for your project. The New Folder dialog options include:

Option Description

Folder name

Enter the name of the new folder.

File extensions

Specify the file types (by extension) that you want to store in this folder. Use an asterisk ("*") as a wildcard character, and separate multiple file extensions with semi-colons (";"). For example, if you want a folder specifically for bitmap graphics and icons, enter "*.bmp; *.ico" in this field.

Location

Enter the location where files associated with this folder will reside. Click the Browse button ("...") to the right of the field to navigate the file system to select a location. If an element in the specified path does not exist, it is created automatically.

Note that like other project folders, only new files created in the workbench, or files that you physically move to the folder actually reside in the folder. All other files remain in their original location and are located via a logical link.

Show full path name

Enable this check box if you want to display the full path names of files located in this folder.

OK

Click OK to create the folder and exit the dialog.

Cancel

Click Cancel to exit the dialog without creating the folder.
