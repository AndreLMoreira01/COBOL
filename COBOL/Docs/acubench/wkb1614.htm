16.4.3 Creating a BEFORE or AFTER Procedure

You can use the AcuBench Event Editor to code and include BEFORE and AFTER procedures in the program's ".evt" COPY file. Launch the Event Editor from the Data Set Designer as follows:

  1. In the Structure view of the Workspace window, open the desired Program node and double-click on the Data Set node. The Data Set Designer window is opened and an icon is displayed for each data set.
  2. Select the desired data set icon.
  3. Display the Property window by selecting Property Window from the View menu. Click on the Event tab.
  4. In the Event tab, locate the type of procedure that you want to add in the Item list and click in the adjacent Value field. Click on the ellipsis (...) button, accept or modify the name in the Add Paragraph dialog, and click OK. The program's ".evt" file is opened in the Event Editor and a new paragraph is inserted. Code the rest of the procedure.
