7.7.1 Adding New Environment Variables

In the Environment tab of the Project Settings dialog, you can add an environment variable as follows:

  1. In the workspace window, right-click on the desired project node and select Settings from the pop-up menu. The Project Settings dialog is displayed.
  2. Click on the Environment tab.

    3. Note that the first item in the list of variables is always selected by default. Also, the selected variable's name and value are displayed in the "Variable" and "Value" fields at the bottom of the tab.

  3. To add an environment variable, click the New button (the leftmost button of the three buttons located in the top-right corner of the "Environment Variables" box). The "Variable" and "Value" fields are cleared.
  4. Enter the variable name in the "Variable" field.
  5. Enter a value in the "Value" field.
  6. Click the Set button. (Set is the rightmost button in the top-right corner of the "Environment Variables" box, the red check mark.) The new environment variable is added to the list and the "Variable" and "Value" fields are cleared.
  7. Click OK to close the Project Settings dialog.
