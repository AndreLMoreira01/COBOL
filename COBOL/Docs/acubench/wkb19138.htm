19.9.12.15 Options/Code Editor/Variable List

A description of the Code Editor's Variable List options appears in the following table:

Option Description

Sort by

Lets you determine how the Variable List box is sorted.

Occurrence in the program

Causes variable names to be sorted in order of their occurrence in the program.

Alphabetically

Causes variable names to be sorted alphabetically.

Level number

Causes variable names to be sorted by level number.

Picture description

Causes variable names to be sorted by picture clause.

Show

Lets you determine what columns of information are displayed with variable names.

Level number

Generates a column in the Variable List box that contains each variable's level number.

Picture description

Generates a column in the Variable List box that contains each variable's picture clause.

Name of the COPY book

Generates a column in the Variable List box that contains the name of the COPY file in which each variable appears.

Include FILLER data item

Lets you determine whether blank FILLER data items are listed in the Variable List box.
