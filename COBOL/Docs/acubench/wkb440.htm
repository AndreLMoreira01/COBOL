4.6.2 Setting Default Options

The Screen Designer's Default options let you determine default control property values for any properties that can have default values. Location properties, for example, cannot have default values, because those values would impair your ability to position individual controls anywhere in the designer window. The default values that you set for a particular control automatically appear in the Property window when you add that control to your screen.

You can set control property default values as follows:

  1. Select the Default category of the Screen Designer options.
  2. To set a default property value, select a control in the list box on the left side of the dialog, then set the desired property value in the list on the right. The result of any change to a control's defaults is reflected in the Property window each time you create that type of control.

    3. Changes to the default properties do not affect exising screens or controls, whose properties have already been established and generated into code.
