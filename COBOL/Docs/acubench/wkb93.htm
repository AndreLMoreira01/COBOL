9.1.1 Moving Through Text

The following default keyboard commands move you through your source code:

To move (to): Default keyboard command

Forward one character

Right arrow

Back one character

Left arrow

Forward one word

Ctrl+Right arrow

Back one word

Ctrl+Left arrow

End of the current line

End

Start of the current line

Home

Previous line

Up arrow

Next line

Down arrow

Previous page

Page Up

Next page

Page Down

Top of the file

Ctrl+Home

Bottom of the file

Ctrl+End

To change these default keyboard commands or establish other keyboard shortcuts, use the Tools/Options/Environment/Keyboard dialog box, described in section 4.4.6, "Setting Keyboard Options."

The Code Editor also has an auto-indent function, enabled from the Tools/Options/Code Editor/Keyword interface. This dialog box provides some options for auto-indent behavior, including keyword indents that you customize. You can direct that the cursor return to column 1 of the next line or to the location of the first non-blank character in the previous line. More information about the auto-indent function in the Tools/Options dialog can be found in section 4.5.4, "Setting Keyword Options."
