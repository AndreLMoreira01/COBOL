17.8 Using acu4glfd

A Windows utility called acu4glfd helps you create file descriptions (".fd") and SELECTS (".sl"). Because the utility uses ODBC technology to obtain table information in order to list table fields, you need to have Acucorp's Acu4GL product installed for the runtime to access the table. It can also determine a unique index, which it uses as the primary key.

You access this utility's interface via the Tools/acu4glfd command. You need to choose and connect to a data source. When the tables contained in the data source are displayed, users can direct the utility to create FDs and SELECTs.

More information about this utility can be found in section 6.2 in the Acu4GL User's Guide.
