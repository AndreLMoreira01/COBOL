19.2.10 Find in Files

Lets you locate text strings in files stored on your drives. When you choose this command, the Find in Files dialog box appears. The Find in Files options include:

Option Description

Find what

If you selected text before choosing the Find in Files command, displays that text string. You can also enter the text string you want to find, or select from a list of up to 15 text strings previously entered for the Find operation.

File types

Lets you select the type of file you want to find.

Options

These choices help you focus your search.

Match case

Causes the search operation to be case-sensitive.

Regular expression

Lets you enter a regular expression as the item to be found rather than a specific text string.

Where

These options let you determine the location of your search.

Search in workspace

Lets you direct that the search occur in your workspace.

Search in directories

Lets you direct that the search occur in your directories.

Search directory options

These options let you focus your directory search.

Look in

Lets you designate a directory location for the search.

Browse

Lets you navigate to a location for the search.

Include subdirectories

Lets you designate whether the search should include subdirectories.

Find

Begins the search and lists search results in the "Find in Files" Output window pane.

Cancel

Click Cancel to exit the Find in Files dialog.
