19.9.12 Options

The Tools/Options dialog is used to configure many of AcuBench's customizable capabilities and user interface options. The dialog organizes options into the following primary categories:

  • Environment
  • Code Editor
  • Screen Designer
  • Code Generator
  • Data Designer
  • Report Writer

These categories are displayed and selected in the tree view that is located on the left side of the dialog. Each category's options are presented on the right side of the dialog. Most categories contain two or more sub-categories that are displayed when you open (click on) the main node.

The Save, Load, and OK functions are always available at the bottom of the dialog. The Save capability allows you to change any default settings and save them to another file. The Load function allows you to load previously saved settings into the current workspace. OK saves the settings to the default "AcuBench70.ini" file.
