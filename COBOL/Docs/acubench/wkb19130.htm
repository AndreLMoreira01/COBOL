19.9.12.7 Options/Environment/Prefix

A description of the Prefix options appears in the following table:

Option Description

Default prefix applied to new documents

Each field in this group defines the prefix file name string assigned when a new document of that type is created. To change the default prefix, enter a new string in the entry field.

Project prefix

The prefix string applied to the name of a new project

Program prefix

The prefix string applied to the name of a new program

Screen prefix

The prefix string applied to the name of a new screen

Report prefix

The prefix string applied to the name of a new report

File prefix

The prefix string applied to the name of a new file

FD/SL prefix

The prefix string applied to the name of a new data layout file

Working directories

These fields define the default names of the working directories. These names are used when a new project is created.

Source directory

The name of the directory for source files

Screen directory

The name of the directory for screen files

Report directory

The name of the directory for report files

Copylib directory

The name of the directory for COPY files

Object directory

The name of the directory for object files

List directory

The name of the directory for listing files

Resource directory

The name of the directory for resource (audio, bitmap, etc.) files

FD directory

The name of the directory for ".fd" and ".sl" files
