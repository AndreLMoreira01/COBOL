19.9.12.4 Options/Environment/Build

A description of the Build Environment options appears in the following table:

Option Description

When running tools

This option controls workbench behavior immediately prior to parsing or compiling.

Always save all documents

Enable this check box to cause all open files to automatically be saved before a tool is run.

When compiling programs that include generated code

These options affect how the workbench behaves when working with programs that depend on generated code.

Always generate code if program structure file is modified

Enable this check box to cause AcuBench to automatically regenerate code before compiling a program if its program structure file has been modified since the last compile.

Prompt before generating source code

Enable this check box to direct AcuBench to always display a confirmation dialog before generating code.

Jump to Event Editor when browsing the error

Enable this check box to cause the workbench to open the Event Editor when browsing errors found in an event paragraph.

When executing source or object files

These options control workbench behavior immediately prior to executing an object.

Always recompile if original source file is modified

Enable this check box to direct the workbench to always recompile the program source file before executing the program, if the program has been modified.

Prompt before recompiling source file

Enable this check box to cause a confirmation dialog to be displayed before recompilation.

When building or reparsing the workspace

These options control workbench behavior immediately prior to building/rebuilding or reparsing the workspace.

Always generate modified program (.psf) files before compiling source files

Enable this check box to cause the workbench to always regenerate code for a modified program prior to building/rebuilding the workspace.

Always generate modified FD/SL (.dlt) files before compiling source files

Enable this check box to cause the workbench to always generate code for modified data layout files prior to building/rebuilding the workspace.
