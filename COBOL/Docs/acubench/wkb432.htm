4.5.4 Setting Keyword Options

You can add, delete, and modify keyword sets with the Code Editor's Keyword options. Default keyword sets are listed in the Keyword set drop-down list. You can also associate an indent value with a keyword and choose a general auto-indent behavior. The indent value applies to the line following the keyword. When you click Keyword in the Options dialog box, the Keyword options appear.

These keyword lists affect only the workbench's colorization and auto-indent functions. Adding or deleting a keyword in a default list, for example, does not affect the behavior of the ACUCOBOL-GT compiler. The compiler's behavior regarding reserved words is changed only through the use of the "-R" compiler option set. More information about this compiler option set is found in Book 1 of the ACUCOBOL-GT documentation set.

You can add a keyword set as follows:

  1. Click Add to bring up the Add Keyword Set dialog box. Enter the name of the new set in the "Keyword set name" text box.
  2. Select Copy from none to create an original keyword set. Add your new keywords in the Keyword text box using the New push button at the top of the list box.
  3. Select Copy from existing set to base your new keyword set on an existing set. Modify your keyword set using the New and Delete push buttons at the top of the list box.
  4. Select Copy from external set to import a keyword set into the application. Click Browse to select from a list of keyword sets.
  5. Add or modify keyword indent values in the "Indent value" column of the text box.
  6. Click OK to save your current settings.
  7. Click Save As to save the current settings to a file that you specify.

To delete a keyword set, select the set that you want to delete in the "Keyword set" drop-down box. Click Delete.

Add a keyword to an existing keyword set as follows:

  1. In the "Keyword set" drop-down box, select the set to which you want to add a keyword.
  2. Use the New (Insert) push button at the top of the "Keyword" list box to add your new keyword.
  3. Add a keyword indent value if desired in the "Indent value" column of the "Keyword" text box.
  4. Click OK.

You can delete a keyword from an existing keyword set as follows:

  1. Select the word that you want to delete from the "Keyword set" drop-down box.
  2. Click the Delete push button at the top of the "Keyword" text box.
  3. Click OK.

Use the following steps to select an auto-indent function:

  1. If you select None, the cursor returns to column 1 of the next line after a carriage return. Use this function if you are working in non-COBOL source code, such as a configuration file.
  2. If you select First non-blank token, a carriage return sends the cursor to the next line at the location of the first non-blank character in the previous line.
  3. The Customized option allows the Code Editor to apply the keyword indent values shown in the "Keyword" text box.
