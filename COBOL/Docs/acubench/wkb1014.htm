10.6.2 Edit Menu Commands

The following Edit menu commands assist you with screen editing:

Undo. Reverses previous editing actions.

Redo. Reverses the most recent undo operation if you have performed no other actions since the undo.

Cut. Removes the selected control from the active screen and places it on the clipboard.

Copy. Places a copy of the selected control on the clipboard.

Paste. Places a selected control that has been cut or copied to the clipboard into the active screen in the same location that it occupies in the old screen. After pasting a control, you can move it to any location in the screen.

Delete. Removes the selected control from the active screen.

Select All. Selects all the elements in the active screen.

Find in Files. Lets you locate text strings in files stored on your drives.

Find in Objects. Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
