19.4.6 Create Alias

Select Project/Create Alias to define the alias that contains the information needed for a server-based runtime to run a remote program in thin client operations.

The Create Alias dialog options include:

Option Description

Alias Entry Name

This entry field pre-fills with the base name of your source file. You may change this name if desired.

Working Directory

This entry field pre-fills with the name of your remote object directory. You may change this name if desired.

Server

A read-only entry field, whose contents can be changed via the Project/Properties command.

Port

A read-only entry field, whose contents can be changed via the Project/Properties command.

Command Line

Entry is derived from current runtime options. Note that the configuration file component of the command is defined from the runtime's Thin Client Options.

Create

Creates the alias entry.

Cancel

Cancels the creation of the alias entry.
