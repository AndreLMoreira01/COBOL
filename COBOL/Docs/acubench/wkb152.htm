15.1 The Report Property Window

Like the screen controls' Property window, the report Property window contains user-configurable properties and styles for each graphical element in a report, as well for the report itself. This chapter describes these report Property window components. Note that you can set the default values for some report properties in the Tools/Options/Report Writer/Default interface. For information about this dialog, refer to section 4.9.2, "Setting Default Options."
