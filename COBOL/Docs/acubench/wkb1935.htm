19.2.14 Go To

Lets you designate a destination for your cursor in the active document. When you choose this menu item, the Go To submenu appears.

The Go To submenu commands are described in the following table:

Command Description

Begin

Moves the cursor to the beginning of the active Code Editor document.

End

Moves the cursor to the end of the active Code Editor document.

Line

Opens the Go To Line dialog box, which displays the total number of lines in the Code Editor document. You can enter the number of the line where you want the cursor positioned.

Previous Error

Moves the cursor in your source code to the next previous compilation error listed in the Output window.

Next Error

Moves the cursor in your source code to the next compilation error listed in the Output window.
