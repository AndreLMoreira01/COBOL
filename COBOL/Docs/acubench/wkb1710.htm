17.3.3 Generating Native Code Object Modules

From within AcuBench, you can translate a portable object code file into a native code object file. To initiate the process, right-click on the desired object file in the File view of the Workspace window and select the Native Code command from the pop-up menu. The Translate Portable Object Code to Native Code dialog box is displayed.

  1. Select the target processor for which you want to generate native code (Intel or Sparc).
  2. Select the Name of object file (-o) option to save the translated file to a new name. Enter a file name in the entry field.
  3. Select other options as desired.
  4. The "Options" field displays the cblutil command string that reflects the options that you've selected. You cannot directly edit this field.
  5. Click Translate to start the translation process.

Note that native code translation is available for thin client operations. Simply enter your remote file name in the "Name of object file (-o)" entry field.
