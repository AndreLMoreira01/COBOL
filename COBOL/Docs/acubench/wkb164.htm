16.2.1 Opening the File Designer

The File Designer interface consists of five tabbed screens labeled File Control, Definition, Key, IO Handling, and XFD.

To create a new data layout file in the File Designer, select New from the File menu and click on the FD/SL tab. Or, in the Data view of the Workspace window, right-click on the Project node and select New FD/SL from the pop-up menu. Complete the New FD/SL dialog and click OK.

To open an existing data layout file, click on the Data Data viewtab of the Workspace window and double-click on the desired data layout name, or right-click on the project node and select Open data_layout_name from the pop-up menu.
