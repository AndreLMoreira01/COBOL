7.3.2.2 Searching for a string in the configuration file

You can use the Find function to search for a text string in the list of variables. Enter the search string in the "Find entry" entry field, and click the Find button or press Enter. The CFE searches all categories and highlights the first located string. When you are finished, click OK to close the editor and save the file.

Note that the CFE cannot be used to search for values or comments. Also, you must search for the initial portion of the text string in order for the CFE to return a successful search. For example, to search for the string "keyboard", you can search for "key" or "keybo", but you cannot locate this text string by searching for "board".
