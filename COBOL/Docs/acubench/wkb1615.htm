16.4.4 The Data Set's Property Window

The Data Set's Property window is used to define the essential values for the data set. This Property window has the following options:

Option Description

The drop-down list       

Select a data set name from the drop-down list at the top of the window to set that data set's values.

Alphabetic tab

This tab lists the data set's properties in alphabetical order.

(Name)

Enter or modify the name for the data set in this field.

Data Layout

Select a data layout from the drop-down list. Only those data layouts that are defined in the associated project appear on the list.

Key

Select a key for the data layout from the drop-down list.

Referenced Data Set

Click in the Value field and then on the ellipsis button to open the Set Reference-Field dialog (described below).

Categorized tab           

This tab presents the same options as the Alphabetic tab but they are grouped into categories.

Event tab

This tab is the access path to the Event Editor.

Item list

The list of BEFORE and AFTER procedures that are typical for a data set

Value

The name of the actual BEFORE or AFTER procedure for this item. To launch the Event Editor, select a procedure from the list, click in the Value field, and click on the ellipsis button that appears.

Set Reference-Field dialog

Data Set Member Files dialog

The Data Set Member Files dialog lists the data sets defined for the program and is used to specify the OPEN statement that is generated for each data set in the ".prd" COPY file, along with the type of file locking used.

You can also use this dialog's I/O Option section to activate or suppress the generation of I/O statements. Click in the I/O Operation field to display a drop-down box of I/O paragraphs defined for the named file in the File Designer I/O Handling tab. Setting the check box next to an I/O function means AcuBench generates that I/O paragraph. Note that the I/O Handling tab's Use User-defined I/O Paragraphs radio button must be selected to display the I/O paragraphs in the I/O Operation field.
