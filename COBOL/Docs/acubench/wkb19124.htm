19.9.12.1 Options/Environment/General

A description of the General Environment options appears in the following table:

Option Description

Automatic save every x minutes

Enables or disables the automatic save setting and allows you to change the default value from 1 to 120 minutes.

Recently used file list

Allows you to set the number of recently used files, up to eight, that appear on the Files menu.

Recently used workspace list

Allows you to set the number of recently used workspace names, up to eight, that appear on the File menu Recent Workspaces list.

On startup:

This radio button group controls how the workbench behaves when it is launched.

Show empty environment

Causes the workbench to start without loading a workspace.

Show new project dialog

Causes the New Project dialog to be displayed when the workbench is launched.

Reload last workspace

Causes the workbench to reload the last workspace that was open.

Reload documents

Enable this check box to direct AcuBench to reopen the documents that were open the last time the workspace was open.

Default source format

This radio button group specifies the default source file format.

ANSI source format

Causes source files to be treated as ANSI source format.

Terminal source format

Causes source files to be treated as terminal source format.

Translate to ANSI/OEM

Causes AcuBench to translate extended ASCII characters that originate in the OEM environment to Windows ANSI extended characters. Characters are retranslated back to OEM when the document is saved.
