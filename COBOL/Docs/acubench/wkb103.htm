10.2 Working with Multiple Screens

You can have more than one Screen Designer window open at the same time. However, you cannot open more than one version of the same screen form at the same time. The ability to open multiple Screen Designer windows is important when you want to make similar screens or make copies of screens and controls.

Keep the following points in mind when working with multiple Screen Designer windows:

  • No matter how many Screen Designer windows you have open, only one window can be active at a time.
  • To make another window active, just click on it, or switch to it via the Windows menu. This action also changes the Property window to that of the currently active Screen Designer window.
  • You can tile or cascade the Screen Designer windows through the Windows menu.
  • When you copy and paste a control from one screen form to another, the new control inherits many of the properties of the original control. However, it does not inherit unique properties such as ID, Name, Screen Variable, and Tab Order. Instead, default values are assigned for unique properties. After pasting a control, you can change the properties as needed. Refer to section 11.7.1, "Using Edit Commands," for more information about pasting controls.
  • You can make a copy of a screen form by saving it under another name via the File/Save As command.
