14.2.4 Using Report Columns and the Report Date Time Control

In this sample, we modify our report to create a report with columns. The same FD fields are placed in the Report Composer interface that were selected in the first exercise. The report orientation is changed from portrait to landscape to allow adequate room for the columns. The report labels are renamed and positioned in the page header section as column headers. The entry fields are positioned as columns, and the report Lines property value is adjusted. The Border Style property for the entry fields is changed to "No Box" and the report date time control is added to the page footer section.

Set up your project and work environment as described in section 14.2, and use the following instructions to finish creating this report.

Create and refine the data set that corresponds to your data layout:

  1. Create a new blank program called Report1b from the File/New/Program interface.
  2. Set up the SalesdataDS data set for this program, as you have with previous sample reports.

Design your report in the Report Composer interface:

  1. Open a new blank graphical report called Report1b from the File/New/Report interface, or right-click on the report node and select New Report.
  2. Select the View/Drag and Drop command. Using the same procedures outlined for Report 1a, select Sales-State, Sales-Branch-No, Sales-LastName, and Sales-FirstName variables in this interface and drag these four items to the detail section of the Report Composer window.
  3. Change the page orientation from portrait to landscape. Select Report1b in the drop-down box at the top of the Property window. click in the Paper Size property Value cell to display a push button which, when clicked, opens the Paper Size dialog box. Change the orientation of your report from portrait to landscape in this dialog box and click OK.
  4. Modify the report.
    • Select each report label and move it from the detail section to the page header section. Align them horizontally in the page header section to make column headers.
    • The report labels in the page header section contain the text "Sales-State", "Sales-Branch-No", "Sales-LastName", and "Sales-FirstName". Rename each report label by double-clicking in each one and typing "State", "Branch", "Last Name", and "First Name", respectively.
    • Reposition the report entry fields under their corresponding labels to create columns. Note that a report entry field is associated with its Value variable. To confirm the column in which a report entry field should be placed, check the Value variable. It should correspond to the title of the column header in your report.
    • In the Property window, set the Border Style property for each report entry field to "No Box". You can select all the report entry fields by holding the Shift key down while selecting them. The Property window change is then applied to all of the selected report entry fields.
  5. Manipulate the section settings.
    • Adjust the Lines value of the page header and detail sections. Select the page header section in the Report Composer window. In the Property window, change the Lines value to 0.3, a measurement in inches. If this setting obscures the headings you created, either increase the Lines value or move your heading up higher in the page header section.
    • Select the detail section and change the Property window Lines value for the detail section to "0.3". If this setting obscures your report entry fields, either increase the Lines value or move your report entry fields up higher in the detail section.
  6. Add a report date time control to the page footer section. In the Report Component Toolbox, click the report date time control icon. When you move the cursor to the page footer section of the Report Composer interface, it appears as cross hairs. Hold the left mouse button down and draw the report control by moving the mouse. In the report date time control Property window, open the Date Format property's drop-down box in the Value cell and select MM/DD/YYYY. Select Center in the Justification property's drop-down box. Change the Border Style property to "No Box". Note that the Time Format property setting of "None" suppresses the time portion of this control.
  7. Generate the program. A report (".rpt") file and a Report Composer Section in the source (".cbl") file are created.

Make one-time modifications to the generated code:

Use the instructions for code modifications described for sample Report 1a. Note that the PERFORM . . . DO-PRINT-RTN statement is automatically generated by AcuBench.

Compile and execute the program.
