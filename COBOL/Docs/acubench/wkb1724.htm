17.6.9 Unloading to Other File Types

The vutil unload function is accessed on the Unload File tab. The unload function creates a binary sequential or line sequential file from the specified Vision indexed file. For a complete description of the vutil unload function, see section 3.3 of the ACUCOBOL-GT User's Guide. Create a line sequential or binary sequential file from an existing Vision file as follows:

  1. Specify the source Vision file in the "Source file" entry box, and specify the destination file in the "Destination" entry box. You must specify file names to enable the Start button. You can use the ellipsis button adjacent to the entry field to navigate to a file. If during your use of vutil you have already specified the file, you can easily select the file name again from the "Source file" or "Destination" drop-down list.
  2. In the Parameters area, select the desired options. The "Option" field displays the command line that reflects the options chosen. You cannot directly edit the contents of this field. For a complete description of unload file options, see section 3.3 of the ACUCOBOL-GT User's Guide.
  3. To unload the Vision file and create the new binary or line sequential file, click the Start button. The status of the unload process is displayed in the output area in the lower half of the Unload File tab.

To interrupt an active unload process, click the Stop button (the Start button becomes a Stop button after the extract begins).

Click Close to exit vutil.
