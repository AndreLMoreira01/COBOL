13.4 The Report Component Toolbox

The Report Component Toolbox contains "controls" or elements that you can use to format your report. They are similar to some ACUCOBOL-GT standard controls. Like those controls, you draw them in the Report Composer window by clicking on the icon in the toolbox and positioning the control in the desired location on the screen. The selector component has the same function as in the Screen Component Toolbox. If you select a report control and you decide you don't want to insert it on your report, clicking the selector allows you to choose a different report element in the toolbox.

Note that some report controls are not available for character reports. For information about report controls and their Property window components, refer to section 15.4, "Report Controls."
