19.9.12.22 Options/Code Generator/Program Tag

The options in this category provide some control over AcuBench program tag generation. These options include:

Option Description

Default Tag Name

Enable the check boxes in this section to determine the program tag pairs you want automatically generated by the workbench.

Description

This column describes each program tag's function.

Use External Working Storage Copy File Tag

Enable this check box if you want AcuBench to generate "extern-def" tags for external ".def" files COPY statements.

Copy File List

Enter the name(s) of your external ".def" files.

Use External Procedure Division Copy File Tag

Enable this check box if you want AcuBench to generate "extern-cpy" tags for external Procedure Division COPY files COPY statements.

Copy File List

Enter the name(s) of your external COPY files.
