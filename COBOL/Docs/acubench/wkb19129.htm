19.9.12.6 Options/Environment/Keyboard

A description of the Keyboard options appears in the following table:

Option Description

Category

Select a category of commands from the drop-down list. Main is a superset of Code Editor and Screen Designer.

Reset All

Click Reset All to cause the shortcut key definitions to revert to the defaults.

Command

This field lists all of the workbench commands that belong to the selected category.

Shortcut Key

This field displays the currently assigned keyboard shortcut combination, if any.

Shortcut key (entry field)

Select a command from the "Command" list to enable the entry box. Enter a key combination for the selected command in the entry field. If the key combination is already assigned to another command, the name of the command is displayed in the "Currently assigned to" field.

Currently assigned to

This field displays the name of every command that is assigned the keyboard shortcut combination entered in the "Shortcut key" entry field.

Assign

Click Assign to assign the key combination in the "Shortcut command" entry field to the selected command.

Remove

Click Remove to remove the current shortcut key definition from the selected command.
