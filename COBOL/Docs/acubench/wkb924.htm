9.3.13 Working with COPY Files

AcuBench also maintains a list of the COPY files currently declared in your program. You can view the COPY File List box via the Edit/Advanced/List COPY Files command.

The COPY File List box is a modeless window that allows you to work in the Code Editor window while the list is displayed. In addition to the name of each COPY file, the list box contains columns showing each COPY file's directory and the name of the file in which it is declared. You can control which columns are visible and how the COPY files are sorted via the Tools/Options/Code Editor/COPY File List interface, which is described in section 4.5.9, "Setting COPY File List Options." You can also determine the sort order by clicking on the column headings in the list box.

You can easily insert into your code any COPY file name that appears in the COPY File List box. Position the cursor in your code where you want the COPY file name to appear. In the COPY File List box, select a name, right-click in the list box, and choose Paste. Note that you can change the cursor's position in the code and paste a COPY file name multiple times.

You can open a COPY file in a variety of ways. You can use the Edit/Advanced/Open COPY File command or the Open COPY File push button on the Editor toolbar. You can also select the file in the COPY File List box, right-click in the box, and select the Open COPY File command. Double-clicking on the COPY file name in the COPY File List box also opens the file.

Right-clicking in the list box gives you access to other functions in addition to Paste and Open COPY File. The Copy to Clipboard command lets you copy a COPY file name to the Windows clipboard. You can move the cursor automatically to the declaration of a COPY file in your program via the Go to Definition command. Finally, if you modify your code while the list box is open, you can update the COPY File List by using the Refresh command.
