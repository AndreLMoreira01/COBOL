6.4.7 Compiling a Program

In the Workspace window, right-click on the program name and select Compile from the pop-up menu, or select the program and press the Compile button on the Project toolbar.

If you have previously selected the Use Thin Client command, your object file is directed to your remote server. Note that the file is listed in the Remote Object folder in the Workspace window File tab.

The compile action applies the file-level or project-level settings established for the file in the Project Settings dialog.

Use Build or Rebuild to compile the entire workspace. If you are using thin client operations, you need to use the Build or Rebuild command twice, once for local object files and once for remote object files.
