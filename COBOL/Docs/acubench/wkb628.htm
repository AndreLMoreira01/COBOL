6.4.2 Adding a Program

Adding a program means to add an existing program structure file. Adding a program structure file does not cause any of the associated COPY files or the program file (".cbl") to be added. Only the program structure file is added. In this way, you can use a program structure file as a template for building a new program.

To add a program to a project, use these steps:

  1. In the Structure view of the Workspace window, right-click on the project to which you want to add the program and select Add Program from the pop-up menu. The Add Program to Project dialog is displayed.
  2. In the dialog, locate and specify the name of the program structure file that you want to add and click OK.
  3. To add the associated program file (".cbl"), use the Add/Remove function.

To add a plain source file, use the Add/Remove Files function described in section 6.3.4.
