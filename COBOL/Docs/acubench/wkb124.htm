12.1.2 Events

The Property window allows you to view the events, or system activities, associated with a window or control. You can specify a COBOL paragraph to be executed whenever the specified event occurs. To see the list of event, exception, and embedded procedure types that are associated with a window or control, make sure that the item is active in the Screen Designer, then choose the Event tab of the Property window. Here, you can choose from You can code individual event procedures in the Event Editor. More information about the Event Editor can be found in section 11.7.3, "Using the Event Editor."
