10.3.2.1 Configuring the initial state

Properties that are special to the ActiveX control are set in the ActiveX Control Properties dialog. To launch this dialog, right-click on the control and select the ActiveX Control Property command from the pop-up menu. The options displayed in this dialog are defined by the ActiveX control. To correctly set some options may require access to the documentation provided by the ActiveX vendor. Note that a property that affects the visual appearance of a control can be immediately verified after changing its value by clicking the Apply button and then viewing the control on the screen form.

After you set the initial state (you can always modify it later if needed) and before you generate code for the screen, you must specify a file to save the values in and a handle to the file. To do this, set the values of Resource and Resource Handle in the Property window. Then, when you generate the screen, the state is saved automatically in the designated file and a holder for the handle to the file is placed in the Working-Storage COPY file. The state file must be included in the set of files installed when the program is deployed or the ActiveX control is displayed with the standard, default values. Alternatively, the resource file can be included in the program's object library.
