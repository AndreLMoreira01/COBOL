7.1.3 Adding and Removing Modes

To add or remove a mode, open the Project Settings dialog (Project/Settings) and click on the Add/Remove Mode icon that is located to the right of the "Settings For" drop-down list. The Add/Remove Project Mode dialog is displayed.

To add a mode, click Add to display the Add Project Mode dialog. Enter a name for the new mode. Select a basis for the initial settings: standard default settings, an existing mode, or a ".pof" file that you specify. Click OK.

To remove a mode, click on the name of the mode that you want to remove and click Remove. Click the Close button to close the dialog.
