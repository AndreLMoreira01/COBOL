7.3.2.1 Locating variables in the "Configuration File Entries" categories

If you are unsure where a particular variable resides, use the Find entry entry field to search the categories and locate the variable. Enter the name of the variable in the "Find entry" entry field and click the Find button or press Enter. If the variable is included in the categories, its name is highlighted in the CFE tree view, and its value and comments are displayed.
