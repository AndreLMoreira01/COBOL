19.9.12.2 Options/Environment/Template

A description of the Template options appears in the following table:

Option Description

Use File/New dialog

These options control whether the New dialog is used when creating a new object of the specified type.

New Program

Enable this check box to cause the File/New dialog to be displayed when creating a new program.

New Screen

Enable this check box to cause the File/New dialog to be displayed when creating a new screen.

New Report

Enable this check box to cause the File/New dialog to be displayed when creating a new report.

New File

Enable this check box to cause the File/New dialog to be displayed when creating a new file.

New FD/SL

Enable this check box to cause the File/New dialog to be displayed when creating a new data layout.

Customize Template

This area supports the addition, modification, and removal of file and screen templates. Templates added here appear in the File/New/File or the File/New/Screen dialog, depending on the file type.

Template for

Lets you select the type of template you want from a drop-down list.

Add

Click Add to add a template file.

Delete

Click Delete to remove a selected template file from the list.

Modify

Click Modify to change the selected template's title, to specify a different file, or to add or change the associated description.
