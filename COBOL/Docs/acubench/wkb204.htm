20.3 The Editor Toolbar

The Editor toolbar commands are described in the following table:

Description Menu/Command Equivalent

Find. Lets you enter the text string you want to find, or lets you select text from a drop-down list of recent search operations.

Edit/Find

Find Next. Advances you to the next occurrence of the selected item in the active document.

Edit/Find Next

Find in Files. Lets you locate text strings in files stored on your drives.

Edit/Find in Files

Replace. Lets you replace items in the text with other items.

Edit/Replace

Replace in Files. Lets you replace text strings in files stored on your drives.

Edit/Replace in Files

Toggle Bookmark. Inserts a bookmark into the Bookmark pane for the selected line.

Edit/Bookmark/Toggle Bookmark

Go to Next Bookmark. Automatically moves the cursor to the next bookmarked line.

Edit/Bookmark/Next Bookmark

Go to Previous Bookmark. Automatically moves the cursor to the previous bookmarked line.

Edit/Bookmark/Previous Bookmark

Clear All Bookmarks. Clears all bookmarks from the active document.

Edit/Bookmark/Clear All Bookmarks

Indent to Next. Changes the indent of the current line to match that of the next line.

Format/Indent to Next

Indent to Previous. Changes the indent of the current line to match that of the previous line.

Format/Indent to Previous

List Paragraphs. Lets you see each paragraph name in the active Code Editor file, along with the name of the COPY file in which it appears. When you select this command, the Paragraph List box appears.

Edit/Advanced/List Paragraphs

List Variables. Lets you see each variable name in the active Code Editor file, along with its level number, picture value, the name of the COPY file in which it appears, and any FILLER data items. When you select this command, the Variable List box appears.

Edit/Advanced/List Variables

List Constants. Lets you see each level-78 constant name in the active Code Editor file, along with its value and the name of the COPY file in which it appears. When you select this command, the Constant List box appears.

Edit/Advanced/List Constants

Open Copy File. Opens the COPY file selected in the source code.

Edit/Advanced/Open Copy File

Verb Block Match. Lets you see the opposite end of a verb block command statement. When you choose this command, the Code Editor automatically moves the cursor to the location of the opposite-end verb in the pair.

Edit/Advanced/Verb Block Match

Find Scope. Automatically selects the entire effective range of a selected variable.

Edit/Advanced/Find Scope

Create/Update CFD. Creates a Control Flow Diagram (CFD) for the active program if one does not exist. If a CFD has already exists for the active program, this command updates the existing CFD to reflect any recent changes.

Edit/Create/Update CFD
