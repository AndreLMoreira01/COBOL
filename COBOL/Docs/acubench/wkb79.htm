7.3.2 The CFE Interface

The components of the CFE dialog include the "Configuration file entries" window, the "Value" window, the "Comment" window, the "Editing entry" entry field, and the "Find entry" entry field. The components are described briefly in the following paragraphs.

The "Configuration file entries" window contains a list of all ACUCOBOL-GT configuration variables, organized by category in a tree view. All entries start with a keyword, (followed by one or more spaces, tabs, or an equal sign, and then a value). The categories Alfred, FileSystem, GT, Licensing, Runtime, and User Defined indicate the operational characteristics of the variables. Each category is described briefly in the following paragraphs.

The Alfred category includes configuration variables that affect the behavior of Acucorp's indexed file record editor, alfred. For more information about alfred, see Chapter 3, Book 1 in the ACUCOBOL-GT documentation set.

The FileSystem category includes operating system environment variables, such as those relating to the default host, file names, or to the Vision file system.

The GT category includes all of the ACUCOBOL-GT compiler configuration variables.

The Licensing category includes the license management configuration variables.

The Runtime category includes all of the runtime configuration variables.

The User Defined category lists the variables that have been defined by the user.
