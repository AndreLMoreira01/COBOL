11.7.1 Using Edit Commands

Use Edit menu commands to modify your screen.

Using the Undo and Redo commands

To reverse your most recent action, choose the Edit/Undo command. You can restore the action that you reversed by choosing Redo. The workbench supports multiple undo and redo actions, in reverse order until the last save performed.

Cutting or copying a screen control

You can cut and copy controls to the clipboard. Simply select the control(s) that you want to cut or copy and choose the Edit/Cut or Edit/Copy commands. When you choose Cut, the control is removed from the screen and placed on the clipboard. When you choose Copy, a copy of the control is placed on the clipboard. You can also cut or copy a control by right-clicking on the control and choosing the appropriate command in the pop-up menu.

Pasting a control

After you cut or copy a control to the clipboard, you can paste it to a screen form. The control is pasted into the new screen in the same location that it has in the old screen. After pasting a control, you can move it to any location in the screen form.

A pasted control retains many of the original control's properties. However, it does not retain such unique properties as ID, name, and tab order. The Screen Designer assigns default unique properties, which you can change.

After pasting multiple, identical controls, you can quickly change an individual property value so that it is unique for each control copy. Select all the control copies that you want to have unique variable names (for example, select five identical entry fields) and change the desired variable name in the Property window. AcuBench detects the multiple selection and asks if you want to give each selected control a unique name for that property. With a Yes response, AcuBench automatically increments the number in the variable name for each control so it is unique.

Using the Tab control commands

The tab control has some special commands that you can access by right-clicking on the tab portion of the control in your screen form. The Insert New Page command lets you add a page to your tab control. Next Page lets you bring forward the next page in a selected tab control, and Previous Page lets you move to the tab's previous page.
