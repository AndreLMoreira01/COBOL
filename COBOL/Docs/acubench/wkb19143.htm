19.9.12.20 Options/Screen Designer/Visibility

A description of the Screen Designer's Visibility options appears in the following table:

Option Description

property list

Lets you control the display of a specific screen control property in all the Property windows in which it is listed. Check the property if you want it displayed, or uncheck the property if you do not want it displayed.

Set

Lets you enable the display of a screen control property.

Clear

Lets you disable the display of a screen control property.

Show appearance

Lets you display or hide all the properties that dictate a screen control's appearance.

Show behaviors

Lets you display or hide all the properties that dictate a screen control's behavior.

Show miscellaneous

Lets you display or hide all the screen control properties that are designated miscellaneous.

Show positions

Lets you display or hide all the properties that dictate a screen control's position.

Show variables

Lets you display or hide all the properties that name a variable associated with a screen control property.
