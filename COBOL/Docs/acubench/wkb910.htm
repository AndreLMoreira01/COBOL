9.2.2 Setting Page Appearance

You can use the Page Setup dialog box to set your page's appearance. This box contains settings for style, line number, frame, and margin. You can also check the results of your dialog box entries in Print Preview.

Define your document's page appearance as follows:

  1. Choose the File/Page Setup command. The Page Setup dialog box appears.
  2. Choose the style you want for the text from the "Style" list box. You can choose black and white, color, or zebra (alternating lines showing gray and white).
  3. If you don't want to print line numbers, make sure the "Print line number" check box is cleared.
  4. If you want a frame around your footer, select the style from the "Frame" list box.
    • If you have selected "Print line number", you can choose from among the None, Inner, Outer, or Both options.
    • If you have cleared "Print line number", you can choose either the None or Inner option.
  5. To change the default document margins, enter the measurements you want in the Left, Right, Top, and Bottom boxes.
  6. Click OK.
