19.1.13 Print Setup

Select File/Print Setup to modify or view your print settings. The Print Setup dialog options include:

Option Description

Name

Select the name of the printer you want to set up.

Properties

Click Properties to change options for the selected printer. The options available depend on the printer selected.

Paper

Select the paper size and paper source for your printer from the Size and Source drop-down lists.

Orientation

Allows you to select the orientation (landscape or portrait) of the paper.

OK

Click OK to save the settings in the Print Setup dialog.

Cancel

Click Cancel to exit the Print Setup dialog without saving any changes.
