21.1.2 Structure View: Project Node Pop-up Menu

Right-clicking on a Project node in the Structure view of the Workspace window creates a pop-up menu containing the following project-related commands:

Menu Item Description

New Program

Lets you create a new program structure file in the project.

Add Program

Lets you add an existing program structure file (".psf") to the project.

Delete project_name

Deletes the named project from the workspace.

Settings

Opens the Project Settings dialog.

Save Project project_name

Opens the Save Project dialog.

Build project_name

Compiles every file in the named project that has been modified since the last build, and their dependents.

Rebuild project_name

Unconditionally recompiles all files in the named project.

Properties

Opens the Project Properties dialog.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Vio

Launches the vio interface.

Version Control

Displays the list of user-defined version control commands. This item appears on the menu only if version control commands have been defined.
