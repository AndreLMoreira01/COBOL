4.2 The Workspace Window

The Workspace window is the primary interface to your project's components. It uses a tree-view style presentation that allows you to interact with components and initiate many AcuBench functions. Every node in the window has a context-sensitive, right-click pop-up menu that offers a selection of actions appropriate for the associated object.

The Workspace window has three tabs at the bottom that display the current workspace's member projects in three organizations, or views:

Structure            

displays programs that have a program structure file (".psf")

File

displays the programs, COPY files, resources, etc. associated with a project, grouped by type

Data

displays the project's data layout files

To select a view, simply click on its tab.
