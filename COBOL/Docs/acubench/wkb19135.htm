19.9.12.12 Options/Code Editor/Keyword

A description of the Code Editor's Keyword options appears in the following table:

Option Description

Keyword set

Contains a list of defined keyword sets. Select the name of the list that you want to modify or delete from the drop-down list. When you add a new keyword list, its name appears in this list.

Add

Lets you add a keyword list via the Add Keyword Set dialog (see below for details).

Delete

Lets you delete the selected keyword list.

Delete and New
push buttons

Located at the top of the keyword list box, these push buttons allow you to add or delete selected keywords in your list.

Keyword

Contains the keywords currently defined in the selected keyword set.

Indent value

Specifies the keyword indent values, if any, for the selected keyword set.

Auto-indent

Lets you set an auto-indent behavior for the Code Editor.

None

With this option, the cursor returns to column 1 of the next line after a carriage return.

First nonblank

With this option, a carriage return sends the cursor to the next line at the location of the first non-blank character in the previous line.

Customized

With this option, the Code Editor applies the keyword indent values shown in the keyword list box.

The Add Keyword Set dialog options include:

Option Description

Keyword Set Name

Lets you enter the name of your new keyword set.

Copy From

These options let you determine a source for your new keyword set, if any.

None

Lets you create a list from scratch.

Existing set

Lets you base your list on an existing keyword list.

External set

Lets you import an external list.

OK

Click OK to save your new keyword set.

Cancel

Click Cancel to exit the Add Keyword Set dialog.
