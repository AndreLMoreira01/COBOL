4.2.2 File View

The Workspace window's File tab displays each project's member files. You use the File view to interact with the files in your projects.

By default, project files are grouped into related types, such as Source, Screen, Report, Copy, Object, Remote Object, List, Resource, and FD files. You can combine folders, or define additional folders. You can also change the default extensions associated with any folder by right-clicking on the folder node and selecting Properties from the pop-up window. Most files can be opened or executed by double-clicking on the file name or file icon.

A complete discussion of working with projects begins in Chapter 6. The following topics are discussed, among others:

  • adding files (section 6.3.4)
  • removing files (section 6.3.6)
  • moving files (section 6.3.7)
  • adding a folder or changing the extensions associated with a folder (section 6.3.3)
