16.4.2 Opening an Existing Data Set

You can open an existing data set in the Data Set Designer as follows:

  1. In the Structure view of the Workspace window, open the desired Program node and double-click on the Data Set node. The Data Set Designer window is opened and an icon is displayed for each data set.
  2. Select the desired data set icon.
  3. Display the Property window by selecting Property Window from the View menu.
  4. View or modify the data set's properties as desired.
