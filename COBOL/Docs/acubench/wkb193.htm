19.1.1 New

The File/New option launches a dialog that allows you to create a new project, program, screen, report, file, or file descriptor (FD/SL). In the New dialog, select the tab that corresponds to the type of object that you want to create and complete the dialog.

New Project

Select the Project tab to create a new project. The options on this tab include:

Option Description

Blank template/Standard Project template

Select the type of template you want to use to create your new project. The Standard Project template creates a project that includes one standard graphical screen (that you can modify as necessary). The Blank template does not include a screen.

Project Name

Type the name of your project in the "Project name" field.

Location

Type the full directory path to the location in which you want to save your new project. Click the button to the right of the "Location" field to browse for a directory path.

Open in new workspace

Select this option if you want to add your new project to a new workspace.

Add to current workspace

Select this option if you want to add your new project to the workspace that is currently open.

More Info

Click this button to view or modify additional information pertaining to the project's file and directory structure and settings for thin client. For more information, see the Project Information dialog description below.

OK

Click OK to save the information for your new project.

Cancel

Click Cancel to exit the New (Project) dialog without saving.

When you click the More Info button on the Project tab of the New dialog, the Project information dialog opens, displaying the project name, directory structure, and thin client options.

The Project Information dialog includes the following options:

Option Description

Project name

Defines the name of your project in the "Project name" field.

Directory

Defines the immediate directory path to the location in which you want to save your new project. Click the button to the right of the "Directory" field to browse for a new directory path.

Working directories

Defines the directory path of your project's source, screen, report, library, object, list, resource, and file descriptor files. All of these directory paths can be modified.

Remote setting for use with Thin Client

Lets you define the server, port, and remote object directory for use with thin client operations. A Validate button lets you validate that a connection can be made to the specified server via the named port.

OK

Click OK to save the information for your new project.

Cancel

Click Cancel to exit the Project Information dialog without saving.

New Program

Select the Program tab to create a new program for your project. The options on this tab include:

Option Description

Blank/Standard Program

Select the type of program you want to create for your project. The Standard Program creates a program that includes one standard graphical screen (that you can modify as necessary). The Blank program does not include a screen.

Program name

Type the name of your program in the "Program name" field.

File name

This field is automatically populated using the name you use to define your program. All program files end with the ".psf" extension.

Location

Type the full directory path to the location in which you want to save your new program. Click the button to the right of the "Location" field to browse for a directory path.

Add to existing project

Select a project to which to add your new program from the drop-down list.

OK

Click OK to save the information for your new program.

Cancel

Click Cancel to exit the New (Program) dialog without saving.

New Screen

Select the Screen tab to create a new screen for your program. The options on this tab include:

Option Description

Screen template types

Select the type of screen you want to create for your project.

 

Blank Graphical creates a blank graphical screen for your project.

Standard Graphical creates a graphical screen that contains two default buttons.

Graphical Password creates a graphical screen that is used to gather user and password information.

Blank Character creates a blank character-based screen for your project.

Standard Character creates a character-based screen that contains two default buttons.

Character Password creates a character-based screen that is used to gather user and password information.

Form name

Type a name for your new screen.

Unique prefix

Type a default prefix to assign to your screen files. The default is FormN, where N is a sequential number.

Create with new program, and add to project

Select this option if you want to create a new program for your new screen. Select the project to which to add this screen from the drop-down list.

Add to existing program

Select this option if you want to add your screen to an existing program. Select the project and program to which to add this screen from the drop-down list.

OK

Click OK to save the information for your new screen.

Cancel

Click Cancel to exit the New (Screen) dialog without saving.

New Report

Select the Screen tab to create a new screen for your program. The options on this tab include:

Option Description

Report template types

Select the type of report you want to create for your project.

 

Blank Graphical creates a blank graphical report for your project.

Blank Character creates a blank character-based report for your project.

Form name

Type a name for your new report.

Unique prefix

Type a default prefix to assign to your report files. The default is ReportN, where N is a sequential number.

Create with new program, and add to project

Select this option if you want to create a new program for your new report. Select the project to which to add this report from the drop-down list.

Add to existing program

Select this option if you want to add your report to an existing program. Select the project and program to which to add this report from the drop-down list.

OK

Click OK to save the information for your new report.

Cancel

Click Cancel to exit the New (Report) dialog without saving.

New File

Select the File tab to create a new source file for your project. The options on this tab include:

Option Description

Blank Text/ Source Template

Select the type of template to use to create your new file. The Blank Text template creates an empty text file, while the Source Template creates a COBOL source file based on the standard template.

File name

Type the name of your new file. The default file extension is ".cbl".

Location

Type the full directory path to the location in which you want to save your new file. The default directory path is the Source folder in your existing project directory. Click the button to the right of the "Location" field to browse for a directory path.

Add to existing project

Set this check box if you want to add this file to your existing project. Select a project to which to add your new file from the drop-down list.

Create Program Structure File (.PSF)

Set this check box if you want AcuBench to create a program structure file (".psf") for the file you are adding.

OK

Click OK to save the information for your new file.

Cancel

Click Cancel to exit the New (File) dialog without saving.

New FD/SL

Select the FD/SL tab to create a new data layout file for your project. The options on this tab include:

Option Description

Blank/Import From Files

Select the type of FD/SL file you want to create for your project. Choosing Blank launches the File Designer and allows you to define new file descriptions or select statements for your project. Import From Files allows you to import existing FD or SL files into your current project.

FD/SL name

Type a name for your new FD/SL files.

Unique prefix

Type a default prefix to assign to your FD/SL files. The default is FDN, where N is a sequential number.

File name

This field is automatically populated using the name you use to define your FD/SL file. All FD/SL files end with the ".dlt" extension.

Location

Type the full directory path to the location in which you want to save your new FD/SL file. The default directory path is the FD folder of your current project directory. Click the button to the right of the "Location" field to browse for a directory path.

Add to existing project

Select a project to which to add your new file from the drop-down list.

OK

Click OK to save the information for your new file.

Cancel

Click Cancel to exit the New (FD/SL) dialog without saving.
