4.5.7 Setting Variable List Options

Variable List options allow you to determine how the Code Editor's Variable List box is sorted and what columns of information are displayed in the box.

Set Variable List options using the following procedures:

  1. Select the Variable List category of the Code Editor options.
  2. "Sort By" options: If you want variable names to be sorted in the Variable List box in order of their occurrence in the program, select Occurrence in the program. If you want variable names listed alphabetically, choose Alphabetically. Variable names can be sorted by level number when you choose the Level number radio button. Finally, variable names are sorted by picture clause if you select Picture description.
  3. "Show" options: If you want the Variable List box to display a variable's level number, select Level number. The box will show a variable's picture clause if you select Picture description. Finally, if you want the Variable List box to display the names of the COPY files with variable names, set the The name of the copybook check box.
  4. "Include FILLER data item": Select this option if you want blank FILLER data items listed in the Variable List box. Clear the check box to omit FILLER items from the list.
