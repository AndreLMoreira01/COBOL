17.3.1 Creating an Object Library

To create an object library, you first specify the primary module, then construct the list of objects to include in the library. You then build or rebuild the workspace, which compiles the programs and invokes cblutil to create the library.

To specify an object library:

  1. Set the primary module and module type.
    1. The primary module and module type are designated in the Library tab of the Project Properties dialog. Use the Project/Properties command to open this dialog. You can also right-click in the Workspace window on the project that contains the primary module and select Properties from the pop-up menu.
    2. Click on the Library tab.
    3. Select a primary module by double-clicking on its name in the "Program File" list. A check mark is placed to the left of the selected program.
    4. Select Object Files from the "Module type" drop-down list. Note that you can also create libraries of program structure files and source files.
    5. Click OK to close the dialog.
  2. Set the library creation options and library list.
    1. The library creation options are set in the Library tab of the Project Settings dialog. Use the Project/Settings command to open this dialog. Or, in the Workspace window, right-click on the project that contains the primary module and select Settings from the pop-up menu.
    2. Click on the Library tab.
    3. In the project tree on the left side of the dialog, locate and click on the project that contains the primary module. The name of the primary module should then appear in the "First module from" field. If it does not, or if you want to change the designated primary module, perform the instructions in step 1.
    4. Enable the Build as library option.

      5. Note: As long as the "Build as library" option is enabled, the specified object library is built every time you build or rebuild the workspace.

    5. Specify the modules you want your object library to contain in the "Additional Modules" entry box. Use the New and Delete buttons located at the upper right of this box to add and delete modules in the box. When you press the New button, you can use the browse button that appears to the right of the blank entry to locate the desired file.
    6. Use the Up and Down arrow buttons to re-order the modules.
    7. Set the verbose mode and delete object after options as desired.
    8. Click OK to save the settings and close the dialog.
  3. Build or rebuild the workspace to create the object library. The result of the library creation action is displayed in the Output window.
