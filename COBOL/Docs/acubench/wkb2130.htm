21.10.5 XFD Tab Pop-up Menu

Right-clicking in the File Designer XFD tab creates a pop-up menu containing the following commonly used commands:

Menu Item Description

Undo

Reverses your most recent editing action.

Redo

Reverses your most recent undo action.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Generate FD/SL

Generates new ".fd" and ".sl" files from the data layout file.

Make XFD File

Generates an XFD file from the selected file description.

Set Default XFD

Resets the file or name directive to its default value, if you have previously changed it.
