17.7 Using alfred

alfred is Acucorp's indexed file record editor. You can use alfred to view, modify, add, or delete individual records stored in an indexed file. alfred can be especially useful when debugging data file layouts and I/O actions. It can also be useful to system administrators who need to make occasional record-level changes to data files. Note that alfred is not well suited to repetitive or large-scale changes.

You can launch the indexed file record editor by selecting Alfred from the Tools drop-down menu. Running alfred from within AcuBench is no different than running alfred from the command line. For a complete description of alfred, see section 3.5 of the ACUCOBOL-GT User's Guide.
