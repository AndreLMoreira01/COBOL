9.3.15.1 Finding a text string in the active document

The Edit/Find command lets you find all occurrences of any text string in the active Code Editor file. You can also direct the Find operation to place a bookmark at each line of code in which the text string occurs, or list all occurrences of the text string in the "Find in Files" Output window pane. Use the following steps for these functions:

  1. Move the cursor to the place in the file where you want the search to begin.
  2. Choose the Edit/Find command. The Find dialog box appears. If you selected text in the document before choosing the Find command, that text automatically appears in the "Find what" text box.
  3. If you did not select text before choosing the Find command, type the text string that you want to find in the "Find what" text box or select from a drop-down list of up to 15 text strings previously entered for the Find operation.
  4. To match the case of the text string exactly as typed, select the Match case check box.
  5. Select Regular expression if you want to enter a regular expression as the item to be found rather than a specific text string. Build the search expression by entering characters and operators.
  6. Select the Wrap around search check box to direct that the search operation continue at the beginning of the active document after it reaches the end of that document.
  7. To direct the search operation to ignore hyphens and underscores, select the Ignore `-' and `_' check box.
  8. To determine the direction of your search, select either the Up or Down radio button.
  9. Click the Find Next button. The Code Editor searches for the specified text string and selects the first occurrence of that string.
  10. To find the next occurrence of the string, click Find Next again. The Code Editor informs you when it has completed your search.
  11. Click the Mark All button to direct the search operation to place a bookmark at each line in which the text string occurs.
  12. Click the Find All button to direct the search operation to find all occurrences of the text string and list them by line number in the "Find in Files" Output window pane. To move the cursor to the location of a found string, double-click its listing in the "Find in Files" Output window pane.

You can leave the Find dialog box open while you continue to use the workbench.
