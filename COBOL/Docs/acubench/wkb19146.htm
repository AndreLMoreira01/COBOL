19.9.12.23 Options/Data Designer/General

The options in this category affect specific aspects of the data designer tools, which include the File Designer, Working Storage Editor, and Linkage Editor. These options include:

Option Description

Comment position

Select a radio button to specify the placement or exclusion of comments. Select Up to cause comments to be placed above the line that they comment. Select Down to cause comments to be placed below the line that they comment. Select None to exclude comments from the generated files.

COPY file color

This area allows you to select the colors used to show data items that are "Linked to" in a data structure (included via the "Link to file" option of the "Add Item" list). Linked items may themselves include COPY statements. Items included as the result of nested COPY statements can also be shown in a contrasting color. To set the color used by each level, double-click in the "Text Color" field of the desired level and select a color.
