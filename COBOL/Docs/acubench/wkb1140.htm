11.8.1 Creating a Menu Bar

If you have opened a standard template from the File/New/Screen dialog box, the Menu Designer icon automatically appears in the upper left corner of your screen. If your screen does not contain this icon, drag a Menu Designer control from the Screen Component Toolbox to any location in the screen form. The Menu Designer control icon does not appear when you run your screen, so its exact location in the screen form is not important.

When you double-click this icon, the Menu Designer dialog box appears with the menu text marker highlighted. This dialog box lets you add, delete, and move menu items. These tasks are detailed in the following sections.

You can set menu properties in the Property window. A property item is a characteristic of the menu's behavior. To set or change a property, enter or select your property item preference in the corresponding value field. Information about properties can be found in Book 2 of the ACUCOBOL-GT manual set. The Menu Designer generates no events.

The following sections provide details for performing Menu Designer operations.
