19.9.12.11 Options/Code Editor/Tabs

A description of the Code Editor's Tabs options appears in the following table:

Option Description

Display tab character with

Lets you determine how a tab location is displayed in your source code. Enter a nonblank character in the box.

Tab size

Lets you determine the default size of a tab, from 1 to 64.

Source format

Lets you choose the source format for which you want to establish a set of tab stops.

Keep tabs

Lets you enable or disable the Code Editor's support of tabs in your code.

Insert spaces

Lets you enable or disable the automatic replacement of tabs with spaces in your code.

Tabs

Lets you set up to 32 custom tab stops in your code. Use the Add or Delete button to change your custom settings.
