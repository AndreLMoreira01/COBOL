9.3.2 Opening a Code Editor File

You can open a new or existing source code file from the File menu. The File/New command displays the New dialog box, from which you can open a blank file or a standard file template from the File tab. If you have added other source code templates, icons for these templates appear in the New dialog box File tab, and you can open them. More information about adding source templates can be found in section 9.3.14, "Using Source Code Templates." Use the File/Open command to open an existing source code file.

Open a new source code file as follows:

  1. Choose the File/New command. When the New dialog box appears, select the File tab.
  2. Select the icon for the desired source code template or select the icon for a blank file.
  3. If desired, change the default entries for file name and directory location.
  4. If you want to add the file to your existing project, make sure that the Add to existing project check box is selected and designate the workspace component where you want the file stored. If you do not want to add the new file to the current project, clear the Add to existing project check box.
  5. Click OK.

To open an existing file, choose the File/Open command to display the Open dialog box. Navigate to the desired file location, and open the file.
