4.5.2 Setting Format Options

You can use the Format portion of the Code Editor Options to change the colors that identify each ANSI display field (Sequence Number area, Indicator area, area A, area B, and Identification area). You can also customize the appearance of code text types (Comment, Keyword, String, Number, and Text) and designate the text string that is inserted in ANSI format columns 73-80 when you modify source code.

The Format options also allow you to change the default column settings of your file. The default settings are the ANSI-specified standard settings. If you are working with terminal source format, you may want to change these default settings. An explanation of terminal source format is found in Book 1 of the ACUCOBOL-GT documentation set.

Set Format options as follows:

  1. Select the Format category of the Code Editor options.
  2. "Source format" option: Select either ANSI Format or Terminal Format in the "Source format" entry field.

    3. Note that the remaining options on the page change depending on which format you have selected.

  3. "Modify code" option: Select this option if you want the Code Editor to insert a specified character string in ANSI format columns 73-80 when a line of code is added or modified. Enter the string you want inserted in the adjacent entry field. This option is disabled when you have Terminal Format selected.
  4. Foreground and background color setting options: If you want to disable all color functions, clear the Enable foreground and background color settings check box. Inhibiting color functions is useful if you are editing a non-source file, such as a configuration file.
  5. The five ANSI display fields (Sequence Number area, Indicator area, area A, area B, and Identification area) and the five code text types (Comment, Keyword, String, Number, and Text) appear in the list box. As you select an area's listing, its background color and the column(s) it occupies appear in the "Background" and "Columns" boxes, respectively. To make color changes to the background, select a color from the "Background" list box.
  6. As you select a text type (Comment, Keyword, String, Number, or Text), its current background and foreground colors appear in the "Background" and "Foreground" boxes, respectively. To make color changes, select other colors in these boxes.
  7. To change the column width of any field, enter a different number for that field in the "Columns" boxes. Only the Sequence Number area shows an active entry in the "Columns from" box unless you have changed column numbers. Note that if you change a column setting, you are simply redefining the size of the background color display. You are not redefining any particular area (for example, area A) for the compiler. Columns may not overlap.
  8. "Font" option: To change your text font, click the Font push button. The Font dialog box appears.

    9. Select the font, style, and size that you want, then click OK. Once you have clicked OK again to close the Options dialog, your text appears in the Code Editor in the new font.

  9. Check your changes in the Example window.
