4.1.3 The Toolbars

The AcuBench toolbars provide push button access to the most frequently used workbench functions.

You can display or hide each toolbar via the View drop-down menu, or by right-clicking on a toolbar.

Placing the mouse pointer over a toolbar button for more than a second causes a tool tip to be displayed. At the same time, a short description of the button's function is also displayed in the status bar.

You can customize the commands on each toolbar, or create new toolbars, using the Tools/Customize dialog (see section 4.10, "The Customize Dialog", for specific instructions, or section 19.9.11, "Customize", for reference information).

The default toolbars include:

  • Standard
  • Project
  • Editor
  • Debug
  • Align
  • Launch Bar

You can display, hide, resize, and move the toolbars. Toolbars that are not docked under the menu bar or to the edge of the window are called floating toolbars.
