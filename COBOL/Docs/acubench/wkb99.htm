9.2.1 Setting Headers and Footers

The Code Editor automatically displays a default header at the center top of each page showing the file name, and a default footer at the center bottom of each page showing the page number. These settings are easy to change in the Page Setup dialog box.

Header and footer values are represented in the listings by the following characters:

Value Character

Full Path Filename

&U

Filename

&F

Page number

&N

Current time

&T

Current date

&D

Header and footer alignment positions are represented in the listings by the following characters:

Position Character

Left align

&L

Center

&C

Right align

&R

You can change header or footer settings as follows:

  1. Choose the File/Page Setup command. The Page Setup dialog box appears.
  2. Click the arrow button beside the "Header" or "Footer" list box, depending on which setting you want to change. A pop-up menu appears.
  3. From the pop-up menu, choose the component(s) of your header or footer.
  4. Choose the position in which you want the component(s) to appear. You can set a separate alignment position for each header or footer value you enter.
  5. Click OK.
