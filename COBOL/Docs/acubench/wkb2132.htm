21.12 Working Storage and Linkage Editor Pop-up Menus

Right-clicking in the data description area of the Working Storage or Linkage Editor creates a pop-up menu containing the following commands:

Menu Item Description

Syntax Check

Runs the compiler to check the Working-Storage item definitions for correct syntax.

Add Item Before

Adds an item before the selected entry.

Add

Adds an item after the selected entry.

Add Sub-item

Adds a subordinate item after the selected entry.

Undo

Reverses your most recent editing action.

Redo

Reverses your most recent undo action.

Cut

Removes the selection from the document and places it on the clipboard.

Copy

Places a copy of the selection on the clipboard.

Paste

Places a selection that has been cut or copied to the clipboard into the active document.

Delete

Deletes the selected item.

Select All

Causes all data items to be selected.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Link Copy File

Lets you link to the contents of another file. The contents of that file are included in a COPY statement when the generated file is compiled. The file can contain only data file descriptors.

Import Copy File

Lets you import (make a copy of) the contents of another file. The file can contain only data file descriptors.

Remove Copy File

Lets you remove a link to a file.

Reload All Copy Files

Lets you reload the contents of a linked file.
