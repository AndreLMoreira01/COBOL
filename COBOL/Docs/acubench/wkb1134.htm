11.7.2 Using Align Commands

You can use the Align commands to align, size, and adjust the spacing for selected controls. The Align menu also includes commands for toggling the screen form grid and guides. Buttons for some of these commands also appear in the Align toolbar.

You can move selected controls to the vertical or horizontal center of the screen or align controls relative to one another. When three or more controls are selected, you can apply even spacing (either horizontal or vertical) in the selected controls. The Adjacent command lets you position two controls so that they share boundary locations, either horizontally or vertically.

You can make selected controls the same width, height, or size (both width and height). Selected controls are sized to match the dominant control, which is the last control selected in a group of controls. You can also resize a selected control just to fit its title or label.

You can lock or unlock the position of the selected control(s) in your screen. Locked controls cannot be moved by dragging, although they can be cut, centered, and sized.

You can determine the order in which the cursor tabs to, or visits, controls on the screen. Because all control generation must be definitively ordered in source code, even noninteractive controls like labels are assigned a tab order number. The Tab command lets you tab through your controls in a screen form, and Shift+Tab lets you tab backward.

The following sections provide instructions for performing these alignment functions.

Centering or aligning controls in the screen

  1. Select the controls that you want to center or align. To select a group of controls at the same time, hold the left mouse button down while you draw a rectangle around the group of controls, or hold Shift down while clicking on each control to be added to the group. In either case, the last control selected is the dominant control in the group. The other controls in the group are aligned with the dominant control.
  2. To center a control, use the Align Control/Center Horizontal or Align Control/Center Vertical command. Depending on which button you clicked, the selected controls move to the vertical or horizontal center of the screen.
  3. To align controls, select from the following Align Control commands, depending on how you want your controls aligned: Left, Right, Top, Bottom, and To Grid. The last command aligns your selected control to the closest grid point in the screen form. Alignment occurs relative to the dominant control's position.

Applying even spacing in a group of controls

  1. Select the controls for which you want to apply even spacing. You must select at least three controls for these commands to be available.
  2. To apply even horizontal spacing, use the Space Evenly/Across command.
  3. To apply even vertical spacing, use the Space Evenly/Down command.

Applying the Adjacent command

  1. Select the controls to which you want to apply this command. Note that the last control selected is the dominant control.
  2. Use the Adjacent/Horizontal command to position the controls so that the right boundary of one control has the same horizontal location as the left boundary of the other control.
  3. Use the Adjacent/Vertical command to position the controls so that the bottom boundary of one control has the same vertical location as the top boundary of the other control.

Sizing controls

  1. Select the controls that you want to resize relative to the dominant control.
  2. To make the controls the same width as the dominant control, use the Make Same Size/Width command.
  3. To make the controls the same height as the dominant control, use the Make Same Size/Height command.
  4. To make the controls the same width and height as the dominant control, use the Make Same Size/Both command.
  5. To resize your control so that it is just large enough to accommodate its title or label, use the Size to Content command.

Locking or unlocking controls

  1. If a control's position is currently unlocked, use the Lock Controls command to lock its position on the screen. You can also select the Lock Control command from the Screen Designer's right-click menu. A locked control's selection handles are white instead of dark colored.
  2. Unlock a locked control by choosing the Lock Controls command a second time.
  3. Note that all Align Toolbar buttons are disabled when a selected control is locked, except for the View Grid button.

Determining tab order

  1. Use the Tab Order command to display the Reset Controls Tab Order dialog box. The dialog box contains a list of the controls on your screen, along with the assigned tab order. The tab order also displays on your screen design.
  2. Change the tab order on your screen by selecting a control and using the up and down arrows.
  3. You can also use the Send to Back and Send to Front commands in the Screen Designer right-click pop-up menu. Send to Back gives the selected control the lowest tab order number, reordering all other controls accordingly. Send to Front gives the selected control the highest tab order number. These controls are not enabled when more than one control is selected.

Note that if you create a frame and place controls inside that frame, the tab order within the frame exists exclusively of the controls outside of it. In this instance, you cannot create a tab order outside of a frame and continue it inside of the frame. The tab order inside of the frame starts at "1." You can, however, change the tab order within the frame itself. Tab order is continuous if you draw a frame around controls that already exist.
