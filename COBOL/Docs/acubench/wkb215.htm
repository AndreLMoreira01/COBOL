21.1.3 Structure View: Program Node Pop-up Menu

Right-clicking on a Program node in the Structure view of the Workspace window creates a pop-up menu containing the following program-related commands:

Menu Item Description

View program_name

Opens the named file in the Code Editor.

Generate Source

Generates code for the program element that has focus.

New Screen

Lets you add a new screen to the program.

Add Screen

Lets you add an existing screen (".smf", ".stf", ".out") to the program.

New Report

Lets you add a new report to the program.

Add Report

Let you add an existing report (".wtf") to the program.

New Data Set

Lets you add a new data set to the program.

External Paragraph

Lets you identify the event paragraphs that you have coded in separate files and that, therefore, should not be generated.

External Variable

Lets you identify the variables that you have coded in a separate file (such as the main program) and that, therefore, should not be generated.

Save program_name.psf

Saves the named program structure file.

Delete program_name.psf

Deletes the named program structure file.

Delete From Disk

Deletes the selected program structure file from the disk.

Compile program_name

Compiles the named COBOL file.

Reparse program_name

Reparses the named COBOL file.

Execute program_name

Executes the associated COBOL object, if present.

Debug (Runtime)

Starts the associated COBOL program in the ACUCOBOL-GT® runtime debugger.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Properties

Displays the Program Properties dialog.

Version Control

Displays the list of user-defined version control commands. This item appears on the menu only if version control commands have been defined.
