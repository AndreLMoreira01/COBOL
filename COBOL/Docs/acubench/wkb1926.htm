19.2.5 Paste

Places a selection that has been cut or copied to the clipboard into the active document or screen. When you choose this command, text is pasted to the cursor location or a screen control is pasted into the new screen in the same location that it has in the old screen. After pasting a control, you can move it to any location in the screen.

A pasted control retains many of the original control's properties. However, it does not retain such unique properties as ID, name, and tab order. The Screen Designer assigns default unique properties, which you can change if desired.

After pasting multiple, identical controls, you can quickly change an individual property value so that it is unique for each control copy. Select all the control copies that you want to have unique variable names (for example, select five identical entry fields) and change the desired variable name in the Property window. AcuBench detects the multiple selection and asks if you want to give each selected control a unique name for that property. With a Yes response, AcuBench automatically increments the number in the variable name for each control so it is unique.

The Paste command is available only if you have cut or copied a selection to the clipboard.
