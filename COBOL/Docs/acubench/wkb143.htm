Chapter 14: Working with Reports > 14.1 Creating a Report >

1. Create the project and set up the work environment.

  • Use the File/New/Project dialog to create a new project, or open the project to which you plan to add a report.
  • Identify an FD/SL pair or data layout (".dlt") file, as well as the corresponding data file.
  • If you are creating a new data layout file from an FD/SL pair, it is a good idea to copy the FD and SELECT COPY files to the FD subdirectory in your project directory. Likewise, if you are using an exist data layout file, copy this file to the project's FD subdirectory.
  • The ACUCOBOL-GT runtime searches for your data file in the directory indicated by FILE-PREFIX. If no FILE-PREFIX is set, it searches for the data file in the project root directory.
