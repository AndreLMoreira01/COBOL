19.9.11 Customize

Select Tools/Customize to display or hide toolbars, to add AcuBench command icons to any toolbar, or to add a link to an external program to a toolbar. Links to external programs are then accessed by clicking the appropriate icon on the toolbar, or by selecting Tools/Launch Tools.

Specific steps for using this interface to create or modify toolbars, or to add links to external programs, can be found in Chapter 4, section 4.10, "The Customize Dialog."

The options available on the Customize dialog are specific to the tab you have selected.

Toolbars tab

allows you to display or hide AcuBench toolbars, or create a new toolbar

Commands tab

allows you to add or remove command buttons from a toolbar

Tools tab

allows you to add links to external applications to the AcuBench interface for easy access to them while using AcuBench

Customize/Toolbars

Options on the Toolbars tab allow you to create a new toolbar and to control which AcuBench toolbars are displayed. The Toolbars tab includes the following options:

Option Description

Toolbars

Displays a list of toolbars available for display in the AcuBench interface. Check the box next to the toolbars you want to display, or uncheck the boxes next to the toolbars you want to hide.

Toolbar name

Displays the name of the toolbar selected in the Toolbars list.

New

Click New to display the New Toolbar dialog that allows you to add a custom toolbar to AcuBench.

Delete

Click Delete to remove a custom toolbar from the Toolbars list. The Delete button is not available for any of the AcuBench default toolbars.

Reset

Click Reset to return to the default setting for the toolbar selected in the Toolbars list.

Reset All

Click Reset All to return to the default setting for all available toolbars.

Close

Click Close when you are finished selecting toolbars to view.

Customize/Commands

When the Commands tab is active, you can add command buttons to existing toolbars in the interface. Select the command category you want to view, click a button to the right to view a description of its associated command, then drag and drop the button onto any toolbar displayed in the AcuBench interface. Click Close when finished. The Commands tab includes the following options:

Option Description

Categories

Displays a list of toolbar button categories you can browse to find the button for the command you want to add to a toolbar.

Buttons

Displays the buttons available for the selected category. Click a button to view a description of the command. To add a button to a toolbar, select the button you want to add, then drag and drop the selected button onto the toolbar of your choice.

Description

Displays the description of the command associated with the selected button.

Close

Click Close when you are finished adding command buttons to toolbars.

Customize/Tools

When the Tools tab is active, you can add new program commands to the AcuBench interface. The Tools tab includes the following options:

Option Description

Menu contents

Displays a list of the external programs you have added to AcuBench. These programs display as submenu items when you select Tools/Launch Tools.

Delete button

Click Delete to remove the selected program from the "Menu Contents" list.

New button

Click New to add a new program to the "Menu Contents" list.

Command

Type the full directory path and file name of the executable you want to run when this menu command is selected. You can browse for a command by clicking the button to the right of the "Command" field.

Arguments

If necessary, enter the appropriate arguments for the command. You can select arguments for your command by clicking the button to the right of the Arguments field.

File Path {FilePath} is the full path name of the files that are contained in the working project or folder.

File Directory {FileDir} is the directory of files that are contained in the working project or folder.

File Name {FileName} is the file name of the files that are contained in the working project or folder.

Current Directory {CurDir} is the current directory containing the AcuBench application.

Workspace Path {WkspPath} is the full path name of the current workspace.

Project Directory {ProjectDir} is the directory containing the working project.

Project Name {ProjectName} is the name of the working project.

Working directory

If necessary, enter the path for the working directory from which the new command can be executed. Select the appropriate variables by clicking the button to the right of the "Working Directory" field.

File Directory {FileDir} is the directory of files that are contained in the working project or folder.

Current Directory {CurDir} is the current directory containing the AcuBench application.

Project Directory {ProjectDir} is the directory containing the working project.

Context menu

Type the text you want to display in the menu when you select Tools/Launch Tools. For example, if you added the command for the Windows Calculator application to the "Menu Contents" list, type "Calculator" in the "Context Menu" field to display "Calculator" in the Launch Tools menu.

Use output window

Enable this option to direct any output generated by the application to be displayed in the Output window.

Prompt for argument

Enable this option to display a dialog in which you can specify options and arguments before launching the external application.
