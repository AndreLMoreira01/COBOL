16.2.2 The File Control Screen

Selections made and data entered on the File Control screen are combined to construct a SELECT statement that is generated into the associated ".sl" COPY file. Therefore, selections made and data entered must conform to the ACUCOBOL-GT rules for a File-Control paragraph SELECT statement (see section 4.3.1 of the ACUCOBOL-GT Reference Manual).

File Control Screen Options

Option Description

File name

This field displays the name of the data layout file. The file name is established when you create a new data layout (in the File/New/FD/SL dialog). You cannot change the name in this dialog. To change the name, in the workspace Data view, right-click on the data layout file and select Properties from the pop-up menu.

Optional

Select this check box to include the OPTIONAL phrase in the SELECT statement.

Advanced button

Opens a dialog with additional options that are specific to the selected file format type (sequential, relative, or indexed). See the individual descriptions below.

Assign to

These fields build the ASSIGN TO phrase.

Device

Select a device type from the drop-down list.

Name

Enter a file name.

File format

These options specify essential components of the SELECT statement.

Format

Select a file format type from the drop-down list.

Keyname

This field is enabled for relative files only. Enter a name for the data item that will hold the value of the relative key.

Access mode

Select an access mode from the drop-down list.

Lock mode

Select a lock mode from the drop-down list.

File Status    

Enter a name for the file-status data item. A data item with the specified name is generated into the Working-Storage COPY file (".wrk").

Comment

An entry area for comments. Comments are placed at the top of the generated ".sl" file.

The Advanced Options for Sequential File dialog

This dialog allows you to specify SELECT statement options that are used only with sequential files. These options include:

Option Description

Reserve

Enable this check box to include the RESERVE phrase in the SELECT statement.

Number

Select this radio button and specify a numeric value in the entry field to use the RESERVE number clause.

Alternate        

Enable this check box to include the ALTERNATE AREA clause of the RESERVE phrase.

No

Select the No radio button to use the RESERVE NO AREAS phrase.

Padding

Enable this check box and enter the padding character in the entry field to include the PADDING phrase.

Record delimiter is standard-1

Enable this check box to include the RECORD DELIMITER phrase.

OK

Click the OK button to accept the settings.

Cancel

Click the Cancel button to close the dialog without saving the settings.

The Advanced Options for Indexed File dialog

This dialog allows you to set options that include SELECT statement phrases that are used only with indexed files. These options include:

Option Description

With

Enable this check box to include the WITH phrase. Select a radio button option to complete the phrase.

Encryption

Enable this radio button to use the WITH ENCRYPTION phrase.

Compression        

Enable this radio button to use the WITH COMPRESSION phrase. Specify a compression factor in the entry field.

Reserve

Enable this check box to include the RESERVE phrase in the SELECT statement.

Number

Select this radio button and specify a numeric value in the entry field to use the RESERVE number clause.

Alternate

Enable this check box to include the ALTERNATE AREA clause of the RESERVE phrase.

No

Select the No radio button to use the RESERVE NO AREAS phrase.

Collating

Enable this check box to include the COLLATING phrase. Enter the name of an alphabet that is declared in SPECIAL-NAMES in the entry field.

OK

Click OK to accept the settings.

Cancel

Click Cancel to close the dialog without saving the settings.
