10.6.1 File Menu Commands

The following File menu commands assist you with screen generation:

New. Lets you open a new screen.

Open. Lets you open an existing screen file from the Open dialog box.

Close. Closes the active screen.

Save. Saves the contents of the active screen.

Save As. Lets you save a copy of the active screen.

Save All. Lets you save the contents of all open files.

Print Setup. Lets you change the printer and certain print properties.

Print Preview. Lets you view the active screen before you print it.

Print. Lets you send the active screen to the printer.
