19.9.12.8 Options/Environment/Miscellaneous

A description of the Miscellaneous Environment options appears in the following table:

Option Description

Automatically modify COPYPATH

These options control how the COPYPATH environment is defined.

When adding a COPY file

Enable this option to direct AcuBench to redefine COPYPATH to include the path of a COPY file when it is being added.

When modifying the COPYPATH in the Add/Remove Files dialog

Enable this option to direct AcuBench to redefine COPYPATH when it is changed via the Add/Remove Files dialog.

Automatically create a program structure file

This option determines whether, when a source file is added to a project, tags are inserted in the file and a program structure file is created.

When adding a source file to the project's source folder

Enable this option to direct AcuBench to create a program structure file (".psf") when adding source files. See section 6.4.3.
