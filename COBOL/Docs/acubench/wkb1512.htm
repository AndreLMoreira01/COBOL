15.4.2 The Report Box

The report box is similar to the ACUCOBOL-GT standard frame control, except that you cannot assign a title to a report box. Similar to the report line, this report element may be useful in the header/footer areas, for example, to frame a title. You could also use it to frame a set of radio buttons. The report box Property window contains properties that are described in the previous section. Please refer to section 15.4.1, "Common Report Control Properties."
