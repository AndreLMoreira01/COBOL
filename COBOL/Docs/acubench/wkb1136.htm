11.7.3.1 Using the Event Editor from the Screen Designer window (standard controls only)

  1. From a Screen Designer window, double-click on the desired standard control. This displays the Add Paragraph dialog box, which contains a drop-down list of all the paragraph names previously generated by the Screen Designer for this program.
  2. You don't have to associate your control with a specific paragraph name, but if you want to, choose from this list. Notice that the Event Editor's "Control", "Message", and "Paragraph" entry fields are automatically filled in after you choose a paragraph name.
  3. Enter your procedural code for the event in the Event Editor window.
  4. Use the Build/Generate command to generate the event COPY files. This code is included the next time you compile your program.
