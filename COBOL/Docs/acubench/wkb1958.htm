19.4.5 Properties

Select Project/Properties to view and modify general and library properties for your project.

On the General tab, you can change the name of your project by typing a new name in the Project name field. The Project directory and Working directories items on the General tab are read-only.

The General tab also includes an area for remote server and port settings for use with thin client operations. These fields are pre-filled with the settings in the Project Settings/Project Information dialog. This dialog also has Apply to Compiler Settings and Apply to Runtime Thin Client Options check boxes. Set these check boxes to direct the workbench to automatically copy thin client server and port information to thin client compiler and runtime options. A Validate button lets you validate your connection to the named remote server.

Use the Library tab to specify the names of the objects to include in the library that is built for this program. See section 17.2.1, "Creating an Object Library," for a description of how to specify and build object libraries.
