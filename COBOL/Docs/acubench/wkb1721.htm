17.6.6 Creating Empty Files

The vutil function that allows you to create a new, empty file is accessed on the Create File tab. The create file function is equivalent to performing an OPEN OUTPUT on a file in COBOL. For a complete description of the vutil create function, see section 3.3 of the ACUCOBOL-GT User's Guide. For general information about Vision files, see section 6.1.3 of the ACUCOBOL-GT User's Guide. Create a new Vision file as follows:

  1. Specify the path and name of the file that you want to create in the "File name" entry box. You must specify a file name to enable the Start button.

    2. Caution: If you specify the name of an existing file, that file will be overwritten.

  2. By default, vutil creates the file in Vision Version 4 format. If you want the file to be a different Vision file format, enable the Vision file format check box in the Parameters area and select the desired version from the drop-down list. The "Option" field displays the command line that reflects the options chosen. You cannot directly edit the contents of this field.
  3. The Data area allows you to specify most aspects of the index file structure, including the keys. Alter the values or accept the defaults as desired. To define the keys, press the Define Key Information button. Use the New, Delete, and Delete All buttons to add or delete entries. Double-click in a field of the key to change its value. When finished, click OK.
  4. To create the file click Start. vutil creates the file and displays the file name in the status area in the lower half of the Create File tab.

Click Close to exit vutil.
