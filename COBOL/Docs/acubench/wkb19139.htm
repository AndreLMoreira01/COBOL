19.9.12.16 Options/Code Editor/Constant List

A description of the Code Editor's Constant List options appears in the following table:

Option Description

Sort by

Lets you determine how the Constant List box is sorted.

Occurrence in the program

Causes constant names to be sorted in order of their occurrence in the program.

Alphabetically

Causes constant names to be sorted alphabetically.

Value

Causes constant names to be sorted by value.

Show

Lets you determine what columns of information are displayed with constant names.

Value

Generates a column in the Constant List box that contains each constant's value.

Name of the COPY book

Generates a column in the Constant List box that contains the name of the COPY file in which each constant appears.
