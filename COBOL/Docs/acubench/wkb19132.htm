19.9.12.9 Options/Code Editor/General

A description of the Code Editor's General options appears in the following table:

Option Description

Line number pane width
x digits

Lets you modify the width of the Line Number pane. Enter a value from 1 to 6 in the entry box.

Line length x columns

Lets you modify the length of a line in the Code Editor. Enter a value up to 2048 in the entry field.

Save Record Delimiter as CR/LF

Lets you choose the carriage return/linefeed (CR/LF) ASCII end-of-line marker used in MS-DOS for your file.

Save Record Delimiter as LF

Lets you choose the linefeed (LF) ASCII end-of-line marker used in UNIX for your file.

Virtual Space

Lets you determine the editor's vertical block select behavior. Clear the check box if you want the vertical block select function to be limited to the width of the last line of the text block.
