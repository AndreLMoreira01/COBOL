4.9.3 Setting Visibility Options

The Visibility dialog allows you to customize the appearance of your report Property window by letting you decide which properties are actually listed and which ones are not. By default, all possible properties are listed in a report control's Property window.

Set report control property visibility options as follows:

  1. Select the Visibility category of the Report Writer options.
  2. Clearing the check box in the Visibility dialog for any given property eliminates it from every report Property window in which it would normally appear. You can also use the Set and Clear push buttons to set and clear properties in the Property window.
  3. If you check/uncheck one of the Show check boxes, you add an entire category of properties to the report Property window or eliminate that category from the Property window, respectively. Each property belongs to only one category.
