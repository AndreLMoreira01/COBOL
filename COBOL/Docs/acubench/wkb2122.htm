21.7 Debugger Pop-up Menu

Right-clicking anywhere in the debugger window creates a pop-up menu containing the following commands:

Menu Item Description

Undo

Reverses your most recent editing action.

Redo

Reverses your most recent undo action.

Cut

Removes the selection from the document and places it on the clipboard.

Copy

Places a copy of the selection on the clipboard.

Paste

Places a selection that has been cut or copied to the clipboard into the active document.

Find

Lets you find multiple occurrences of a text string in the active document.

Find Next

Advances you to the next occurrence of the selected item in the active document.

Toggle Bookmark

Toggles the appearance of a bookmark for the current line. After inserting bookmarks, you can move automatically to the next or previous bookmark in your code via the Bookmark toolbar buttons or the Bookmark submenu commands.

List Paragraphs

Lets you see each paragraph name in the active Code Editor file, along with the name of the COPY file in which it appears. When you select this command, the Paragraph List box appears.

List Variables

Lets you see each variable name in the active Code Editor file, along with its level number, picture value, and the name of the COPY file in which it appears. When you select this command, the Variable List box appears.

List Constants

Lets you see each level-78 constant name in the active Code Editor file, along with its value and the name of the COPY file in which it appears. When you select this command, the Constant List box appears.

Open Copy File

Opens the COPY file selected in the source code.

Quick Watch

Opens the Quick Watch window, which displays the selected variable and its value.

Add to Watch Window

Transfers the selected variable to the AcuBench integrated debugger's Watch window. This command is disabled if the program is not being run in the debugger.

Create Program Structure File

Creates a program structure file (".psf") for the file in the active Code Editor window.
