14.1 Creating a Report

In Chapter 13, we described the various pieces of the AcuBench integrated development environment that contribute to the Report Composer function. This chapter contains step-by-step guidelines for using these Report Composer components to build a report. This chapter also includes detailed descriptions of the sample reports that are available on the Acucorp product CD-ROM. Chapter 15 includes detailed information about the report sections and the report controls that you can use to determine a report's appearance. Helpful descriptions of report section and control properties also appear in Chapter 15.

When you want to create a report in AcuBench, a logical starting point is the definition of the data that you want processed and formatted into that report. After you have determined the information your report should contain, you can design its appearance. The following general steps describe how to create a report in AcuBench:

  1. Create or open the project and set up the work environment.
  2. Create and refine your data layout file (".dlt") in the workspace Data view.
  3. Create and refine the data set that corresponds to your data layout.
  4. Design your report in the Report Composer interface.
  5. Make one-time modifications to the generated code.
  6. Compile the program and run it.

The sections that follow provide the details.
