10.5 The Property Window

The Screen Designer's Property window is your interface for controlling the appearance and behavior of your selected screen form or control(s). Change the font in your Property window by right-clicking in the window and selecting Font. This command displays the Font dialog box, from which you can choose a different font. You can display or hide the Property window by selecting the View/Property Window command or by right-clicking in the window and choosing Hide. You can also move the window from its default position (right side of the workbench) or change it to a floating window by right-clicking in the window and choosing the Allow Docking command.

The Property window has three tabs: Alphabetic, Categorized, and Event. The Alphabetic tab displays all properties in alphabetical order. In the Categorized tab, properties are listed by category, for example, control appearance or behavior. Each property belongs to only one category. The Event tab contains all the events associated with the screen form or control. More information about events can be found in Book 2, Chapter 6 of the ACUCOBOL-GT documentation set.

When you open a new screen form or create a new standard control, the corresponding Property window contains a set of default property values, which you can change by clicking on the value and entering a new one. The workbench dynamically calculates other values in the Property window based on your control's initial location and size. You may sometimes click on a property to change it and cause a push button to appear in the Value column. Clicking on the push button displays a dialog box for that property. You may also see a drop-down box with several options for the property value. When you have selected multiple controls, the Property window contains the property items that the controls have in common.

The Property window's contents can be customized in the Tools/Options/Screen Designer dialog box. Default values can be changed in the Default portion of this dialog. The Visibility portion of the dialog allows you to control which properties are displayed in the Property window. Information about the Tools/Options/Screen Designer dialog box can be found in section 4.6, "Setting Screen Designer Options."

If you create an ActiveX control, the Property window contains a subset of that control's properties, including those for control name, size, location, and tab order. You can view other properties by right-clicking in the control and choosing ActiveX Control Properties.

After you save your screen as an ".stf" file, you can obtain a printed version of its Property window. Simply open the ".stf" file and print it using a word processing print facility.
