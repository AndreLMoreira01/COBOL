9.3.15.4 Replacing a text string in all files

Replace in Files allows you to search for and replace a string in any readable file on the system for which you have "write" permission. The search string can include wildcard and special characters to build a search expression. Note that replacements cannot be undone.

Perform a search and replace operation using the Replace in Files command as follows:

  1. Choose the Edit/Replace in Files command. The Replace in Files dialog box appears. If you selected text in an active Code Editor file, that text automatically appears in the "Find what" text box.
  2. If you did not select text, type the text string that you want to find in the "Find what" text box, or select a previous search string from the drop-down list.
  3. Select Regular expression if you want to enter a regular expression as the item to be found rather than a specific text string. Build the search expression by entering characters and choosing operators.
  4. Enter the string you want to use as a replacement in the "Replace" field, or select a previous replacement string from the drop-down list.
  5. Select a file type from the "File types" drop-down list.
  6. To match the case of the text string exactly as typed, click the Match case check box.
  7. Select the location of the files in which you want to replace strings. You can search the entire open workspace, or you can search in a specified directory.
  8. To search for the string in files of a specific directory, type the full directory path in the "Look in" field, or click the Browse button to locate a directory. Check the Include subdirectories check box to include the contents of the main directory's subdirectories in the search.
  9. To start the search and replace action, click the Replace All button.
