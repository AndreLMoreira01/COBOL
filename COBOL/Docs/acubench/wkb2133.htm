21.13 Report Composer Pop-up Menu

Right-clicking anywhere in a Report Composer window creates a pop-up menu containing commonly used editing and navigation commands.

Menu Item Description

Cut

Removes the selection from the report and places it on the clipboard.

Copy

Places a copy of the selection on the clipboard.

Paste

Places a selection that has been cut or copied to the clipboard into the active report.

Delete

Removes the selection from the active Report Composer window.

Select All

Selects all the elements in the active Report Composer window.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Property Window

Lets you display or hide the Property window.

Component Toolbox

Toggles the display of the Screen Component Toolbox.

Drag-and-Drop

Toggles the display of the Drag and Drop window.

Section Controller

Opens the Section Controller interface, which lets you define the sections your report contains.

Custom Color

Lets you customize the Report Composer's color palette.

Change Prefix

Lets you change the active report's unique prefix.

Lock Controls

Lets you lock or unlock a control's position on the report.

Zoom

Displays a submenu containing a choice of viewing options, in percentages.

Generate WTF Document

Lets you generate the ".wtf" file for the active report.
