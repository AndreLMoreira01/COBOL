21.1.6 Structure View: Screen or Report Pop-up Menu

Right-clicking on a screen name in the Structure view of the Workspace window creates a pop-up menu containing the following commands:

Menu Item Description

Open form_name

Opens the named screen or report in the appropriate designer.

Delete form_name

Deletes the named screen or report from the program.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
