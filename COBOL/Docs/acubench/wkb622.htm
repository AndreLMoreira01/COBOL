Moving a Program or Screen to Another Project

In the Structure view, locate the program or screen to be moved and drag-and-drop it on the target project node. The program or screen is placed in the appropriate subnode. If a program or screen of the same name is already present in the target project, the component is not moved and a message is displayed.
