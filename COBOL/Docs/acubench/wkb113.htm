11.2 Using Screen Templates

You can add screen templates to the File/New dialog box via the Template portion of the Tools/Options/Environment dialog box. The Add function displays the Add New Template File dialog box, with which you locate the desired screen template file. An icon for your template appears in the "Customize Template" box in the Tools/Options/Environment/Template interface and in the File/New/Screen dialog box. After you have added a template to your project, you can modify the template file information or delete the file via the Tools/Options/Environment/Template interface. More information about the Tools/Options/Environment/Template dialog box can be found in section 4.4.2, "Setting Template Options."

You can also create your own graphical or character-based screen template for your project and store it in a folder of your choosing.

Use the following steps to create a screen template:

  1. Design your screen.
  2. Right-click in your screen and select the Generate STF Document command. In the Save As dialog box, navigate to the folder where you want your template stored.
  3. To add the template to your project, right-click on the Screen icon in the workspace Structure view and choose the Add Screen command. In the Add Screen to Program dialog, navigate to your template file and open it. Once the screen template is in your project, you can alter it to suit your individual needs.
