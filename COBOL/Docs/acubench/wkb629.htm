6.4.3 Upgrading an Existing Source Program

When you add an existing COBOL program to an AcuBench project, you can upgrade the program to include AcuBench tags and, at the same time, create a program structure file. The creation of a program structure file is essential if you want to use the workbench's design tools and automated code generation facilities.

To add a program without converting it to an AcuBench program, disable the option cited in the next section in step 1, and use the Add/Remove function in the usual way (see section 6.3.4).

To add a program and create a program structure file without inserting tags in the source file, create a program as described in section 6.4.1, disable the option cited in the next section in step 1, and add the program with the Add/Remove function.

You can also create a program structure file for your program via the Add/Remove Files dialog. Set the check box for the file in the "Create .psf" column in the Files in Project area or use the dialog's Set PSF push button for a selected file. A Create Program Structure File command also appears in the Code Editor's right-click pop-up menu.

You can add and upgrade an existing program as follows:

  1. In the Tools/Options/Environment/Miscellaneous dialog, enable the Automatically create a program file when adding a source file to the project's source folder option. (It is advised that you disable this option after you have converted all of the programs that you want to add.)
  2. Launch the Add/Remove Files function by clicking the Project toolbar button. Click on the Source tab to make it active.
  3. From the drop-down list attached to the "Insert into" field at the bottom left of the Add/Remove Files dialog, select the project to which the program will be added.
  4. Locate and select the source file to be added in the "Files to be added" section of the dialog.
  5. After the desired source file has been selected, click the Add button and then the OK button. AcuBench creates a program structure file and inserts tags into the source file.
