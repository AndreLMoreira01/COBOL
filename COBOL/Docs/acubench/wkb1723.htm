17.6.8 Extracting Records

The vutil extract record function is accessed on the Extract tab. The extract function allows you to display records stored in the specified file. The target file cannot be encrypted. If you attempt to use the extract function to view a record in an encrypted file, the message "encrypted, extract not allowed" is displayed. For a complete description of the vutil extract function, see section 3.3 of the ACUCOBOL-GT User's Guide. Extract and display records using the following steps:

  1. Specify a data file name in the "Data file" entry box. You must specify a file name to enable the Start button. You can use the ellipsis button adjacent to the entry field to navigate to a file. If during your use of vutil you have already specified the file, you can easily select the file name again from the "Data file" drop-down list.
  2. In the Parameters section, select the options you want. The "Option" field displays the command line that reflects the options chosen. You cannot directly edit the contents of this field. For a complete description of the extract parameters, see section 3.3 of the ACUCOBOL-GT User's Guide.
  3. Click the Start button to begin the extract. Matching records are displayed in the output area in the lower half of the Extract tab.

    4. If you did not set values for the key value or the number of records, you will be prompted for the values after the extract begins. A small entry field labeled "Input value" is displayed above the output area. To determine what value is being prompted for, look at the name of the last field in the output display. For example, if the last line reads "key value:," you need to enter a key value in the "Input value" entry field.

To interrupt an active extract process, click the Stop button (the Start button becomes a Stop button after the extract begins).

Click Close to exit vutil.
