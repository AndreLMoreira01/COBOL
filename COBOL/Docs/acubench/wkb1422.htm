14.4 Character-based Sample Reports

The AcuGT/sample/reports directory also contains sample reports for a character-based environment. The procedures for building character-based reports are for the most part the same as those for graphical reports. Some graphical report controls are not available for character reports, including the report grid, image, occurs, and table controls.

Modifications to generated code are also slightly different in a character environment. Specifically, the PERFORM statements in the Acu-Initial-Routine paragraph in a report's ".cbl" file have a different format. A PERFORM Acu-reportname-PRINT-T statement causes the report to print. With the PERFORM Acu-reportname-PRINT-TOFILE-T statement, the report is written to disk.
