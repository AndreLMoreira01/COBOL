19.9.12.10 Options/Code Editor/Format

A description of the Code Editor's Format options appears in the following table:

Option Description

Source format

Lets you select the source code format whose appearance you want to change.

Modify code

Lets you direct AcuBench to place a fixed text string in columns 73-80 (ANSI source format only) when a line of code is added or modified. Enter the desired text string in the entry field.

Enable foreground
and background
color settings

Lets you enable or disable all the color functions described in this dialog box.

Columns

Lets you change the column width of the editing area selected in the list box on the left.

Background

Lets you determine the background color of the editing area or text type selected in the list box on the left.

Foreground

Lets you determine the foreground color of the text type selected in the list box on the left.

Font

Opens the Font dialog box, in which you can change the font, style, and size of your text.
