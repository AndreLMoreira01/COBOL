2. Create and refine your data layout file (".dlt") in the workspace Data view.

If you are using an existing ".dlt" file, do the following:

  • In the Data view of your project, right-click the project icon and select Add FD/SL.
  • Browse to the directory containing your ".dlt" file, select the file, and click Open. You return to the data file.
  • If necessary, click the plus sign next to the project icon to expand the project, then double-click the data layout icon to open the File Designer.

If you are using creating a new ".dlt" from existing FD and SELECT COPY files, do the following:

  • In the Data view, right-click the project icon and select New FD/SL from the pop-up menu.
  • Change the FD/SL name from FD1 to a file name of your choice, then select Import from files and click OK.
  • In the Import FD/SL Files dialog, click on the browse button. By default, AcuBench browses the FD directory of your project. Navigate to the location of the FD file you wish to use, select it, and click Open.
  • Control returns to the Import FD/SL Files dialog box, which is automatically pre-filled with the selected SL file name. Click OK to open the selected FD/SL in the File Designer interface.

In the File Designer interface, do the following:

  • Select the IO Handling tab.

    • This interface is used to define the I/O paragraphs associated with your data layout file and code them in the Event Editor. For reports, this generally means adding two paragraphs, one to START the file READ based on a key value and one to READ each record for the report.

  • To create a new I/O paragraph, click Add to create a paragraph title area at the bottom of the listed paragraphs. This button is located on the toolbar about halfway down the dialog screen.
  • The newly created Item cell has the title "Start-1". Double-click this cell and type a new paragraph description for the code that will START the file. The Acucorp sample programs use Before Master Print Loop.
  • The new Value cell also has the title "Start-1". Double-click this cell, and type a new paragraph name. This is the name of the paragraph that is generated in the ".evt" COPY file. The sample programs name this paragraph Before-Master-Print-Loop.
  • Click the ellipsis push button at the end of the Value cell to open the Event Editor, with the cursor positioned in the named paragraph.

    • The following code template could be used for this paragraph: 

    MOVE LOW-VALUES TO [KEY] 
START [FILE] KEY > [KEY] 
READ [FILE] NEXT RECORD 
     AT END MOVE "10" TO [FILE-STATUS] 
END-READ.
  • Repeat the three previous steps to create an End-Master-Print-Loop paragraph. The purpose of the End-Master-Print-Loop in a report is to drive the report with READ NEXT statements. The following code template could be used for this paragraph: 
    • READ [FILE] NEXT RECORD 
     AT END MOVE "10" TO [FILE-STATUS] 
END-READ
  • Select the Use User-defined I/O Paragraphs radio button to cause data set code generation to include the paragraphs you define in this interface.
  • Close the File Designer, then right-click the data layout icon in the Data view and select Generate FD/SL.

    • New FD and SELECT COPY files are created in the project's FD directory.
