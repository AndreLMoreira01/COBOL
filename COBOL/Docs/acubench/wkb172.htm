17.1 ACUCOBOL-GT Utilities

AcuBench provides access to several utilities that are bundled with ACUCOBOL-GT®. These utilities are accessed from the workbench Tools drop-down menu. Selecting a utility from this menu causes the workbench to display a tool-specific graphical interface that makes it easy to specify options and to run the utility. These stand-alone interfaces can remain open even after you close AcuBench.

This chapter contains information about the AcuBench graphical interfaces for the following ACUCOBOL-GT utilities:

  • cblutil, the object file utility
  • vio, the file transfer utility
  • logutil, the transaction log file utility
  • vutil, the Vision files utility
  • alfred, the indexed file record editor

For more detailed information about these utilities and the options they provide, refer to Chapter 3 of the ACUCOBOL-GT User's Guide.

Four other ACUCOBOL-GT utilities are started from the workbench Tools menu: AXDEFGEN, NETDEFGEN, acu4glfd, and xml2fd. These utilities are discussed in the respective user's manuals noted in the following paragraphs.

With ACUCOBOL-GT's ActiveX Definitions Generator (AXDEFGEN), you can add ActiveX controls and COM objects to your ACUCOBOL-GT applications. The AXDEFGEN dialog box lists all the ActiveX controls and other COM objects that are registered on your system. For detailed information about the AXDEFGEN utility interface, refer to Chapter 3 in A Guide to Interoperating with ACUCOBOL-GT.

The ACUCOBOL-GT .NET Definitions Generator (NETDEFGEN) utility lets you view and select .NET assemblies for translation to a COBOL COPY file. For details about this utility, refer to section 3.5.4.9, "NETDEFGEN Utility," in A Guide to Interoperating with ACUCOBOL-GT.

A Windows utility called acu4glfd helps you create file descriptions and SELECTs. Detailed information about using this utility can be found in Chapter 6 in the Acu4GL User's Guide.

The xml2fd utility helps you derive an FD and SELECT statement from an XML file. More information about this utility appears in section 8.2, "Working with XML Data," in A Guide to Interoperating with ACUCOBOL-GT.
