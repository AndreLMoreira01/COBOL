16.2.6 The XFD Screen

XFD tab options include:

Option Description

Field Name

Lists the file descriptor name and each of its field names and picture values.

File Directive

Displays a default XFD name. You can change the name if desired.

Set XFD directive for each data item

Set the directives for your FD in this section.

Name Directive

Enter the desired field name in the entry field.

Use Group Directive

Set this check box if you want to apply the USE GROUP directive to a group item.

Data Type Directive

Set this check box to use the following set of radio buttons to set a field's data type.

Alpha

Enable this radio button to treat the selected data item as alphanumeric.

Binary

Enable this radio button to treat the selected data item as any type of alphanumeric data.

Date

Enable this radio button to store the selected data item as a date. Enter the date format in the entry field to the right of the Date radio button.

Numeric

Enable this radio button to treat the selected data item as an unsigned integer.

Var-Length

Enable this radio button to designate the data item as having a variable length.

When Directive

Set this check box to include multiple record definitions in the XFD file. Enter the WHEN directive syntax in the associated entry field. Click the ellipsis button to display the Expression Builder dialog, where you can also construct WHEN directive syntax.

Table Name

Enter a table name in the When directive entry field, if desired.

Comment Directive

Set this check box to include comments in your XFD file. Enter your comment in the associated entry field.
