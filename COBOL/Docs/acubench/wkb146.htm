4. Design your report in the Report Composer interface.

  • Open a new blank graphical or character-based report from the File/New/Report interface. Note that some report controls are not available for a character report.
  • Right-click in the Report Composer window and select the Drag-and-Drop command or use the View/Drag-and-Drop menu command. In the Drag and Drop interface, choose the desired report control type and then select the appropriate FD items for your report. Use a right mouse click to drag these items to the Report Composer window. You can also drag report controls to the Report Composer window from the Report Component Toolbox.
  • Generate the program. A report (".rpt") file and a Report Composer Section in the source (".cbl") file are created.
