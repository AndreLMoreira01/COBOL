4.5.3 Setting Tabs Options

With the Code Editor Tabs options, you can direct that the editor support the tabs in your source code or automatically convert the tabs in your code to spaces. You can also set up to 32 custom tab stops.

Set any needed Tabs options as follows:

  1. Select the Tabs category of the Code Editor options.
  2. You can choose to display a special character to indicate the location of a tab in your code by inserting a non-blank character in the "Display tab character with" entry field. The default character is "^".
  3. To change the default size of a tab (8), enter another number between 1 and 64 in the "Tab size" entry field. If you have enabled the Keep Tabs radio button and you press the Tab key or the editor encounters a tab character in the code, the cursor shifts forward the number of places indicated in the "Tab size" entry field. When you delete a tab character, the editor shifts the cursor back the number of places indicated in this entry field.
  4. Select either ANSI Format or Terminal Format in the "Source format" entry field. You can establish different sets of tab stops in this dialog box for ANSI source format and terminal source format files.
  5. If you want the Code Editor to support the insertion of tabs using the default tab size, set the Keep Tabs radio button. The "Tabs" entry field and list box are disabled when "Keep Tabs" is set.
  6. If you want the editor to convert the tabs in your code to spaces or if you want to set individual tab stops in your code, set the Insert Spaces radio button.
  7. To add tab stops, enter each desired setting in the "Tabs" entry field and click Add. Your new settings appear in the list box. You can set up to 32 tab stops. To delete a tab stop, select the setting you want to delete, and click Delete. The setting disappears from the list box. Note that the default tab size specified in the "Tab size" entry field resumes after your last custom tab setting. For example, if your last custom tab setting is at column 70, and your default tab size is 8, then your next tab stop would be at column 78.

Tabs may also be used to move blocks of text in your code. Highlight the lines of code you want to move and press the Tab key. The entire block of text shifts the number of places indicated in the "Tab" entry field.
