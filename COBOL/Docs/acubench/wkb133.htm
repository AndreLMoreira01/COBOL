13.2 The Report Composer Window

The primary interface in AcuBench for designing your report is the Report Composer window. You use the File/New/Report command to open the New Report dialog. You can choose either a blank graphical report template or a blank character report template from this interface. Change the default form name and prefix, if desired. You can then choose to add the report to an existing program or create it with a new program in one of the projects in the associated drop-down box. When you click OK, the Report Composer window opens.

You can create a portion of either a graphical report or a character report in this window. Note that by default the interface opens with three main report sections-Page Header, Detail, and Page Footer-in which you can arrange your report elements. With the Section Controller interface (described in the next section), you can add other areas to the Report Composer interface, for example, a report header or footer. You can control the size of individual report sections by positioning the cursor at the bottom of a section and dragging the border. A page header appears at the beginning of each page of the report. The code that is generated into the report file (".rpt") automatically checks for page breaks, so you don't need to add code to manage the printing of page headers. The report's detail section constitutes the main body of the report, which includes the information you want communicated in your report. A page footer appears at the end of each page of the report.
