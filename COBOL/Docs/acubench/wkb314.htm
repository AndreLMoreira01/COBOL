3.5.1.2 Project directories

When you create a project, you can change the name of any project subdirectory. If your project will not make use of any automated code generation tools, you can choose to omit a subdirectory altogether (by making that entry field blank). To modify a subdirectory name or eliminate it, select the More Info button on the File/New/Project dialog box.

Note that renaming or eliminating a subdirectory does not change the name of the corresponding folder in the Workspace window. However, the logical link between the Workspace folder and the corresponding directory is updated so that the correct related files are displayed.

If you change a name or eliminate a directory you may need to adjust some of the default project settings. To view and set these options, open the Project Settings dialog by right-clicking on the project node in the File view, and select Settings from the pop-up menu.

  • If you change the name of the "object" directory, or eliminate it, you must change:
    1. the argument for Compiler Options/Standard Options/"Name of object file (-o)".

      2. The default setting is: ".\object\@.acu". You should change "object" to the new name of the object directory, or remove it if you have eliminated the directory.

    2. the Environment definition of CODE_PREFIX.

      3. CODE_PREFIX is defined in the Environment tab of the Project Settings dialog. You should replace "object" with the new name of the object directory, or remove it if you have eliminated the directory.

  • If you change the name of the "list" directory, or eliminate it, you must also change the argument for Compiler Options/Listing Options/"Name of list file (-Lo)".

    • The default setting is: ".\list\@.lst". Change "list" to the new name of the listing directory, or remove it if you have eliminated the directory.

  • If you change the name of the "copylib" directory, you should also change the environment definition of COPYPATH.

    • COPYPATH is defined in the Environment tab of the Project Settings dialog. In the definition, replace "copylib" with the new name of the object directory, or remove it if you have eliminated the directory.
