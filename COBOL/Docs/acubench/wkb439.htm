4.6.1 Setting General Screen Designer Options

General options for the Screen Designer let you determine the attributes of the grid that underlies the screen form, and specify how controls interact with that grid. The grid underlying the screen form corresponds to the screen's CELL SIZE property, and is, by default, set to 10 pixels (width) by 10 pixels (height).

You can set the following General Screen Designer options:

  1. Select the General category of the Screen Designer options.
  2. To set the CELL WIDTH for your screens, enter a number between 3 and 20 in the "Grid width" entry field (default is 10).
  3. To set the CELL HEIGHT for your screens, enter a number between 3 and 20 in the "Grid height" entry field (default is 10).
  4. If you select the Snap to grid check box, any control you place on the grid automatically aligns on grid boundaries. Even if this option is selected, you can still select a control and move it in any direction one pixel at a time by using the keyboard's arrow keys.
  5. If Auto alignment is selected, and you position a control underneath another control in the Screen Designer, the new control's left boundary automatically aligns with the existing control's left boundary when those boundaries are within 5 pixels of each other horizontally.
