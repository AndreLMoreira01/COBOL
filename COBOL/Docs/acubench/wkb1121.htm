11.5 Setting Screen Form Properties

Once you have opened a new screen form, you need to set its properties. The Property window contains the properties that you set to command a screen's appearance and behavior. You can find your desired properties on the Alphabetic or Categorized tab. To set or change a property, enter or select your property item preferences in the corresponding property Value column. Clicking in the Value column may present you with a drop-down box of choices or a push button to display a dialog box. The Event tab displays the events generated by the screen. More information about specific properties and events can be found in Book 2 of the ACUCOBOL-GT documentation set.

Resize layout manager

ACUCOBOL-GT includes a layout manager facility that can help manage various aspects of a screen's appearance. When attached to a screen, a layout manager can help determine the placement and size of the controls in that screen. Individual layout managers have their own rules regarding how controls are sized and placed.

By default, a screen does not have a layout manager attached to it. For these screens, controls are sized and positioned according to properties like LINE, COLUMN, SIZE, and LINES.

AcuBench currently supports a resize layout manager, which handles the appearance of a screen's controls when that screen is resized. The resize layout manager automatically resizes and repositions controls when the associated window is resized. Controls use a Layout Data property to tell the resize manager which automatic actions should be applied. Only controls with a non-zero Layout Data property value are affected by the resize manager. Information about layout managers in general and the ACUCOBOL-GT resize layout manager in particular can be found in section 4.8, "Layout Managers," in ACUCOBOL-GT User Interface Programming. Refer to section 11.6.4, "Layout Data Control Property," in this manual for more information about the Layout Data control property.

In AcuBench, you enable the resize layout manager for a particular screen by setting its Layout Manager property to LM_RESIZE in the screen's Property window. (The default setting is "None".) When Layout Manager is set to LM_RESIZE, AcuBench automatically generates a COPY file called "lmresize.def", which includes a unique value for the Layout Manager Handle item in the Property window. You can change this value, but you should note that when you have multiple resizable screens, this value must be unique for each screen to avoid a runtime error.

Note that in order for the layout manager to function properly, you must also change the screen's Auto Resize property to FALSE.

The resize layout manager facility is demonstrated in an AcuBench sample project located in the Support area of the Acucorp Web site. To download the program, open your Web browser and navigate to www.acucorp.com/support/public/sample_programs/index.php. Select the "Graphical User Interface (GUI) Sample Programs" category from the drop down list, and click on "layoutmgr.zip".
