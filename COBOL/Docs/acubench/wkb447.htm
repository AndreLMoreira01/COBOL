4.9.1 Setting General Report Writer Options

General options for the Report Writer let you determine Report Composer window grid attributes, how report control elements interact with the grid, and the unit of measure used to size report components. The grid underlying the report form corresponds to the grid underlying the screen form, and is a method of specifying the default CELL SIZE for new reports.

Set any desired General Report Writer options as follows:

  1. Select the General category of the Report Writer options.
  2. To set the width of your grid cell in pixels, enter a number in the "Grid width" entry field (default is 10).
  3. To set the height of your grid cell in pixels, enter a number in the "Grid height" entry field (default is 10).
  4. If you select the Snap to grid check box, any report control you place on the grid in the Report Composer window automatically aligns on grid boundaries. Even if this option is selected, you can still select a report control and move it in any direction one pixel at a time by using the keyboard's arrow keys.
  5. If you select the Auto alignment check box and then position a report control underneath another control, the new control's left boundary automatically aligns with the existing control's left boundary when those boundaries are within 5 pixels of each other horizontally.
  6. In the Unit frame, select the Inch radio button if you want the Report Composer function to measure report components in inches. Select the Centimeter radio button is you want the unit of measure to be centimeters.
