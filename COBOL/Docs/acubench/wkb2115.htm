21.1.13 Data View: Data Layout File Pop-up Menu

Right-clicking on a data layout in the Data view of the Workspace window creates a pop-up menu containing the following commands:

Menu Item Description

View file_name.fd

Opens the named ".fd" file in the Code Editor.

View file_name.sl

Opens the named ".sl" file in the Code Editor.

Open data_layout

Opens the named data layout in the File Designer.

Event Editor

Opens the Event Editor.

Generate FD/SL

Generates new ".fd" and ".sl" files from the data layout file.

Make XFD File

Generates an XFD file from the selected file description.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Delete data_layout.dlt

Deletes the named data layout file from the project.

Delete From Disk

Deletes the selected data layout file from the disk.

Save data_layout.dlt

Saves the named data layout file.

Refresh data_layout

Causes the named data layout file (".dlt") to be rebuilt from its corresponding ".fd" and ".sl" files.

Properties

Displays the Program Properties dialog.

Version Control

Displays the list of user-defined version control commands. This item appears on the menu only if version control commands have been defined.
