9.3.10 Working with Paragraph Names

AcuBench maintains a list of the paragraphs currently declared in your program. You can view the Paragraph List box via the Edit/Advanced/List Paragraphs command or the List Paragraphs push button in the Editor toolbar. You can also access this list by right-clicking anywhere in the Code Editor window and choosing List Paragraphs.

The Paragraph List box is a modeless window that allows you to work in the Code Editor window while the list is displayed. In addition to the name of each paragraph, the list box contains a column showing the COPY file in which it appears. You can control whether the COPY file column is visible and how the paragraphs are sorted via the Tools/Options/Code Editor/Paragraph List interface, which is described in section 4.5.6, "Setting Paragraph List Options." You can also determine the sort order by clicking on the column headings in the list box.

You can easily insert into your code any paragraph name that appears in the Paragraph List box. Position the cursor in your code where you want the paragraph name to appear. In the Paragraph List box, select a name, right-click in the list box, and choose Paste. Note that you can change the cursor's position in the code and paste a paragraph name multiple times. You can also double-click a name to insert it at the cursor's location.

Right-clicking in the list box gives you access to other functions in addition to Paste. The Copy to Clipboard command lets you copy a paragraph name to the Windows clipboard. You can move the cursor automatically to the entry point of a selected paragraph in the Procedure Division via the Go to Definition command. Finally, if you modify your code while the list box is open, you can update the Paragraph List by using the Refresh command.
