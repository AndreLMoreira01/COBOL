21.2 Toolbar Pop-up Menu

Right-clicking anywhere on a toolbar creates a pop-up menu containing the following commands:

Menu Item Description

Standard

Lets you hide or display the Standard toolbar.

Project

Lets you hide or display the Project toolbar.

Editor

Lets you hide or display the Editor toolbar.

Debug

Lets you hide or display the Debug toolbar.

Align

Lets you hide or display the Align toolbar.

Launch Bar

Lets you hide or display the Launch Bar toolbar.

Customize

Opens the Customize dialog that lets you customize existing toolbars and define new toolbars.
