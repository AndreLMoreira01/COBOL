3. Create and refine the data set that corresponds to your data layout.

  • Create a new blank program from the File/New/Program interface, or open the program to which to plan to add a report.
  • Right-click on the program's Data Set node in the Structure view, and select New Data Set. Because the data set name defaults to "DS1", you might want to use the Property window to assign a more descriptive name.
  • In the Property window, click in the Value field next to Data Layout, then use the drop-down box to select the data layout file that you have just created.
  • Right-click anywhere in the Data Set Designer and choose Referenced FD/SL. In the Data Set Member Files dialog, set the data set Open Mode to Input, then click OK.
