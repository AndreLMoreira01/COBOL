9.3.15.2 Replacing a text string in the active document

When you need to replace a particular text string in the active Code Editor file, the Code Editor allows you to quickly locate that string wherever it appears and make the needed change. Replace a text string as follows:

  1. Move the cursor to the place in the file where you want the search to begin.
  2. Choose the Edit/Replace command. The Replace dialog box appears. If you selected text in the document before choosing the Replace command, that text automatically appears in the "Find what" text box.
  3. If you did not select text before choosing the Replace command, type the text string that you want to find in the "Find what" text box or select from a drop-down list of up to 15 text strings previously entered for the Find or Replace operation.
  4. Type the replacement text in the Replace with entry field or select from a drop-down list of text strings previously entered for the Replace operation.
  5. To match the case of the text string exactly as typed, click the Match case check box.
  6. Select Regular expression if you want to enter a regular expression as the item to be found rather than a specific text string. Build the search expression by entering characters and operators.
  7. Select the Wrap around search check box to direct that the search operation continue at the beginning of the active document after it reaches the end of that document.
  8. To direct the search operation to ignore hyphens and underscores, select the Ignore '-' and '_' check box.
  9. To determine the direction of your search, select either the Up or Down radio button.
  10. Click the Find Next, Replace, or Replace All button:
    • Clicking Find Next lets you confirm each change. The Code Editor searches for the specified text string and selects the first occurrence. Click Replace to change the text, or click Find Next to leave it unchanged and search for the text string's next occurrence.
    • Clicking Replace All changes all text string occurrences without asking for confirmation.
  11. The Code Editor informs you when it has completed your search. Click Cancel when all your replacements are made.

You can leave the Replace dialog box open while you continue to use the workbench.
