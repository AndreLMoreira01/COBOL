15.3.2 The Page Header and Footer

The page header is printed once at the beginning of each page of a report, and a page footer is printed once at the bottom of each page. When you open a new Report Composer window, it contains a page header section and a page footer section by default. You can delete either the page header or the page footer section via the Section Controller interface. A report can have only one page header or footer.
