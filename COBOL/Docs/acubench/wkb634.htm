6.4.8 Executing a Program

In the Workspace window, right-click on the program name, program source file, or program object file and select Execute or Execute with Parameters from the pop-up menu.

If you have previously selected the Use Thin Client command, and you have not yet created an alias file entry for the program to be executed, you are prompted to do so via the Create Alias Entry dialog. For more information about the Create Alias Entry dialog, refer to the description of the Create Alias command in section 19.4.6, "Create Alias." This dialog allows you to designate an alias entry name and the working directory. Server and port entry fields are derived from entries in the Project Properties dialog. The command line entry field is derived from the Project Settings/Runtime tab Thin Client Options.
