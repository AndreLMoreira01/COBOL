16.2.5 The IO Handling Screen

I/O handling options include:

Option Description

Use Default Code Generation

Set this radio button if you want AcuBench to use its default automatic code generation behavior.

Use User-defined I/O Paragraphs

Set this radio button if you want AcuBench to generate code for the I/O paragraphs defined in the I/O Handling tab.

Do Not Generate Any I/O Paragraphs

Set this radio button if you don't want AcuBench to generate I/O paragraphs.

User-defined I/O Paragraphs

A list of user-defined I/O paragraphs. AcuBench lists 12 paragraph names by default. Three buttons appear in the upper right corner of the list.

Delete

This button deletes the selected I/O paragraph.

Clear Paragraph

This button clears the Value column of a selected paragraph.

Add

This button adds a new paragraph to the user-defined I/O paragraph list.

Any command not included in the default list of user-defined I/O paragraphs can be created with the Add push button.

After you define an I/O paragraph, click in the Value column of the desired paragraph to display the Event Editor. The Add Paragraph dialog initially contains a paragraph name suggested by the workbench. You can change the paragraph name here if you want and then code the paragraph in the Event Editor.
