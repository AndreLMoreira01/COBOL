19.2.15 Bookmark

The Bookmark command lets you insert a bookmark into the Bookmark pane for the selected line. Once bookmarks are in place, you can instantly move the cursor to the next or previous bookmark in your code via the Bookmark toolbar buttons or the Bookmark submenu commands. The Bookmark pane must be open for an icon to be visible. However, navigation between bookmarks still functions whether or not the Bookmark pane is displayed.

To save bookmarks, you need to save the workspace while the bookmarked Code Editor file is open. If you close the Code Editor window before saving the associated workspace, your bookmarks are lost.

The Bookmark submenu commands are described in the following table:

Command Description

Toggle Bookmark

Sets a bookmark or removes an existing bookmark in the active Code Editor document.

Next Bookmark

Moves the cursor to the next bookmarked line in the active Code Editor document. This command is available only if you have inserted bookmarks.

Previous Bookmark

Moves the cursor to the previous bookmarked line in the active Code Editor document. This command is available only if you have inserted bookmarks.

Clear All Bookmarks

Clears all your bookmarks in the active Code Editor document. This command is available only if you have inserted bookmarks.
