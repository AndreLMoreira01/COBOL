9.3.11 Working with Variable Names

The workbench also maintains a list of the variables currently declared in your program. You can view the Variable List box via the Edit/Advanced/List Variables command or the List Variables push button in the Editor toolbar. You can also access this list by right-clicking anywhere in the Code Editor window and choosing List Variables.

The Variable List box is a modeless window that allows you to work in the Code Editor window while the list is displayed. In addition to the name of each variable, the list box contains columns for each variable's level number, picture value, and the COPY file in which it appears. You can control which columns are visible, how the variables are sorted, and whether blank FILLER data items are listed in the box via the Tools/Options/Code Editor/Variable List interface, which is described in section 4.5.7, "Setting Variable List Options." You can also determine the sort order by clicking on the column headings in the list box.

You can easily insert into your code any variable name that appears in the Variable List box. Position the cursor in your code where you want the variable name to appear. In the Variable List box, select a name, right-click in the list box, and choose Paste. Note that you can change the cursor's position in the code and paste a variable name multiple times. You can also double-click a name to insert it at the cursor's location.

Right-clicking in the list box gives you access to other functions in addition to Paste. The Copy to Clipboard command lets you copy a variable name to the Windows clipboard. You can move the cursor automatically to the declaration of a variable in Working-Storage via the Go to Definition command. If your program is running in Debug Mode, you can add a variable to the debugger's Watch window with the Add to Watch Window command. Finally, if you modify your code while the list box is open, you can update the Variable List by using the Refresh command.
