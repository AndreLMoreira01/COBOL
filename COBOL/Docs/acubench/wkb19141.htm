19.9.12.18 Options/Screen Designer/General

A description of the Screen Designer's General options appears in the following table:

Option Description

Grid width

Lets you set the width of a grid cell in pixels, from 3 to 20.

Grid height

Lets you set the height of a grid cell in pixels, from 3 to 20.

Snap to grid

Lets you enable or disable the function that forces controls placed on the screen to align on grid cell boundaries.

Auto alignment

Lets you enable or disable the function that automatically aligns the left boundary of a screen control with that of the control above it, if the left boundaries are within 5 pixels of each other horizontally.
