8.3.1 File Menu Commands

The following File menu commands assist you with source code generation and editing:

New. Lets you open a new source file.

Open. Lets you open an existing source file.

Close. Closes the active document.

Save. Saves the contents of the active document.

Save As. Lets you save a copy of the active document.

Save All. Lets you save the contents of all open documents.

Page Setup. Lets you define headers, footers, and some page attribute settings for the active document.

Print Setup. Lets you change the printer and certain print properties.

Print Preview. Lets you view the active document in its print format before you print it.

Print. Lets you send the active document to the printer.
