18.1.2 Debugger Output

Debugger messages appear in the Output window, which usually appears at the bottom of the AcuBench screen. You can also send tracing information to the Output window by selecting Trace Option/Trace to Debug Window from the Debug menu.

When you direct output to the Output window, you should also send that output to a file using the "-e" compiler option. You can establish an error file as follows:

  1. Open the Project Settings window and select the compiler's Standard Options from the Catalog drop-down list box.
  2. Select Name of error file (-e), then specify a name for the error file in the entry field to the right of the check box.
  3. Click OK.
