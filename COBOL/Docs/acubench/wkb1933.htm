19.2.12 Replace

Lets you replace items in the text with other items. When you choose this command, the Replace dialog box appears. The Replace dialog options include:

Option Description

Find what

If you selected text before choosing the Replace command, displays that text string. You can also enter the item you want to find, or select from a list of up to 15 text strings previously entered for the Find operation.

Replace with

Lets you enter the item with which you want to replace the text string.

Match case

Causes the search operation to be case-sensitive.

Regular expression

Lets you enter a regular expression as the item to be found rather than a specific text string.

Wrap around search

Lets you direct the workbench to continue the search operation at the beginning of the active document after it reaches the end of that document.

Ignore "-" and "_"

Lets you direct the workbench to disregard hyphens and underscores while searching for a text string.

Direction

Lets you designate the direction that you want the search to take relative to your current location in the text. The default direction is down.

Find next

Advances the search to the next occurrence of the item.

Replace

Replaces the current occurrence of the text string.

Replace all

Replaces all occurrences of the text string without confirming the change.

Cancel

Click Cancel to exit the Replace dialog.
