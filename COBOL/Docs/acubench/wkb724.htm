7.6.2 Listing Options

The Listing options allow you to capture many types of compilation information in a listing file. Set listing options as follows:

  1. Open the Project Settings or Program Compile Options dialog box and select Listing Options from the Catalog drop-down list.
  2. Observe that the name of the listing file is displayed in the field beside the "Name of listing file" check box. All listing information is placed in this file.
  3. Select the desired options.
  4. If you enable the "Set page length of number lines" check box, enter a page length in the adjacent entry box.
