7.3.2.6 Setting or changing a comment on a variable

You can set or modify a comment on a variable as follows:

  1. In the "Configuration file entries" box, locate and click on the variable that you want to comment. The variable's value and existing comment text, if any, is displayed in the adjacent "Value" and "Comment" boxes.
  2. Click in the Comment box and enter or edit text. There is no need to enter the (#) before the comment. The CFE inserts it for you. (The symbol will be visible in the CFE the next time you select the variable.
  3. When you finish modifying the file, click OK to add the comment to the configuration file, save the file, and close the CFE.
