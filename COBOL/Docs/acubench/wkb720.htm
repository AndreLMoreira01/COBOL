7.4 Setting Project Compile Options

Project compile options are the set of compiler options that are applied when the project is built (compiled). Project compile option settings are stored in the project file (".pjt").

To set project compile options, use the following steps:

  1. Launch the Project Settings dialog by selecting the Project/Settings command or by right-clicking on the desired project node in the workspace window and then selecting Settings from the pop-up menu.
  2. Check the Project tree and "Settings For" values at the left of the dialog to ensure that you are setting options for the desired project and mode. A title field in the dialog contains the name of the project whose settings you are currently changing.
  3. Select a category of options from the Catalog drop-down list box.
  4. Browse the option list and enable, disable, or specify the desired options.
  5. Click OK when complete.

You can use the "Additional options" entry field, located to the right of the Catalog drop-down list, to directly specify any compile option. Simply enter the compile flag followed by any required value.
