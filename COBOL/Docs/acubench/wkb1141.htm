11.8.1.1 Adding an item to a menu

  1. In the Menu Designer dialog box, double-click the menu text marker. A new text field appears in place of the menu text marker. If the menu already has menu items, the new item appears below and at the same level as the selected menu item.
  2. If you want to add an item before another menu item, highlight the menu item and press the Insert item before button at the top of the Menu Designer window. You can also right-click in the dialog box and select the Insert Item Before command. The new item appears above and at the same level as the selected menu item.
  3. Enter a name for the menu item. Press the Enter or Tab key.
  4. To indicate that a letter in a menu or submenu item is a hot key and should be underlined, place the ampersand (&) character before the letter you want underlined.
  5. If you want to insert a separator, select a menu item. Right-click in the designer and use the Insert Separator command. A separator is inserted before the selected menu item.
