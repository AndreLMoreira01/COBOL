8.4.1 Standard Toolbar

The Standard toolbar is a palette of command buttons for most commonly used word processing functions. You can display or hide the Standard toolbar by choosing the View/Toolbar/Standard command. You can also display or hide the Standard toolbar by right-clicking in the toolbar and choosing Standard, or by selecting Customize and then Standard in the Customize dialog box. The Standard toolbar contains the following commands:

New. Generates the New dialog box, from which you can open a new project, program, screen file, and source file. It also provides access to the File Designer interface.

Open. Lets you open an existing file.

Save. Saves the active file using its current file name.

Save All. Lets you save all open files.

Print. Lets you send the active file to a printer.

Print Preview. Lets you see how the active file looks before you print it.

Cut. Removes the selected item and places it on the clipboard.

Copy. Places a copy of the selected item on the clipboard.

Paste. Places a selection that has been cut or copied to the clipboard into the active file.

Undo. Reverses recent editing actions.

Redo. Reverses recent undo operations if you have performed no other actions since the undo.

Workspace. Toggles the view of the workspace in the workbench.

Output Window. Toggles the view of the Output window in the workbench.

Property Window. Toggles the view of the Property window in the workbench.

Component Toolbox. Toggles the view of the Screen Component Toolbox in the workbench.

About. Displays workbench identification and copyright information.
