14.4.4 Using Report Headers and Footers and the Report Radio Button and Check Box Controls

To build this report, follow the instructions for graphical Report 1d in section 14.2.6, "Using Report Headers and Footers and the Report Radio Button and Check Box Controls." Open a blank character report rather than a graphical one and name it Report2d. Add report controls as described for the graphical report, and modify the Acu-Initial-Routine paragraph using one of the PERFORM statements in section 14.4.
