9.1.2 Moving to a Specified Location

You can automatically move the cursor to the beginning of a document, the end of a document, or any specified line in a document using the commands in the Edit/Go To submenu. You can also position the cursor in your source code at the previous or next source code compilation error listed in the Output window.

To move to the beginning or end of the document, choose either Begin or End from the Edit/Go To submenu, depending on where you want to go.

You can move to an individual line in the document as follows:

  1. From the Edit menu, choose Line from the Go To submenu. The Go To Line dialog box appears.
  2. Enter the number of the line to which you want to move. The dialog box displays the total number of lines in the document. The line number you enter must fall within the line count range displayed in the dialog box.
  3. Click OK.

Use the following steps to move to the previous or next compilation error:

  1. In the Output window's list of compilation errors, double-click on an error line to move the cursor to that line in the source code.
  2. From the Edit menu, choose Previous Error from the Go To submenu to move the cursor to the source code location of the previous compilation error in the Output window list. Choose Next Error to move the cursor to the location of the next compilation error in the list.
