19.2.11 Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

The Find in Objects options include:

Option Description

Find what

If you selected text before choosing the Find in Objects command, displays that text string. You can also enter the text string you want to find, or select from a list of up to 15 text strings previously entered for the Find operation.

Project

Lets you specify the project you want searched.

Target

Lets you specify the programs you want searched. If you want to search multiple programs, separate the program names in this entry field with semicolons.

Match case

Causes the search operation to be case-sensitive.

Match whole word only

Causes the search operation to match an entire word as a stand-alone case rather than as part of another, larger word.

Find Property Name

Causes the search operation to include screen and report property labels.

Output to pane (Find in Files)

Directs the search results to be displayed in the Find in Objects Output window. Double-clicking a text string in the Output window opens the corresponding file for editing purposes.

Find

Begins the search.

Cancel

Click Cancel to exit the Find in Objects dialog.
