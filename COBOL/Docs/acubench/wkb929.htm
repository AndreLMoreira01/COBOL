9.3.15.3 Finding a text string in all files

The Code Editor lets you locate all the occurrences of any text string in all the files on your computer's drives whether or not the files are open. As the Code Editor finds an occurrence matching your description, it displays the location, line number, and statement for it in the "Find in Files" Output window pane. To move to the location of a found string, double-click its listing in the "Find in Files" Output window pane.

The Edit/Find in Files command allows you to search for a string in any readable file on the system. As with the Find command, the search string can include wildcards and special characters to build a search expression.

You can perform a search using the Find in Files command as follows:

  1. Choose the Edit/Find in Files command. The Find in Files dialog box appears. If you selected text in an active Code Editor file, that text automatically appears in the "Find what" text box.
  2. If you did not select text, type the text string that you want to find in the "Find what" text box, or select a previous search string from the drop-down list.
  3. Select Regular expression if you want to enter a regular expression as the item to be found rather than a specific text string. Build the search expression by entering characters and choosing operators.
  4. Select a file type from the "File types" drop-down list.
  5. To match the case of the text string exactly as typed, click the Match case check box.
  6. To search for the string in files of a specific directory, type the full directory path in the "Look in" field, or click the Browse button to locate a directory. Check the Include subdirectories check box to search the specified directory's subdirectories.
  7. To start the search, click Find. When the search is complete, a message is displayed in the "Find in Files" Output window. To stop a search, select Stop Find in Files from the Edit menu. The Stop Find in Files command is available only during an active search.

The results of the search are displayed in the Output window. Double-clicking on any file listed in the Output window causes the file to be opened to the line on which the search string was found.
