10.4 The Drag and Drop Window

In addition to selecting controls in the Screen Component Toolbox and drawing them on the screen, you can use the Drag and Drop interface to add graphical elements to your screen.

To access the Drag and Drop window, select Drag-and-Drop from the View menu or use the Screen Designer right-click menu. The Drag and Drop window lists all of the data items available for your screen, including the field name, PIC information, and any OCCURS clauses.

To use the Drag and Drop option, first select a control type from the drop-down box in the top, right portion of the window. Next, select one or more data items to associate with that control. Shift-click to select multiple contiguous data items, or Ctrl-click to select multiple non-contiguous items. When you finish selecting the data items, let up on the mouse button, then click and drag any selected item to the screen form. In most cases, AcuBench adds the control(s) that you have specified to the screen form. For some complex controls, AcuBench opens an Autoload window, allowing you to make further refinements before adding the control to the screen. More information about the Autoload function can be found in section 11.6.3, "Autoload Controls." Once the control appears on the screen form, you can align or edit the controls in any way you like.

By default, the Drag and Drop interface shows "All Names": all FD fields, Working-Storage items, and Linkage items. To narrow down the available options, use the drop-down box in the top, left portion of the window to select a specific FD, Working-Storage only, or Linkage items only.
