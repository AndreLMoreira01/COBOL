21.1.5 Structure View: Report Node Pop-up Menu

Right-clicking on a Report node in the Structure view of the Workspace window creates a pop-up menu containing the following report-related commands:

Menu Item Description

View program_name.rpt

Opens the named file in the Code Editor.

New Report

Lets you create a new report for the program.

Add Report

Lets you add an existing report (".wtf ") to the program.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
