8.3.2 Edit Menu Commands

The following Edit menu commands assist you with source code editing:

Undo. Reverses previous editing actions.

Redo. Reverses the most recent undo operation if you have performed no other actions since the undo.

Cut. Removes the selection from the active document and places it on the clipboard.

Copy. Places a copy of the selection on the clipboard.

Paste. Places a selection that has been cut or copied to the clipboard into the active document at the cursor location.

Delete. Removes the selection from the active document.

Select All. Selects all the elements in the active document.

Find. Lets you find multiple occurrences of a text string in the active document.

Find Next. Advances you to the next occurrence of the selected item in the active document.

Find in Files. Lets you locate text strings in files stored on your drives.

Replace. Lets you replace items in the text with other items.

Go To. Lets you relocate your cursor at a specific destination in the active document. Also allows you to proceed to the previous or next source code compilation error listed in the Output window.

Bookmark. Lets you insert a bookmark into the Bookmark pane for the selected line, move to the previous or next bookmark, or clear all bookmarks from the active document.

Advanced. Lets you perform the following tasks: view the paragraph, variable, constant, and COPY file lists for the active document; view defined code templates; open COPY files; insert files; find the scope of a variable; and find a verb block match.

Create/Update CFD. Creates a Control Flow Diagram (CFD) for the active program if one does not exist. If a CFD already exists for the active program, this command updates the existing CFD to reflect any recent changes in the program.
