15.2 The Report

Like the Screen Designer's screen form, the report itself has a set of properties that you can modify to change a report's overall appearance. The report is named in the Property window, and some of its fundamental characteristics, such as style and orientation, are defined. The following paragraphs describe the report's properties. A property marked with an asterisk (*) can have its default value set in the Tools/Options/Report Writer/Default dialog.

(Name)

The default name of the report is Report1. You can change this name if desired. The name shown in the Property window also appears in the Output File Name property listing, and it appears in a number of paragraph names in the generated code.

Num Columns (N-Top)*

This property describes the number of columns in an N-Top style report. AcuBench always applies the default setting of "1" for an N-Top report. Num Columns is disabled for a Standard style report. Refer to the Report Style report property later in this section for more information about Standard and N-Top style reports.

Output File Name

As mentioned earlier, this file name is derived from the (Name) property value described above. This is the name of the file that is written to disk.

Output File Variable

The Value cell has an ellipsis push button that opens the Select Variable interface. If a variable is selected, subtle modifications to the generated code occur, such that it references the Output File Variable instead of the Output File Name directly. This property has no default value.

Paper Size

The Value cell push button opens the Paper Size interface. This dialog allows you to choose the dimensions of the printed paper and whether the printed page has a portrait or landscape orientation. It also allows you to set headers, footers, and margins. The default value for this property is "Letter".

Report Style

A drop-down box for this property allows the selection of "Standard" (the default) or "N-Top". If you want to print or print preview your report, set this property to "Standard". When you set this property to "N-Top", your HTML report is created so that it can be viewed in a browser that has a horizontal scroll bar. Because "N-Top" is a display orientation only, reports with this property are not meant to be printed.

Target Browser*

A drop-down box allows the selection of "Specify Internet Explorer" (the default) or "Standard HTML".

Title*

The Title property entry appears in the report's window title when the report is displayed in Internet Explorer. It also appears in the first title line of the print preview. This property has no default value. Note that displaying the report in the ACUCOBOL-GT Web Browser control does not preserve the title. (The title of the Web Browser control window is set with the Title property for the screen in the Screen Designer.)

Watermark*

Use this property to select a graphic to set as a background image for a report. Only ".bmp" and ".jpg" image types can be used.

Watermark Style

A drop-down box allows the selection of "None", "Center", or "Tile". A Watermark Style of "None" (the default) causes the code generator to ignore any value set for Watermark. If you wish to have a watermark, select the "Center" or "Tile" property value. In general, you may want to center large images and to use "Tile" for small ones.
