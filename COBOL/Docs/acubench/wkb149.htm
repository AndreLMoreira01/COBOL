14.2 Graphical Sample Reports

Some sample graphical reports have been created to help you get started with the Report Composer function. Your Acucorp product CD-ROM contains several sample graphical and character-based reports. Each report builds on the functions learned in the previous report, ranging from a simple label/entry field report at first to complex reports that use tables and grids. You can find the samples in the "acugt/sample/reports" directory. Our samples all use the same data files, namely "sales.dat", "sales.vix", "SalesData.fd", and "SalesData.sl", which can be found in the reports/Data and FD folders.

The first two steps in creating these reports are the same for all the samples. They correspond to the first two report creation steps outlined in section 14.1.
