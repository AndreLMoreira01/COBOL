19.3.1 Toolbars

The Toolbars command allows you to hide or display the workbench toolbars, including the Standard, Project, Editor, Debug, and Align toolbars and the Launch Bar. When you select this command, a submenu containing a list of the toolbars appears. Click on a menu item to toggle its display. A check mark appears next to the menu item when the toolbar is displayed. You can also toggle the display of any toolbar by right-clicking on the toolbar.

The Toolbars command also allows you to customize the appearance of your toolbars or create new toolbars. When you select the Customize submenu item from Toolbars, the Customize dialog box displays. You can toggle the display of the various toolbars in the Toolbars tab and select their components in the Commands tab. These two tabs also allow you to name and create a new toolbar. The name subsequently appears in the View/Toolbars submenu. A launch bar of non-workbench external applications can be created in the Tools tab. You can also display the Customize dialog box by right-clicking on any toolbar.
