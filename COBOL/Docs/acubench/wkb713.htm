7.3.2.4 Adding a pre-defined variable

To add a pre-defined variable (a variable that appears in the "Configuration file entries" list), locate and click on the desired variable. Set the value as described in the following section. You can add user-defined variables as well. See the "To add user-defined variables" section that follows.
