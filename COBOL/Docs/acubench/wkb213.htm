21.1.1 Workspace Window: Open Area Pop-up Menu

Right-clicking in any part of the Workspace window not occupied by a tree element creates a pop-up menu containing the following workspace commands:

Menu Item Description

Save Workspace

Saves all open files in the current workspace.

Close Workspace

Closes the current workspace.

Open Project

Displays the Open Project dialog, which lets you open a project (".pjf") file.

Set Active Mode

Displays the Set Active Mode dialog, which lets you change the current mode (the set of compile and runtime settings).

Build Workspace

Compiles every file in the workspace that has been modified since the last build, and their dependents.

Rebuild Workspace

Unconditionally recompiles all files in all projects in the workspace.

Reparse All

Reparses all source files in the workspace.

Allow Docking

Lets you toggle the appearance of the Workspace window as either floating or docked.

Hide

Lets you hide the Workspace window from view. To restore it to view, select Workspace from the View menu.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
