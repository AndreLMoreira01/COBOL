21.1.10 File View: Folder Pop-up Menus

Right-clicking a Source, Screen, Report, Copylib, Object, List, Resource, or FD folder in the File view of the Workspace window creates a pop-up menu containing the following commands:

Menu Item Description

New File

Lets you add a new file to the project. The default file type (source, screen, report, COPY, and so on) depends on the folder that you have selected.

Add/Remove Files

Opens the Add/Remove Files dialog that is used to add and remove files from the project.

Empty

Removes all files in the selected folder.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.

Properties

Displays the selected folder's properties.

Version Control

Displays the list of user-defined version control commands. This item appears on the menu only if version control commands have been defined.

The right-click menu for the Remote folder is similar to the other folders in the File view, but truncated. Only the Empty, Find in Objects, Properties, and Version Control options, as described in the table above, appear.
