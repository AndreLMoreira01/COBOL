19.8.6 Sequence Number

Allows you to re-number your source code and display the new numbering in the Sequence Number area. This command is not available if your code is in terminal source format.

The Sequence Number dialog box options include:

Command Description

Action

These options determine the appearance of sequence numbers in the Sequence Number area.

Make Starting No./Interval

Enable the Make radio button to set the starting number and interval size for the numbers in the Sequence Number area.

Clear

Enable this radio button to clear all sequence numbers from the Sequence Number area.

Line range

These options determine whether your entire source file or a selected portion of the file is re-numbered.

All

Enable this radio button to re-number your entire source code file.

Selection

If you have selected a block of text in your code, this radio button is automatically enabled.

Lines

Enable this radio button if you want to re-number specific lines of code. Enter the range in the entry fields.

OK

Click OK to save your sequence number options.

Cancel

Click Cancel to exit the Sequence Number dialog.
