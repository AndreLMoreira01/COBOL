7.6.7 Mapping Options

The Mapping options cause the compiler to produce simple reports about the program being compiled. Only one report can be produced at a time. Mapping options prevent the compiler from creating an object file. Mapping options usually prevent errors or warning messages from being displayed. For a complete description of all Mapping options, see Book 1, Chapter 2, "Compiler and Runtime" in the ACUCOBOL-GT documentation set.

You can set Mapping options as follows:

  1. Open the Project Settings or Program Compile Options dialog box and select Mapping Options from the Catalog drop-down list.
  2. To save the results to a file, enable the Name for output file ... check box and enter a file name in the adjacent entry box.
  3. Select a list map report type.
  4. Select a list map report style.

Note that when any mapping option is used, the compiler ignores text that appears between the EXEC and END EXEC pre-compiler statements.
