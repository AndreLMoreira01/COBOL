10.3.1 Standard Controls

The standard controls and graphical elements that appear in the Screen Component Toolbox are described in the following table. Detailed information about standard controls can be found in section 12.2, "Standard Controls."

Element Description

Selector

Clears your selection of a control in the Screen Component Toolbox and lets you select, move, and size another screen control in the screen form.

Bar

Lets you draw a horizontal or vertical line.

Bitmap

Lets you specify a display area for a bitmap graphic ("*.bmp", "*.dib", "*.jpg", "*.jpe", "*.jpeg").

Check Box

Lets you create a box with an accompanying text label that the user can click to select or uncheck an option. Typically, you use a check box for an option that can be turned on and off. You can also use check boxes for options that users can select in combination (such as bold, italic, and underline formatting).

Combo Box

Lets you create a combined edit box and list box control. The user can either select an item from the list or enter text in the edit box.

Date Entry

Lets you create a rectangular field which displays date and/or time information in the format that you specify. Some formats include a graphical calendar from which users can select a date.

Entry Field

Lets you draw a rectangular field in which the user can enter text or data.

Frame

Lets you draw a box used for grouping related controls together. You can put controls into frames to separate them visually from other controls.

Grid

Lets you create a two-dimensional table of data fields. Each element of this table, called a cell, can hold text or a bitmap, or both.

Label

Lets you enter a text label (such as a caption under a graphic) that cannot be changed by the user.

List Box

Lets you create a rectangle containing a list of items from which the user can choose. If many items are in the list, the user can scroll through the list using the scroll bar.

Menu Designer

Lets you create a menu bar for a screen. Menus can consist of menu titles, menu items, and separator bars. When you double-click a Menu Designer icon in your screen, the Menu Designer dialog box appears. This dialog box lets you add, delete, move, and rename menu items.

Push Button

Lets you draw a rectangle with an accompanying text label that the user can click to perform an action (such as OK or Cancel).

Radio Button

Lets you create a circular button with an accompanying text label that the user can click to select an option and simultaneously deselect another option. Unlike check boxes, normally only one radio button can be selected from a group.

Scroll Bar        

Lets you draw a rectangle containing a scroll box and a scrolling arrow at each end. The user typically uses a scroll bar to navigate through information in a list, but a scroll bar can also serve as an input device.

Status Bar

Lets you create a status bar at the bottom of your screen.

Tab

Lets you create a control that combines a box with a tab for a control that looks like a file folder. The user clicks on any tab to bring it forward.

Toolbar Designer

Lets you create a collection of controls arranged in a toolbar. The toolbar appears below the screen's menu bar (if present), or at the top of the window if no menu bar exists. When you double-click a Toolbar Designer icon in your screen, the Toolbar Designer window appears. You can put any type of control in a toolbar except for a menu or another toolbar

Tree View

Lets you create lists of hierarchical data, such as the contents page of a Windows help file or the view you get when you open Windows Explorer. This list is indented to show the relationships in the data.

Web Browser

Lets users view the main window of the Microsoft Internet Explorer on a Screen Designer-created screen. It provides the functionality for displaying Web pages containing HTML, scripting, and ActiveX control and Java applet content. This control also hosts OLE document objects, supports OLE hyperlinks, and allows users to view Windows objects such as folders and files.
