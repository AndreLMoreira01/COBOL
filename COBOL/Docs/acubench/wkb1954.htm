19.4.1 Add/Remove Files

To add files to or remove files from a project, select Project/Add/Remove Files.

The Add/Remove Files dialog options include:

Option Description

Source, Screen, Report, Copylib, Object, List, Resource, FD tabs

You can add all types of project files from the Add/Remove Files dialog. Use the tabs to toggle between the different types of files in your project. Each Add/Remove Files dialog is the same for each type of project file.

Files List

Lists the files available to add from the directory selected in the directory list on the right. You can select one or more files from this list to add to your project. You can browse different directories by opening or closing the folders displayed in the directory list.

Note: To select multiple files from this list, select the files you want to add to your project while pressing and holding the Ctrl key.

Files of type

Defines the type of file you are looking to add to your project from the selected directory. This drop-down list is updated automatically according to the file type tab that is active. For example, if you are adding or removing source files on the Source tab, this list displays source file type extensions. You can select a different file type if desired.

Drives

Displays the drive you are searching for files to add to your project. You can change drives by selecting another drive from this drop-down list.

Files in project

Lists the files that already exist in your project. This list displays file types specific to the active file type tab. For example, if you are working in the Screen tab, only screen file types are listed. If you want AcuBench to create a program structure file (".psf") for the file you are adding, set the associated check box in the Create .psf column. To remove a file from this list, select the file you want to remove and click Remove.

Note: To select multiple files from this list, select the files you want to remove from your project while pressing and holding the Ctrl key.

Automatically parse program for COPY files

Check this check box if you want AcuBench to search for COPY statements in all of the added source files. When this option is selected, the Copy Path button is active, allowing you to modify the COPYPATH environment variable.

Copy Path

This button is active only when the "Automatically parse program for COPY files" check box is selected. Click Copy Path to define the COPYPATH environment variables in your project. For more information, see the Copy Path dialog below.

Insert into

Defines the project to which you want to add files. You can select a different project from the drop-down list if desired.

Add

Click Add to add the files selected in the Files List to your project.

Add All

Click Add All to add all of the files listed in the Files List to your project.

Remove

Click Remove to remove the selected file or files from the Files in Project list.

Remove All

Click Remove All to remove all of the files listed in the Files in Project list.

Set PSF

Click Set PSF to set or clear selected check boxes in the "Create .PSF" column in the Files in Project area.

Network

Click Network to map an additional network drive to your computer. The newly added drive displays and becomes available for use in the Drives drop-down list.

Note: This is a standard Windows drive mapping utility. The drives you map using AcuBench are also available in the normal Windows environment.

OK

Click OK when you are finished adding or removing files to exit the Add/Remove Files dialog.

Cancel

Click Cancel to exit the Add/Remove Files dialog without saving your changes.

Add Copy Path dialog

The Add Copy Path dialog options include:

Option Description

Directory

Displays the directory paths that you can use in your COPYPATH environment variable. You can browse for different directories by opening or closing the folders displayed in the directory list.

Drives

Displays the drive you are searching for files to use with the COPYPATH environment variable. You can change drives by selecting another drive from this drop-down list.

Default copy path(s)

Lists the default copy paths already set for your project. To remove a copy path from the list, select the copy path and click Remove.

<<Add

Click Add to add the copy path selected in the Directory list to your COPYPATH environment variable. You can add only one path at a time.

Remove>>

Click Remove to remove a selected copy path from the Default copy path(s) list. You can remove only one path at a time.

OK

Click OK when you are finished adding a copy path to exit the Add Copy Path dialog.

Cancel

Click Cancel to exit the Add Copy Path dialog without saving your changes.

Network

Click Network to map an additional network drive to your computer. The newly added drive displays and becomes available for use in the Drives drop-down list.

Note: This is a standard Windows drive mapping utility. The drives you map using AcuBench are also available in the normal Windows environment.
