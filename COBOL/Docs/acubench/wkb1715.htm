17.6 Using vutil

AcuBench provides a graphical interface to the functions performed by the ACUCOBOL-GT Vision file utility vutil. vutil operates on Vision indexed files and can be used to extract file structure information, extract records, and rebuild corrupted files. This chapter describes only the AcuBench interface to vutil. For a complete description of vutil, see section 3.3 of the ACUCOBOL-GT User's Guide.

Note: vutil does not make use of the runtime configuration file. Settings made in a runtime configuration file do not affect vutil in any way.

To open vutil, select the Vision File Utility command from the Tools drop-down menu. The Vision File Utility dialog includes several tabs, each of which provides access to a different vutil function. To select a function, simply click on the corresponding tab.

Because vutil operates on Vision indexed files, each vutil function requires that you specify the name of a Vision file. As long as the vutil dialog is open, the name of each file specified is added to a drop-down list associated with the file name entry field. You can easily select a previously specified file from the drop-down list. To navigate to a file, click on the ellipsis button adjacent to the entry field.

To stop a vutil function that is in progress, click the Stop button (the Start button becomes a Stop button when a vutil function is in progress). If you attempt to select another vutil tab while the current action is in progress, the message "Vision File Utility is running. Please wait." is displayed.

To close the vutil interface, stop or allow the current action to complete and click Close.
