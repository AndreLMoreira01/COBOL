9.1.3 Using Bookmarks

Bookmarks are markers that the Code Editor allows you to insert in your source code for reference purposes. Bookmarks also provide fast, simple navigation through your code. These markers do not alter your code. They simply allow you to move around in your file more easily. The Code Editor displays an icon in the Bookmark pane to indicate that a line is bookmarked.

To save bookmarks, you need to save the workspace while the bookmarked Code Editor file is open. If you close the Code Editor window before saving the associated workspace, your bookmarks are lost.

Bookmarks also function as AcuBench debugger breakpoints in your code. More detailed information about breakpoint behavior can be found in Chapter 18, "The AcuBench Integrated Debugger."

Bookmarks are only visible in the Bookmark pane, so that Code Editor window component should be open if you want to see the marks. However, you can still move easily among your bookmarked lines of code, even if you cannot see the marks. Navigation between bookmarks is not affected by icon visibility.

You can insert a bookmark in your code as follows:

  1. Display the Bookmark pane if you want the icon(s) to be visible. If the pane is not currently displayed, select the View/Bookmark Pane command.
  2. Go to the line you wish to bookmark and select anywhere on that line.
  3. From the Edit menu, choose the Bookmark/Toggle Bookmark command or press the Toggle Bookmark push button in the Editor toolbar. A bookmark icon appears next to the selected line in the Bookmark pane.

To move the cursor to the previous bookmarked line in your code, choose the Edit/Bookmark/Previous Bookmark command or press the Previous Bookmark push button on the Editor toolbar. To move the cursor to the next bookmarked line of code, choose the Edit/Bookmark/Next Bookmark command or press the Next Bookmark push button on the Editor toolbar.

To clear all the bookmarks from the active document, choose the Edit/Bookmark/Clear All Bookmarks command or press the Clear All Bookmarks push button on the Editor toolbar.
