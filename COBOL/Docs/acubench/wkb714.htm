7.3.2.5 Setting or changing the value of a variable

You can set or modify the value of a variable as follows:

  1. In the "Configuration file entries" box, locate and click on the variable you want to add. Its value displays in the "Value" area. Any associated comments display in the "Comment" box.
  2. Click on the Value label. If the variable has a pre-defined set of legal values, these display in a list. Click on a value to select it. If the variable does not have a set of pre-defined legal values, then double-click in the white space immediately below the label and enter the value (integer, floating-point, or character) that you want to set for this variable.
  3. Click OK to save the entry and close the CFE.
