21.1.7 Structure View: Data Set Node Pop-up Menu

Right-clicking on a Data Set node in the Structure view of the Workspace window creates a pop-up menu containing the following data set-related commands:

Menu Item Description

View program_name.prd

Opens the named file in the Code Editor.

Open

Opens the Data Set Designer.

New Data Set

Opens the Data Set Designer and creates a new data set.

Referenced FD/SL Files

Opens the Data Set Member Files dialog.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
