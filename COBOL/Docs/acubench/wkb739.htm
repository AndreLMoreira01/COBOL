7.7.3 Deleting Environment Variables

In the Environment tab of the Project Settings dialog, you can delete an environment variable as follows:

  1. In the workspace window, right-click on the desired project node and select Settings from the pop-up menu. The Project Settings dialog is displayed. Click on the Environment tab.
  2. Select the desired environment variable from the "Environment Variables" list.
  3. Click the Delete button (the center button of the three located in the top-right corner of the "Environment Variables" box, showing a red "X"). The selected environment variable is removed from the list.
  4. Click OK to close the Project Settings dialog.
