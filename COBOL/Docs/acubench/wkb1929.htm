19.2.8 Find

Lets you find multiple occurrences of a text string. The Find command is not enabled for the Screen Designer. When you choose this command, the Find dialog box appears. The Find dialog options include:

Option Description

Find what

If you selected text before choosing the Find command, displays that text string. You can also enter the item you want to find, or select from a list of up to 15 text strings previously entered for the Find operation.

Match case

Causes the search operation to be case-sensitive.

Regular expression

Lets you enter a regular expression as the item to be found rather than a specific text string.

Wrap around search

Lets you direct the workbench to continue the search operation at the beginning of the active document after it reaches the end of that document.

Ignore "-" and "_"

Lets you direct the workbench to disregard hyphens and underscores while searching for a text string.

Direction

Lets you designate the direction that you want the search to take relative to your current location in the text. The default direction is down.

Find next

Advances the search to the next occurrence of the item.

Mark all

Directs the workbench to place a bookmark at each line in which the text string occurs.

Find all

Directs the search operation to find all occurrences of the text string and list them by line number in the "Find in Files" Output window pane.

Cancel

Click Cancel to exit the Find dialog.
