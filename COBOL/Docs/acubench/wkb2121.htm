21.6.1 Paragraph, Variable, Constant, and COPY File List Pop-up Menus

Right-click anywhere in the Paragraph or Constant List boxes to see a pop-up menu containing the following commands:

Menu Item Description

Paste

Pastes the selected item name to the current cursor position in the source code.

Copy to Clipboard

Copies the selected item name to the clipboard.

Go to Definition

Opens the file containing the definition of the selected item, and positions the cursor at the start of the line where the definition begins.

Refresh

Refreshes the list.

The right-click menu for the Variable List is similar to the one described above, but contains the following additional option (just before Refresh):

Menu Item Description

Add to Watch Window

Transfers the selected variable to the AcuBench integrated debugger's Watch window. This command is disabled if the program is not being run in the debugger.

The right-click menu for the COPY File List is also similar to the basic menu, but contains one additional command at the top of the menu:

Menu Item Description

Open Copy File

Opens the selected COPY file.
