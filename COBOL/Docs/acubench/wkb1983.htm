19.6.11 Auto Step

Auto Step lets you execute "step" commands repeatedly until the program's end. You can change the speed at which the command executes, from "1" (slowest) to "9" (fastest) via the AutoStep Properties dialog.

The AutoStep Properties dialog options include:

Command Description

Standard

Lets you set a standard speed for Auto Step command execution.

Level

Select a number from 1 (slowest) to 9 (fastest) to determine Auto Step command speed.

Speed

Displays the speed in milliseconds/step for the level selected (for "1", the speed is 2700 ms/step, and for "9", the speed is 300 ms/step). This field is read-only.

Customize

Lets you set a custom speed for Auto Step command execution.

Speed

Enter the speed at which you want the Auto Step command to execute, in milliseconds/step.
