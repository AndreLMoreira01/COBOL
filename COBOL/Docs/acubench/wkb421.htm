4.4.2 Setting Template Options

The Template options allow you to:

  • bypass or use the File/New dialog when creating new programs, screens, reports, files, or file descriptors
  • add or remove source, screen, or report templates

The "Customize Template" section is used for adding, removing, or modifying source, screen, or report file templates. Source file templates are discussed in section 9.3.14, and screen templates are discussed in section 11.2. Report templates are described in section 14.2.9. All files added using this dialog are displayed in the File/New/File, File/New/Screen, or File/New/Report template group, depending on the template type.

Follow these steps to set Template options:

  1. Select the Template category of the Environment options.
  2. "Use File/New" options: Toggle the check boxes in the "Use New/File dialog when creating" group to control when the File/New dialog is displayed. When you de-select an option, the File/New dialog does not appear when you opt to create a new element of the specified type. Instead, a default element type is automatically created. The defaults are:

    Program       

    Standard Program (a program structure file with an empty screen)

    Screen

    Blank Graphical (an empty graphical screen)

    Report

    Blank Graphical (an empty graphical report)

    File

    Source Template (a file with COBOL divisions and AcuBench tags)

    FD/SL

    Blank (launches the File Designer, bypassing the Import option)

  3. "Customize Template" interface: To add new templates to the File/New dialog, first select the type of template from the "Template for" drop-down box.
  4. Use the buttons in the "Customize Template" area to modify the set of file templates offered in the File/New/File, File/New/Screen, or File/New/Report dialog. Click Add to specify the name of a template to be added to the dialog. Select a template in the list and click Delete to remove a template. Select a template and click Modify to change the template name, file name, or description of an existing template. You cannot change the set of default templates using this interface.
