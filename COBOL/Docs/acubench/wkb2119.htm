21.5 Property Window Pop-up Menu

Right-clicking anywhere in the Property window creates a pop-up menu containing the following commands:

Menu Item Description

Font

Lets you change the font and size of the text in the Property window.

Allow Docking

Lets you toggle the appearance of the Property window as either floating or docked.

Hide

Lets you hide or display the Property window.
