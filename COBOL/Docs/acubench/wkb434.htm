4.5.6 Setting Paragraph List Options

The Paragraph List options allow you to specify how the Code Editor's Paragraph List box is sorted, and to determine which columns of information are displayed in the box.

You can set Paragraph List options as follows:

  1. Select the Paragraph List category of the Code Editor options.
  2. "Sort By" options: If you want paragraph names to be sorted in the Paragraph List box in order of their occurrence in the program, select the Occurrence in the program radio button. If you want paragraph names listed alphabetically, choose the Alphabetically radio button. You can also sort COPY file names alphabetically and have paragraph names sorted by occurrence within each COPY file by choosing the Occurrence within alphabetic sort of copybooks radio button. Finally, you can sort COPY file names alphabetically and have paragraph names sorted alphabetically within them by selecting Alphabetically within alphabetical sort of copybooks.
  3. "Show" option: If you want the Paragraph List box to display the names of the COPY files, in addition to the names of the paragraphs, select the The name of the copybook option.
