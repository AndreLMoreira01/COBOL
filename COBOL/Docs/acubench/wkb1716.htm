17.6.1 Increasing the Maximum File Record Size

The vutil augment function is accessed on the Augment tab. The augment function lets you increase the maximum record size of a Vision file. This option is useful if you want to add fields to a record without having to rebuild the entire data file. For a complete description of the vutil augment function, see section 3.3 of the ACUCOBOL-GT User's Guide. Increase the maximum record size of a Vision file as follows:

  1. Specify a data file name in the "Data file" entry field. You must specify a file name to enable the Start button. You can use the ellipsis button adjacent to the entry field to navigate to a file. If during your use of vutil you have already specified the file, you can easily select the file name again from the "Data file" drop-down list.
  2. In the Parameters section, specify the desired maximum record size. You may specify a new maximum record size that is smaller than the current maximum record size, but not smaller than the current minimum record size. The "Option" field displays the command line that reflects the new record size. You cannot directly edit the contents of this field.
  3. To start the augment function, click Start.

To interrupt the augment function in progress, click the Stop button (the Start button becomes a Stop button after the function begins).

Click Close to exit vutil.
