21.10.4 I/O Handling Tab Pop-up Menu

Right-clicking in the File Designer I/O Handling tab creates a pop-up menu containing the following commonly used commands:

Menu Item Description

Add I/O Paragraph

Lets you add a paragraph to the list of user-defined I/O paragraphs.

Clear Paragraph

Lets you remove a user-defined paragraph and its value from the User-Defined I/O Paragraphs area.

Delete Function

Removes the contents of the Value column for a selected user-defined paragraph.

Undo

Reverses your most recent editing action.

Redo

Reverses your most recent undo action.

Select All

Lets you select all the paragraphs listed in User-Defined I/O Paragraphs area.

Find in Objects

Lets you locate all instances of a given string stored in the Property window and the graphical design interfaces, such as the Working Storage Editor or the File Designer.
