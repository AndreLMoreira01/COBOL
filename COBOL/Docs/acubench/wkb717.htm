7.3.2.8 Adding user-defined variables

The CFE allows you to add user-defined variables and comments.

  1. In the "Configuration file entries" tree view, right-click on the User Defined category and select New from the menu.
  2. In the field that displays, enter the name of the variable. Because this is a new variable, no value or existing comment text displays in the adjacent "Value" and "Comment" boxes.
  3. To ensure that the variable does not already exist, click on the Value label. If the variable has a pre-defined set of legal values, these display in a list. Click on a value to select it. If the variable does not have a set of pre-defined legal values, then double-click in the white space immediately below the label and enter the value (integer, floating-point, or character) that you want to set for this variable.
  4. Click in the Comment box and enter or edit text. There is no need to enter the (#) before the comment. The CFE inserts it for you. (The symbol will be visible in the CFE the next time you select the variable.)
  5. When you finish adding to the file, click OK to save the entry and close the CFE.
  6. Once user-defined variables have been added, they are stored in the "User Defined" category in the "Configuration file entries" tree view. You can locate and edit them as you would pre-defined variables.
