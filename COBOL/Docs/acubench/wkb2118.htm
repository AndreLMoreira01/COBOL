21.4 Screen and Report Component Toolbox Pop-up Menu

Right-clicking anywhere in the Screen Component Toolbox or Report Component Toolbox creates a pop-up menu containing the following commands:

Menu Item Description

List View

Lets you toggle the view of the controls in the Screen or Report Component Toolbox between bitmap buttons with pop-up hints and a list with labels.

ActiveX Control Components

Lets you determine which ActiveX controls appear in the ActiveX portion of the Screen Component Toolbox.

Allow Docking

Lets you toggle the appearance of the Screen or Report Component Toolbox as either floating or docked.

Hide

Lets you hide or display the Screen or Report Component Toolbox.
