19.2.16 Advanced

The Advanced command includes a submenu that allows you to perform a variety of tasks. This submenu's commands are described in the following table:

Command Description

List Paragraphs

Lets you see each paragraph name in the active Code Editor file, along with the name of the COPY file in which it appears. When you select this command, the Paragraph List box appears.

List Variables

Lets you see each variable name in the active Code Editor file, along with its level number, picture value, and the name of the COPY file in which it appears. When you select this command, the Variable List box appears.

List Constants

Lets you see each level-78 constant name in the active Code Editor file, along with its value and the name of the COPY file in which it appears. When you select this command, the Constant List box appears.

List COPY Files

Lets you see the name of each COPY file in the active Code Editor file, along with its directory and the name of the file in which it is declared. When you select this command, the COPY File List box appears.

Code Template

Lets you see all the code macros defined for the workbench. When you select this command, a list box of all the macros defined in the Tools/Options/Code Editor/Code Insight dialog box appears. If you double-click any macro in the box, the code associated with that macro is inserted into your active Code Editor document at the cursor location.

Open COPY File

Opens the COPY file that is selected in the active Code Editor document.

Insert File

Lets you insert data contained in another file into the active Code Editor document. When you choose this command, the Insert File dialog box displays (see below), allowing you to navigate to the desired file.

Find Scope

Automatically selects the entire effective range of a selected variable.

Verb Block Match

Lets you see the opposite end of a verb block command statement (for example, PERFORM or END-PERFORM). If you have selected one verb in the pair when you choose this command, the Code Editor automatically moves the cursor to the opposite-end verb.

The Insert File dialog options include:

Option Description

Look in

Locate and select the file you want to insert. The drop-down list allows you to select the directory or folder that contains the file, and the files included in the selected directory or folder are listed in the box below.

File name

Displays the name of the file to be inserted. This field is populated using the file name you select from the box above, or you can type the name of the file (and the full directory path if necessary). You can also use the "*" wildcard character to help define the file you want to insert.

Files of type

Allows you to select a specific file type to insert. The default file extension for source files is ".cbl".

Open

Click Open to insert the selected file.

Cancel

Click Cancel to exit the Insert File dialog without inserting a file.
