4.1.6 The Screen and Report Component Toolboxes

The AcuBench interface includes two toolboxes, one for the Screen Designer and one for the Report Composer. The Screen Component Toolbox contains the palette of ACUCOBOL-GT screen controls. It is divided into two sub-groups: Standard and ActiveX. Click the Standard button to display the ACUCOBOL-GT standard controls. Click the ActiveX button to display the list of ActiveX controls that you have made available to your project. For a list of ActiveX controls that are available to be listed in the Screen Component Toolbox, either right-click the ActiveX button and select ActiveX Control Components or select Axdefgen from the Tools menu. The Screen Component Toolbox is discussed in detail in section 10.3.

The Report Component Toolbox contains the "controls," or elements, that you can use to design different sections of a report. Some report controls are similar to screen controls. The Report Component Toolbox is covered in section 13.4.
