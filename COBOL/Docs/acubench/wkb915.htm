9.3.4 Moving a Selection

The Code Editor has the familiar Cut and Paste commands to help you move any selection to another location. Simply select the text you want to move and choose the Edit/Cut command. Position the cursor where you want the selected item to appear and choose the Edit/Paste command. If you want to move a block of text, press the Alt key and drag the cursor across the desired columns to select them. Then use the Cut and Paste commands.
