19.9.12.24 Options/Data Designer/Graphical FD, WS, and Linkage

The Data Designer options include three categories for designating level-number interval settings for the File Designer, Working Storage Editor, and Linkage Editor graphical designers. These three interfaces are identical except for the default Element Prefix setting. The options in this interface include:

Option Description

Set Element Prefix

Lets you change the default prefix setting in the entry field.

Elementary Level Settings

Lets you set the level-number intervals for the second, third, and fourth sub-item levels of generated code, as well as an interval for any subsequent levels. You cannot change the first-level setting. Your settings are reflected in the display box.
