7.6.1 Standard Options

The Standard compiler options allow you to specify basic preferences such as the names of object and error files, remote server name and port number, and the handling of warning messages.

You can set standard options as follows:

  1. Launch the Project Settings or Program Compile Options dialog box (the latter sets the Standard options for an individual file). The Standard Options category is the default selection in the Catalog drop-down list.
  2. Set the desired options. Observe that the default object file extension is displayed in the field beside the "Name of object file" check box. If your project's object files reside on a remote server, the server name and port number that you entered in the Project Information dialog appear in the Remote Compile area, and the "Name of object file" check box is disabled. The entry fields may be edited.
  3. If you select the "Name of error file" check box, you must enter a file name in the adjacent entry field.
