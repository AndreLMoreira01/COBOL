5.2.1 Displaying the Version Control Window

You can display the version control window using the following steps:

  1. On the AcuBench menu bar, select Tools/Options. The Options window is displayed.Click the plus sign ("+") next to the Environment menu item to view its subitems.
  2. Click Version Control to display the Version Control configuration window.
