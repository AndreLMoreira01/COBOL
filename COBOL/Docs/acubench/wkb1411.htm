14.2.2 Create and Refine Your Data Layout File (".dlt") in the Workspace Data view

  1. Create a data layout. In the workspace Data view, right-click on the project name, and select New FD/SL. Change the FD/SL name from FD1 to Salesdata. Select Import from files and click OK.
  2. In the Import FD/SL Files dialog, click on the browse button with the FD File Path entry field. By default, AcuBench browses the FD subdirectory of your project. Select SalesData.fd and click Open. Control returns to the Import FD/SL Files dialog box, which is automatically pre-filled with "SalesData.sl". Click OK to open the selected FD/SL in the File Designer interface.
  3. In the File Designer interface, select the IO Handling tab. Here, you define the I/O paragraphs associated with your data layout file and code them in the Event Editor. For our sample reports, this means adding two paragraphs-Before Master Print Loop and End Master Print Loop.
  4. To create a new I/O paragraph, click the Add push button to create a paragraph title area at the bottom of the listed paragraphs. This push button has a sunburst icon and is located at the top of the User-defined I/O Paragraphs section.
  5. The newly created Item cell has the title "Start-1". Double-click this cell and type Before Master Print Loop.
  6. The new Value cell also has the title "Start-1". Double-click this cell and type Before-Master-Print-Loop. This is the name of the paragraph that is generated in the ".evt" COPY file. Click the ellipsis button at the end of the Value cell to open the Event Editor with the cursor positioned in the named paragraph.
  7. The purpose of the Before-Master-Print-Loop in a report is to START the file and perform the first READ. Enter the following code for this paragraph: 
    8. MOVE LOW-VALUES TO SALES-KEY. 
START SALESDATA KEY > SALES-KEY 
READ SALESDATA NEXT RECORD 
     AT END MOVE "10" TO SALESDATA-STATUS 
END-READ.
  8. Repeat steps 4-7 for the creation of the End-Master-Print-Loop paragraph. The purpose of the End-Master-Print-Loop in a report is to drive the report with READ NEXT statements. Enter the following code: 
    9. READ SALESDATA NEXT RECORD 
     AT END MOVE "10" TO SALESDATA-STATUS 
END-READ
  9. Select the Use User-defined I/O Paragraphs radio button to ensure that your newly coded I/O paragraphs are included with the data set code generation.
