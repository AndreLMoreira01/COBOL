7.3.2.3 Displaying configuration variable values and comments

When the CFE opens the configuration file named in the "Alternate configuration file name (-c)" or "Remote configuration file name (-c)" option, a list of variables contained in the file is compiled and placed in the "Editing entry" drop-down list at the top left of the screen.

To display the value of an existing variable, select the variable from the "Editing entry" drop-down list. The selected variable's value displays in the "Value" area. The comment that immediately precedes the variable in the file, if any, displays in the "Comment" box. In addition, the "Configuration file entries" tree is automatically opened to the variable.
