7.1.2 The Project Settings Dialog

AcuBench uses a tabbed dialog box, called the Project Settings dialog, to facilitate the setting of compiler, runtime, and environment options. A fourth tab, the Library tab, is used to automate the process of creating an object library (see Chapter 17, "Working with ACUCOBOL-GT Utilities")

On the left side of the dialog is a project tree. Above it is the "Settings For" drop-down list. Use the tree view and drop-down list to select the project and mode that you want to change. The drop-down list shows the name of the mode being edited, and a field above the screen's four tabs shows the name of the project whose settings you are currently changing. To add a new mode, see section 7.1.3.

Compiler and runtime options are categorized into several groups. Use the "Catalog" drop-down list to select a category of options. Mutually exclusive options are presented in sets of radio buttons. Other options are presented with check boxes. Some options require a file or directory name or some other text; these options have an adjacent entry field. The aggregate command line of the selected options appears in the "Project Options" field at the bottom of the dialog. Individual command line flags can be entered directly in the "Additional options" entry field.

The buttons at the bottom of the Project Settings dialog allow you to save options to, or load options from, a file that you specify. Project option settings are saved in a file with the extension ".pof".

Option Description

Set as default

Saves the current settings in the file "default.pof". The settings in "default.pof" are used to create the Release and Debug modes for any new project that is created. (The mode stored in "default.pof" becomes Release mode, and a Debug mode is created by adding debugging flags to the compiler and runtime command lines.)

Save As

Saves the current settings to a file that you specify. Enter or select a file in the Save As dialog box.

Reference

Loads project option settings from a file that you specify. When you click Reference, the Open dialog box is displayed.

OK

Records the new settings and closes the dialog.

Cancel

Closes the dialog without recording any changes.
