5.5.3.2 .NET development

The application that will consume services must be modified to invoke the new component. You use the tools and language originally used to develop the application to execute these changes, in this case, Visual Studio .NET.

Using Visual Studio .NET, modify your .NET application as follows:

  1. Add a reference to the C# proxies that were generated from your SDL files. To add the proxies, do the following:
    1. In the Solution Explorer window, locate the project name that requires access to your COBOL service.
    2. Right-click on the project name and select Add Existing File.
    3. In the resulting dialog box, select the C# proxies that were generated for your ACUCOBOL-GT service and click OK. These proxies have a ".cs" filename extension.
  2. Build your .NET application as usual.
    1. Using the classes exposed through the proxies, instantiate an object for the COBOL component.
    2. Invoke the object and its methods to facilitate connectivity with the COBOL component. Name the service to be invoked, and pass the host name and port number as parameters.
