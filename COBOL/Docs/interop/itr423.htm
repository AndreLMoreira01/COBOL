4.18 ActiveX Troubleshooting

Symptom Cause

Control does not work within AcuBench (e.g., you receive a message such as "Can't create OCX.MSFlexGrid Control").

Control is not properly installed or registered on the development system. Refer to section 4.2 for instructions on installing and registering controls.

Control does not appear within the AXDEFGEN list.

Control is not properly installed or registered on the development system. Refer to section 4.2 for instructions on installing and registering controls.

Control does not work correctly with the ACUCOBOL-GT runtime on the end-user's machine.

Control is not properly installed or registered on the end-user's system. Refer to section 4.10 for instructions on distributing ActiveX files along with your application.
