10.4.1.1 Connecting to the queue manager

To connect to the WebSphere MQ queue manager from your ACUCOBOL-GT program, use the following code: 

CALL 'MQCONN' 
    USING QM-NAME, by reference HCONN, 
    COMPCODE, REASON

in the following modes:

Input Mode:

Variable Description Definition

QM-NAME

Variable string that contains the name of the queue manager

User-defined

Output Mode:

Variable Description Definition

HCONN

S9(9) BINARY handle to queue manager

User-defined

COMPCODE

S9(9) BINARY returns the completion code

User-defined

REASON

S9(9) BINARY returns the reason

User-defined
