2.3.1.2 Supported parameter types

Following is a list of Java parameter types that are supported by ACUCOBOL-GT: 

V - void 
Z - boolean 
B - byte 
C - char 
S - short 
I - int 
J - long 
F - float 
D - double 
X - string 
LString; 
Ljava/lang/String; 
Ljava/lang/Object; 
 
[Z - boolean array 
[B - byte array 
[C - char array 
[S - short array 
[I - int array 
[J - long array 
[F - float array 
[D - double array 
[X - string array 
[LString; - string array 
[Ljava/lang/String; - string array 
[Ljava/lang/Object; - object array
