To define the runtime as a helper application in older versions of Netscape

The following procedure was written for Version 3 of Netscape Navigator. Other versions of Netscape will operate differently. To configure Netscape Navigator Version 3 to use ACUCOBOL-GT as a helper application, do the following:

  1. Start Netscape Navigator.
  2. Select General Preferences from the Options menu.
  3. Click the Helpers tab.
  4. Click Create New Type to open the Configure New Mime Type dialog box.



  5. Click inside each field in the box and do the following:

    6. Field Action

    MIME Type

    Type "application"

    MIME SubType

    Type "vnd.acucobol"

  6. Click OK. The Helpers tab reappears.
  7. In the "File Extensions" field, type "acu".
  8. Click Launch the Application.
  9. Browse to and select the ACUCOBOL-GT runtime file "wrun32.exe". This is the command line used to launch the application. If users want to specify command line options, they may do so by typing the option after the executable name.

    10. The Helpers tab should now look similar to this:



  10. Click OK.
