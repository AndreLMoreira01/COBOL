5.4.7 VALUE OF LABEL Clause

This clause describes the values contained in a file's label records.

General Format 

[ VALUE OF LABEL IS label-lit ]

Syntax Rule

Label-lit must be a non-numeric literal.

General Rule

This clause is ignored by the compiler.
