11.3.2 acushare -kill

To halt acushare, simply enter "acushare -kill" on the command line.

Should acushare terminate unexpectedly (for example, due to a SIGKILL signal), you should remove the stranded shared memory segment with the "-clean" option before restarting. For more information, see section 11.3.3, "acushare -clean."
