HELP_PROGRAM

If the program uses help automation (see Book 2, Chapter 10), this variable should be assigned the name of the help processor program. The help processor's entry point is always a COBOL program. The program can be the help processor itself, or a shell to some other help processor, such as Windows Help. The default value of HELP_PROGRAM is "AcuHelp".
