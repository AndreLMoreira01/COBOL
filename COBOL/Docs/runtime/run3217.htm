Chapter 3: Runtime Configuration File > 3.2 Configuration File Entries >

RESTRICTED_VIDEO_MODE

This value determines which rules will be followed when the program is positioning attribute characters for "magic cookie" terminals (terminals whose attributes occupy screen positions). For more details, see section 2.5, "Restricted Attribute Handling," in Chapter 2. The default is "0".
