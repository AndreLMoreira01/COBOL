3.6.2 Starting acurcl on a Windows Server

Issue the "acurcl -start" command from the Administrator account or from an account that belongs to the Administrators group. You can also click the Start button on the Windows graphical control panel's Services tab.

On Windows servers, it is best to specify "acurcl -start" with no options.

You can stop AcuConnect at any time by issuing the "acurcl -kill" command or by clicking Stop on the control panel's Services tab in Windows. See section A.7, "acurcl -kill," for a discussion of the "-kill" command options.
