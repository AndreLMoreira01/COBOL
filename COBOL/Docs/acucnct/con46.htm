4.1.4 Setting Up the Host Name

The runtime system on each client machine needs to know the name of the server host machine. This is accomplished in the client configuration file, "client.cfg." Either you can use the CODE_PREFIX variable to define the remote pathname, or you can define code name aliases. Code name aliases are faster, because the runtime does not have to search remote machines. If you use CODE_PREFIX, we recommend that you place the remote paths at the end of the file. Refer to section 4.2.1, "Defining Remote Application Path," for details.
