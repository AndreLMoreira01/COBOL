A.1 acurcl Command

The acurcl command is your control and maintenance interface to AcuConnect®. The acurcl command is used on the server at the command line (or with the graphical control panel under Windows NT/2000/2003) and initiates the following actions:

  • starts and stops the acurcl program and installs and removes acurcl as a Windows service
  • controls the configuration of running instances of AcuConnect
  • reports AcuConnect system status
  • creates and maintains the server access and server alias files

When "acurcl" is entered with no options, a list of options (or a graphical control panel in Windows) displays.

The acurcl command formats on UNIX servers are: 

acurcl 
acurcl -access 
acurcl -alias 
acurcl -config [server] [-n  port] 
acurcl -info [server] [-n port] 
acurcl -kill [server] [-f] [-n port] 
acurcl -start [-c config_file] [-e error_file] [-l] 
              [-t #] [-f] [-n port] 
acurcl -version

In addition to these command formats, the following formats are available on Windows servers: 

acurcl -install [server] [-depends ServiceDependency]. 
                [any valid start options] 
acurcl -query [server] [-n  port] 
acurcl -remove [server] [-n port] [-f]

AcuConnect graphical control panel

When the acurcl command is entered with no options in Windows, a graphical control panel with tabs appears. The tabs correspond to the various actions available from the command line. You can access help from the acurcl command graphical control panel by pressing the F1 key, as well as by clicking the Help push button.

The Access tab allows you to maintain the AcuAccess file. Click Open to select a file to maintain, and the contents of the selected file appear. Select a user to modify or delete, or click New to add a new user.

The Config tab allows you to set and view the current configuration settings of a running server. Click Query to select a server/port combination, and configuration settings appear. Select a configuration variable to modify, or click New to add a new one. You can also enable an automatic refresh function for the Info tab here.

The Alias tab allows you to maintain AcuConnect thin client aliases. Click Open to select a file to maintain, and the aliases in the file appear. Select an alias to modify or delete, or click New to add a new alias.

The Info tab allows you to list child processes. Click Query to select a server/port combination. Child processes are listed for that server, along with the version of the server. Select a child process to terminate if desired. Click Refresh to update the information displayed on the Info tab.

The Services tab allows you to add, start, stop, and remove services, as well as view or modify the properties of an existing service. When this tab is chosen, services that are currently running are marked with a green bullet, and those stopped are marked with a red bullet. Select a service to see or modify the current properties, or to start, stop, or remove it. Click New to add a new service.

Note that some of the tabs are available only if you have Administrator privileges.
