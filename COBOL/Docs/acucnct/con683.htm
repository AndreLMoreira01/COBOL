6.4 Starting AcuConnect

For a thin client to access programs on the server, AcuConnect must be running. To start and stop AcuConnect (as well as perform several system administration functions such as checking system status, maintaining the server access file, or installing acurcl as a Windows service), you issue the "acurcl" command on the server at the command line or use the graphical control panel's Services tab in Windows NT, 2000, and 2003.

This section provides specific instructions for issuing the "acurcl -start" command on the server. Chapter 8 describes acurcl's systems administrator functions, and Appendix A provides a complete reference to acurcl command formats and options.
