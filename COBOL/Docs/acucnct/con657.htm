6.3.2 Assigning Values to Server Configuration Variables

AcuConnect includes a server configuration file named "acurcl.cfg." On UNIX servers, AcuConnect searches for this file in /etc by default. On Windows servers, AcuConnect searches in OS drive:\etc on the disk device that AcuConnect is installed on. On VMS servers, AcuConnect searches in the etc directory where ACUCOBOL-GT is installed. If the file is given another name, or located in another directory, the full location and name of the file must be specified with the "-c" option when you start AcuConnect. See section 6.4, "Starting AcuConnect," for more details.
