7.1 Preparing the Client

In the AcuConnect® thin client environment, the client is also known as the display host. It is where screen, mouse, and keyboard handling is performed, but no actual application processing is performed on the client. Unlike servers (or application hosts) which are considered to be "thick," display host clients are "thin" due to their lack of application components and substantive processing work. Thin clients communicate with their application hosts through a split runtime, the thin portion of which is installed on the client.

There are only two steps to preparing the client for AcuConnect thin client operation. You must:

  1. Install client software.
  2. Run the ACUCOBOL-GT® Thin Client.
