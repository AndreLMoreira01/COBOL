5.2 Preparing Your Data

There is nothing special that you need to do to prepare your data for use in a thin client environment. Because your data resides on a server (either the same server as the application or on a different data server) access to that data is the same as it would be in a standard client/server environment using AcuServer® or Acu4GL®. The server runtime just passes the results back to the ACUCOBOL-GT Thin Client for display. If the data resides on the same server as the application runtime, no extra communication software is required.
