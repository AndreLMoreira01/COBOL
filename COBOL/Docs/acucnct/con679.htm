6.3.3.3 Modifying an alias

To modify an alias in the alias file, choose item 3 from the alias menu. You receive the following prompt: 

Modify an alias 
Enter the alias name:

Enter the name of the alias that you want to modify. If the named alias does not exist in the alias file, a message displays: 

Alias does not exist

and you are returned to the alias menu. Otherwise, you see the current working directory and are asked for a new one. Press Return to leave the working directory unchanged, or type the name of a different directory. After you enter a directory, you see the command line as it currently exists in the alias file, and are asked for a new command line. Press Return to leave the entry unchanged or enter a new command line. You are then returned to the alias menu.
