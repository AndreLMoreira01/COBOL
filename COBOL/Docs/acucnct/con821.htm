8.8.2.4 TC_TV_SELCHANGING configuration variable

The tree view control normally generates a Msg-Tv-Selchanging event in some unexpected circumstances. For example, it is generated when a tree view becomes active, even if the selection in the tree view is not changing. The frequency with which these events are passed between the client and the server can cause performance issues.

The TC_TV_SELCHANGING configuration variable lets you control how often Msg-Tv-Selchanging events are generated in a thin client environment. Different values for this variable allow you to suppress the generation of this message event to varying degrees. For more information about this configuration variable, refer to section 6.3.1.43, "TC_TV_SELCHANGING."
