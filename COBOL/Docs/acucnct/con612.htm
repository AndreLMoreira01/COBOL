6.3.1 Configuring the Remote Application Runtime

Runtime configuration files have a similar function in the thin client configuration as they do in AcuConnect's distributed processing deployment. The file is stored on the server along with the application, and its name is specified in a server alias file that you establish when first setting up the system. (See section 6.3.3, "Creating a Server Alias File," for details). The following paragraphs describe the important variables for the thin client environment listed in the table.
