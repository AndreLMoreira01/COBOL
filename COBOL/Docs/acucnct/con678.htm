6.3.3.2 Removing an alias

To remove an alias from the alias file, choose item 2 from the alias menu. You receive the following prompt: 

Remove an alias  
Enter the alias name:

Enter the name of the alias that you want to remove. If the named alias does not exist in the alias file, a message displays: 

Alias does not exist

and you are returned to the alias menu. Otherwise, you see the current working directory and command line for the alias, and are asked to confirm deleting the entry: 

Are you sure you want to remove this alias?

Enter Y or N. In either case, you are returned to the alias menu. If you answered Y, then the alias is no longer an entry in the alias file.
