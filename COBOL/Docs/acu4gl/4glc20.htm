ORA_LOGIN

ORA_LOGIN indicates the user name under which you want to connect to the database system.

Example

To connect to the database with the user name MYNAME, you would specify: 

ORA_LOGIN  MYNAME

in the configuration file.

If ORA_LOGIN is not set, the runtime attempts to do an automatic login for you. For autologin to succeed, you must have set up an account OPS$USER where USER is the same as your computer login name.
