4GL_WHERE_CONSTRAINT

4GL_WHERE_CONSTRAINT allows you to override any value given in the external variable A4GL-WHERE-CONSTRAINT. This configuration variable can have a value with any length (limited only by machine memory). If a value is set in 4GL_WHERE_CONSTRAINT, any value in the external variable is ignored. See section 9.1.2, "The WHERE Constraint," for information on the external variable.
