INF_LOGIN

INF_LOGIN indicates the user name under which you want to connect to the database system. This is an optional variable. If INF_LOGIN and INF_PASSWD are not set, then a default login is performed. If these variables are set, there must be a matching UNIX login name and password.

Example

To connect to the database with the user name MYNAME, you would specify: 

INF_LOGIN  MYNAME

in the configuration file.
