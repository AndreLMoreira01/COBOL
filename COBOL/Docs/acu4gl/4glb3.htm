B.2 Acu4GL for Microsoft SQL Server Installation and Setup

You must perform several steps before you begin using Acu4GL for Microsoft SQL Server on a new system. The following topics provide this information:

  • Installing Acu4GL for Microsoft SQL Server on a Client Machine
  • Acu4GL for Microsoft SQL Server: Setting Up a User Account
  • Acu4GL for Microsoft SQL Server: Setting Up the User Environment
  • Acu4GL for Microsoft SQL Server: Designating the Host File System

The Microsoft SQL Server RDBMS, version 6.5 or later, must be installed and configured prior to the installation of Acu4GL for Microsoft SQL Server.

Microsoft SQL Server's "Query Analyzer" product, an interactive query tool, is also necessary for installing the ACUCOBOL-GT stored procedures.

Acucorp does not provide these products.

Note: The default sort order for Microsoft SQL Server is case-independent. Make sure that this is what you really want when you install the server. Acu4GL for Microsoft SQL Server will use the sort order as installed, and can not change its behavior to do case-dependent ordering if that is not what the server uses.

First you must install the files from the Acucorp diskette onto the client machine. Then follow the instructions below for the server machine. Final setup steps on the client machine complete the installation.
