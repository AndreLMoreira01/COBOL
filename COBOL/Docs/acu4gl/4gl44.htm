4.3 Directives Supported by the Acu4GL Interfaces

The Acu4GL interfaces support the following directives:

  • ALPHA
  • BINARY
  • COMMENT
  • DATE
  • FILE
  • NAME
  • NUMERIC
  • SECONDARY_TABLE
  • USE GROUP
  • VAR_LENGTH
  • WHEN
  • XSL
