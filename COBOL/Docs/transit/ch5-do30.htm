5.2.27 SPECIAL-NAMES

The following system names are supported by ACUCOBOL-GT in the SPECIAL-NAMES paragraph: 

SYSPCH 
SYSPUNCH 
C01 through C12 
CSP 
S01 through S05

The SYSPCH and SYSPUNCH names do not represent an actual card punch in ACUCOBOL-GT. Instead, output directed to the card punch is written to the file SYSPUNCH.TXT. (See DISPLAY UPON SYSPUNCH, above, for details.)

The names C0 through C12 and S01 through S05 are accepted but ignored by ACUCOBOL-GT, since they have no meaning for printers attached to a DOS/Windows or Unix system. The name CSP suppresses spacing.
