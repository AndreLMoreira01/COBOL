5.21.4 Events

The WEB-BROWSER control generates the following events:

MSG-WB-BEFORE-NAVIGATE

MSG-WB-DOWNLOAD-BEGIN

MSG-WB-DOWNLOAD-COMPLETE

MSG-WB-NAVIGATE-COMPLETE

MSG-WB-PROGRESS-CHANGE

MSG-WB-STATUS-TEXT-CHANGE

MSG-WB-TITLE-CHANGE
